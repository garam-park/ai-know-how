# Task Management Rule

- tasks 폴더는 plan 과 done 폴더로 나뉘어져 있습니다. plan 폴더에는 앞으로 해야할 일들을 기록하고, done 폴더에는 완료된 일들을 기록합니다. 각 파일은 Markdown 형식으로 작성되어야 하며, 파일 이름은 작업의 제목을 간결하게 표현해야 합니다.
- 계획한 작업은 tasks/plan/{yyyymmdd}/{task-title}.md 형식으로 작성되어야 합니다.
- 완료된 작업은 tasks/done/{yyyymmdd}/{task-title}.md 형식으로 이동합니다.
