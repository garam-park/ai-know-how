# RD-12 WF-08~10 데이터수집·카탈로그·시스템관리 보완

## 메타

| 항목 | 값 |
|------|-----|
| 우선순위 | 12 |
| 선행작업 | RD-04 (검색/필터 고도화) |
| 대상 화면 | WF-08-01~06, WF-09-01~06, WF-10-01~05 (총 17개) |

## 배경

이 3개 그룹은 공통적으로 동일한 미비 패턴을 보임: 테이블 정렬/필터 미완, 다운로드/내보내기 기능 미흡, 상세 검색 기능 제한. RD-04에서 공통 함수가 추가되면 이를 각 화면에 적용하는 작업.

## 수정 사항 — 공통 패턴 (17개 화면 모두 적용)

- [ ] `initTableSort()` 적용 — 각 화면의 주요 데이터 테이블에 정렬 기능 연결
- [ ] `initTableFilter()` 적용 — 테이블 상단 검색 입력란 연결
- [ ] 다운로드/내보내기 버튼 클릭 시 토스트 알림 ("CSV 다운로드 준비 중...")

## 수정 사항 — WF-08 데이터수집전처리 (6개)

### WF-08-01 데이터수집현황
- [ ] 수집 소스별 상세 드릴다운 링크 추가

### WF-08-02 OCR처리현황
- [ ] 처리 결과 상세 모달 (원본 이미지 플레이스홀더 + 추출 텍스트 비교)

### WF-08-03 RPA수집모니터
- [ ] 에러 로그 상세 모달 (에러 메시지, 스택트레이스 플레이스홀더, 재시도 버튼)

### WF-08-04 STT처리현황
- [ ] 품질 점수 기준선 표시 (차트에 80% 수평선 + "기준" 라벨)

### WF-08-05 개체명추출결과
- [ ] 더미 데이터 보강 (개체명 유형별 10건 이상: 인명, 지명, 기관명, 날짜, 금액)

### WF-08-06 데이터품질대시보드
- [ ] 품질 추이 라인차트 추가 (최근 30일, 완전성/정확성/적시성 3개 라인)
