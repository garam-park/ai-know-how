## 수정 사항

### WF-01-02 실시간현황모니터

- [ ] 게이지 차트를 RD-01의 `createGauge()` 함수로 교체
- [ ] 자동 새로고침 시뮬레이션 추가 (10초 간격 setInterval, 더미 값 변경)
- [ ] 실시간 타임스탬프 표시 ("최근 갱신: 00:00:00")

### WF-01-04 공지-알림센터

- [ ] 읽음/안읽음 상태 토글 버튼 구현 (클릭 시 스타일 변경)
- [ ] 알림 유형별 필터 탭 추가 (전체/시스템/민원/보안)
- [ ] "모두 읽음" 일괄 처리 버튼

### WF-01-05 마이대시보드

- [ ] 위젯 카드에 "×" 제거 버튼 추가 (클릭 시 카드 숨김)
- [ ] "위젯 추가" 버튼 → 모달에서 위젯 목록 선택 UI
- [ ] 위젯 배치 순서 변경은 드래그앤드롭 대신 ↑/↓ 버튼으로 대체 (Mid-fi 수준)

### WF-01-06 시스템상태모니터링

- [ ] 센서 상태값 자동 갱신 시뮬레이션 (15초 간격, 랜덤 미세 변동)
- [ ] 임계값 초과 시 행 배경색 경고(노랑) / 위험(빨강) 자동 적용
- [ ] 경보 이력 미니 타임라인 (최근 5건)
