## 탭별 내용

### 1. 개요 탭
- **설명**: 데이터셋의 상세 설명 (여러 문단 가능)
- **키워드 태그**: 관련 키워드 칩 리스트 (클릭 시 검색)
- **분류**: "분야: 교통, 주제: 실시간, 조직: 교통과"
- **활용사례**: 링크 목록 또는 카드 (예: "교통 혼잡도 분석", "버스 운행 모니터링")

### 2. 스키마 탭
필드 목록 테이블:
| 필드명 | 타입 | 설명 | 필수여부 |
|--------|------|------|--------|
| id | String | 고유 식별자 | Yes |
| name | String | 데이터셋명 | Yes |
| timestamp | DateTime | 수집 시간 | Yes |
| value | Number | 측정값 | No |

- 테이블 정렬 가능
- 필드 클릭 시 상세 정보 표시 (타입, 범위, 예시값)

### 3. 미리보기 탭
- 샘플 데이터 테이블 (10행 정도)
- 스크롤 가능, 페이지네이션 또는 "더 보기" 버튼
- 필드명 헤더
- 샘플 데이터 행

### 4. API 탭
- **엔드포인트 URL**: 코드 블록으로 표시
  ```
  https://api.data-hub.jeju.go.kr/v1/datasets/{dataset-id}/data
  ```
- **인증방법**: 설명 텍스트 (API Key, OAuth 등)
- **요청 예시**: 코드 블록 (JSON 형식)
  ```json
  {
    "start_date": "2026-03-01",
    "end_date": "2026-03-31"
  }
  ```
- **응답 예시**: 코드 블록 (JSON 형식)
  ```json
  {
    "data": [...],
    "count": 1000
  }
  ```
- "API 문서" 링크
