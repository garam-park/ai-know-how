## 화면 구성

### 상단 정보 영역
- 시나리오명 (예: "2024년 여름 혼잡기 대비 - 주차장 확장 시뮬레이션")
- 적용 정책 요약 카드 3개
  - 카드 1: 정책 유형 (아이콘 + 텍스트)
  - 카드 2: 적용 규모 (증설 면수/요금 수준 등)
  - 카드 3: 기간 (적용 시작~종료)

### 중앙 비교 영역 (2개 선택 가능)
- **옵션 1**: 좌우 분할 지도
  - 좌측: Before (정책 미적용)
  - 우측: After (정책 적용)
  - 각 지도에 히트맵 오버레이(수요/공급 현황)
  - 줌/패닝 동기화

- **옵션 2**: 슬라이더 비교
  - 단일 지도에 슬라이더
  - 좌측: Before, 우측: After
  - 드래그로 비교 영역 선택

### 하단 효과 지표 카드 영역
6개 주요 지표 카드 (격자 배치, 2행 3열)

| 순번 | 지표 | 표시형식 |
|------|------|---------|
| 1 | 민원 감소율 | 수치(%) + 증감 화살표 + 비교 백분율 |
| 2 | 주차 가용률 변화 | 수치(%) + 증감 화살표 |
| 3 | 혼잡도 변화 | 수치(%) + 증감 화살표 |
| 4 | 비용 효율 | 수치(%) + 투자대비 기대효과 |
| 5 | 시민 만족도 예측 | 점수(5.0/10.0) + 별점 표현 |
| 6 | 투자 대비 효과 | ROI(%) + 연간 예상 절감액(₩) |

- 각 카드: 좌측에 수치/차트, 우측에 해석 텍스트
- 긍정 지표: 초록색, 부정 지표: 빨강색
- 화살표: 상향(증가)/하향(감소) 방향 표시
