## 화면 구성

### 상단 영역
- **정렬 옵션**
  - 라디오 버튼: 건수순 / 증가율순 / 가나다순
- **뷰 전환**
  - 토글 버튼: 카드뷰 / 테이블뷰

### 메인 영역
- **카드 그리드 레이아웃** (반응형 3열)
  - 각 카드: 읍면동 하나씩 표현
  - 약 30개 카드 표시

### 카드 내부 구성
- **헤더**
  - 지역명 (예: 제주시 건입동)
  - 민원 총건수 (예: 45건)

- **증감 지표**
  - 전주 대비 증감율 (예: +12%, ↑ 강조 표시)
  - 색상: 증가(빨강) / 감소(파랑)

- **주요 유형 미니 파이차트**
  - 상위 3개 유형만 표시
  - 범례와 백분율

- **액션**
  - 카드 클릭 시 해당 지역 상세 대시보드로 이동 (링크)

## 사용 컴포넌트
- 정렬/뷰 전환 컨트롤
- 비교 카드 컴포넌트
- 미니 파이차트 (Chart.js/D3.js)
- 공통 레이아웃 (WF-00-02)
