## 화면 구성

### 상단 옵션 영역
- **답변 톤 선택** (드롭다운)
  - 공식 (formal)
  - 친절 (friendly)
  - 간결 (concise)

- **답변 유형 선택** (라디오 버튼 또는 탭)
  - 민원 답변
  - 정책 안내
  - 공문/공지

### 좌측 패널 (원문 민원)
- **민원 내용** (읽기 전용)
  - 민원 제목
  - 본문 내용
  - 분류 태그 (예: 교통/주차)
  - 감성 태그 (긍정/중립/부정)
  - 작성자, 작성일시 (마스킹)

### 우측 패널 (AI 생성 답변)
- **AI 생성 답변 초안** (편집 가능)
  - 텍스트 에디터 (Rich Text Editor)
  - 실시간 글자 수 카운터
  - 자동 저장 표시

### 하단 워크플로우 버튼 영역
- **초안 저장** 버튼 (회색)
- **검토 요청** 버튼 (파랑)
- **승인** 버튼 (회색 - 조건부 활성화)
- **발송** 버튼 (빨강 - 조건부 활성화)

### 하단 우측 (유사 답변 참고)
