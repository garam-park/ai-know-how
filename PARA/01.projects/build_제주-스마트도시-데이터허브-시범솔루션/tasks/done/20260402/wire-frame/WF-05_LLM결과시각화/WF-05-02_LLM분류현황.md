## 화면 구성

### 상단 영역
- **분류 유형 선택**: 라디오 버튼 또는 드롭다운
  - 단일분류
  - 멀티분류
  - 계층적분류

### 메인 콘텐츠 영역 (좌측)
- **트리맵 차트**: 계층적 분류 비율 시각화
  - 각 영역 크기 = 분류별 비율
  - 색상 구분 = 카테고리
  - 마우스 오버 시 정확도 표시

### 우측 사이드패널
- **분류 드리프트 감지 알림**
  - 경고 아이콘
  - 분류 분포 변화 안내
  - 재학습 필요 여부 판단
  - 권장 액션 버튼

### 하단 영역
- **분류별 상세 테이블**
  - 칼럼: 분류명, 건수, 정확도, 최근 변화율
  - 정렬/필터 기능
  - 페이지네이션
