## 화면 구성
Leaflet 기반의 인터랙티브 지도 컴포넌트 페이지. 제주도 중심의 맵을 기본으로 하며, 다양한 지도 레이어 및 마커를 포함.

## 지도 기본 설정

### 초기화 설정
- **중심점**: 제주도 중심 (위도: 33.3617, 경도: 126.5292)
- **초기 줌 레벨**: 11 (제주도 전체 포함)
- **최소 줌**: 9 (광역 뷰)
- **최대 줌**: 18 (상세 뷰)
- **맵 스타일**: OpenStreetMap 기본 타일 (또는 위성/지형 옵션)

### 지도 타일 옵션
- 기본 (Roadmap): OpenStreetMap
- 위성 (Satellite): Sentinel-2 위성 이미지 (선택)
- 지형 (Terrain): 지형 레이어 (선택)

## 맵 요소

### 1. 마커 (Markers)
**설명**: 특정 위치를 나타내는 핀
**사용 사례**: 공영주차장, 환경 센서, 민원 위치

**구성**:
- 아이콘: 기본 빨강색 핀, 또는 카테고리별 맞춤 아이콘
- 팝업: 마커 클릭 시 정보 표시
- 라벨: 마커 위에 번호 또는 이름 표시

**제주도 주요 주차장 (9개소 더미 데이터)**:
1. 제주국제공항 주차장: (33.5063, 126.4929)
2. 롯데면세점 주차장: (33.3192, 126.5289)
3. 신라호텔 주차장: (33.3156, 126.5328)
4. 중앙로 지하주차장: (33.3182, 126.5334)
5. 탑동 공영주차장: (33.3214, 126.5385)
6. 건입동 주차장: (33.3297, 126.5416)
7. 제주항 주차장: (33.3084, 126.5125)
8. 칠성로 주차장: (33.3245, 126.5245)
9. 첫마음길 주차장: (33.3120, 126.5198)

**마커 팝업 내용**:
```
[주차장명]
수용률: 85%
이용가능: 45대
주소: ...
```

### 2. 클러스터 마커 (Clustered Markers)
**설명**: 마커가 많을 때 자동으로 그룹화하는 기능
**동작**:
- 줌 아웃 상태: 마커들을 원형 클러스터로 표시
- 클러스터 숫자: 그룹 내 마커 개수 표시 (예: "12")
- 줌 인: 클러스터가 해제되고 개별 마커 표시
- 클러스터 클릭: 해당 영역으로 줌 인

**색상**: 개수에 따라 3단계
- 적음 (5~10): 연한 파랑
- 중간 (11~20): 주황색
- 많음 (20+): 빨강색

### 3. 히트맵 레이어 (Heatmap Layer)
**설명**: 밀도 기반 색상 그라데이션으로 데이터 분포 표시
**사용 사례**: 교통 혼잡도, 민원 밀도, 주차장 이용 집중도

**구성**:
- 색상 그라데이션: 파랑(저) → 빨강(고)
- 투명도: 0.7 (배경 지도 보임)
- 반지름: 15px (가우시안 블러)

**더미 데이터**: 제주시 주요 지역의 민원 밀도 (200+ 포인트)

### 4. 폴리곤 레이어 (Polygon Layer)
**설명**: 지역 경계를 나타내는 도형
**사용 사례**: 읍면동 행정 경계, 도시 구간 표시

**제주도 행정 경계**:
- 제주시 (중심 주변)
- 서귀포시 (남쪽)
- 북제주군 (북쪽, 선택 시 표시)
- 남제주군 (남쪽, 선택 시 표시)

**구성**:
- 테두리: 검정색, 2px, 불투명
- 채우기: 구간별 서로 다른 연한 색상, 0.2 투명도
- 호버 시: 채우기 투명도 증가 (0.5)

**팝업** (폴리곤 클릭 시):
```
[지역명]
면적: XX.XX km²
인구: XX,XXX명
주차장 수: XX개
```

### 5. 서클 레이어 (Circle Layer)
**설명**: 반경을 나타내는 원형 영역
**사용 사례**: 서비스 범위, 영향 범위 표시

**예시**: 주차장에서 500m, 1km 반경 원 표시

**구성**:
- 테두리: 색상별, 2px
- 채우기: 연한 색상, 0.2 투명도
- 반지름: 픽셀 아님, 미터 단위 (L.circle)
- 범례: "500m", "1km" 레이블

### 6. 팝업 (Popup)
**설명**: 마커/도형 클릭 시 정보 표시
**구성**:
- 제목 (굵음)
- 내용 (텍스트 또는 테이블)
- 닫기 버튼
- 최대 너비: 250px

**마커 팝업 예시**:
```
제주국제공항 주차장
━━━━━━━━━━━━━━━━━━━
운영상태: 정상 (초록 뱃지)
수용 용량: 450대
현재 이용: 382대
이용률: 85%
위치: 33.5063, 126.4929
━━━━━━━━━━━━━━━━━━━
[상세보기 버튼]
```

### 7. 범례 (Legend)
**설명**: 지도 요소의 의미 설명
**위치**: 우측 하단
**구성**:
- 마커 아이콘 + 레이블 (예: "공영주차장")
- 색상 스케일 (히트맵)
- 폴리곤 색상 (행정 경계)
- 서클 반지름 (범위 표시)

**접기/펼치기**: 우측 상단 버튼으로 범례 토글

### 8. 레이어 토글 컨트롤 (Layer Toggle Control)
**설명**: 레이어 표시/숨김 제어
**위치**: 좌측 상단
**항목** (체크박스):
- [ ] 마커 (공영주차장)
- [ ] 클러스터 마커
- [ ] 히트맵 (민원 밀도)
- [ ] 폴리곤 (행정 경계)
- [ ] 서클 (서비스 범위)

**동작**: 체크박스 클릭으로 해당 레이어 표시/숨김

### 9. 스케일 표시 (Scale Control)
**위치**: 좌측 하단
**내용**: 거리 스케일 (예: "1km ─────")
**자동 갱신**: 줌 레벨 변경 시 스케일 자동 계산

## 인터랙션 및 기능

### 마우스 제어
- 드래그: 맵 이동
- 스크롤: 줌 인/아웃
- 더블 클릭: 줌 인
- 클릭: 마커/도형 정보 표시

### 터치 제어 (모바일)
- 1손가락 드래그: 맵 이동
- 2손가락 핀치: 줌 인/아웃
- 더블 탭: 줌 인

### 검색 기능 (선택 사항)
- 검색 박스: 좌측 상단
- 입력 예시: "주차장명" 또는 "지역명"
- 결과: 해당 위치로 줌 이동, 마커 강조

### 지도 타입 전환
- 우측 상단: 버튼 그룹
  - 기본지도 (Roadmap)
  - 위성 (Satellite)
  - 지형 (Terrain)
