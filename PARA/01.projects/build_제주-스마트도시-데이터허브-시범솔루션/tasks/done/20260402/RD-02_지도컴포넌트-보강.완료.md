## 배경

## 수정 사항

### map.js 공통 지도 라이브러리

- [ ] **폴리곤 오버레이 함수** — `addPolygonLayer(map, geojsonData, styleOptions)`
  - GeoJSON 기반, 색상/투명도 매핑, 호버 시 정보 팝업
  - WF-02-08 GIS집중구역, WF-03-05 요금제추천에서 사용

- [ ] **서클 레이어 함수** — `addCircleLayer(map, points, radiusField, colorField)`
  - 반경 크기 비례 표현, 색상 매핑, 클릭 이벤트
  - WF-02-02 히트맵에서 보조 레이어로 사용

- [ ] **히트맵 레이어 함수** — `addHeatmapLayer(map, points, options)`
  - Leaflet.heat 플러그인 활용, 강도 매핑, 반경·블러 조절
  - WF-02-02 히트맵분석에서 사용

- [ ] **마커 클러스터링** — `addClusteredMarkers(map, markers, options)`
  - Leaflet.markercluster 활용, 줌 레벨별 자동 그룹화
  - WF-03-05 공유주차 마커, WF-04-04 센서마커에서 사용

- [ ] **레이어 토글 컨트롤** — `addLayerControl(map, layers)`
  - 체크박스 기반 레이어 표시/숨김, 범례 표시
  - 모든 지도 화면에서 공통 사용

- [ ] **지도 내 검색 함수** — `addSearchControl(map, options)`
  - 텍스트 입력 → 좌표 이동 (더미: 제주도 내 주요 지점 5개)
  - WF-02-08, WF-03-05에서 사용
