## 주의 사항

1. **장비 도입 리드타임**: GPU 서버, 망연계 시스템 등 장비 도입은 발주가 늦어지면 전체 일정에 연쇄 영향
2. **LLM API 키/라이선스**: LG 엑사원 유료 버전 라이선스 계약 시점 확인 필요
3. **현장 설치 일정**: 이노뎁의 센서/CCTV 설치는 외부 환경(날씨, 허가 등)에 영향받음
4. **중간보고(4개월차)**: 이 시점까지 데이터 파이프라인과 LLM 기본 기능이 동작해야 함
