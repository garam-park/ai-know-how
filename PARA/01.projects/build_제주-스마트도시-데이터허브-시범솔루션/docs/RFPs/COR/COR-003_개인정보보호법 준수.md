# COR-003 개인정보보호법 준수

| 항목                  | 내용                |
| --------------------- | ------------------- |
| **요구사항 분류**     | 제약사항            |
| **요구사항 고유번호** | COR-003             |
| **요구사항 명칭**     | 개인정보보호법 준수 |

## 정의

개인정보보호법 준수

## 세부내용

- 개인정보를 처리함에 있어 『공공기관의 개인정보보호에 관한 법률』 등 관련법규 및 규정을 준수하여야 하며, 『개인정보 보호 가이드라인』을 적용하여야 함
  - 개인정보 열람시 접근 이력 기록 및 필요시 사유 입력
  - 개인정보 열람에 대한 이력관리 통계 또는 조회 화면 제공
  - 개인정보 중 고유식별정보에 대한 암호화
  - 관련된 정보의 접근, 열람, 저장 등의 모든 로그를 저장 관리
