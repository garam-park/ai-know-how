# 제주 스마트도시 데이터허브 시범솔루션 — 문서 구조

> 제주 스마트도시 데이터허브 시범솔루션 발굴사업 관련 문서 저장소
> 총 사업비: 19억원 (VAT 포함), 개발기간: 계약 후 8개월

## 디렉토리 구조

```
docs/
├── README.md              # 이 파일: 전체 구조 안내
├── originals/             # 원본 문서 (hwpx, xlsx 등)
│   ├── 제안요청서(...).hwpx
│   └── 설계내역서(...).xlsx
├── RFPs/                  # RFP 요구사항 분석 문서
│   ├── README.md          # RFP 요구사항 분류 코드 정리 (인덱스)
│   ├── COR/               # 준수 요구사항 (5개)
│   ├── DAR/               # 데이터 요구사항 (8개)
│   ├── ECR/               # 장비·컴퓨팅 자원 요구사항 (4개)
│   ├── PER/               # 성능 요구사항 (2개)
│   ├── PMR/               # 프로젝트 관리 요구사항 (5개)
│   ├── PSR/               # 프로젝트 지원 요구사항 (4개)
│   ├── QUR/               # 품질 요구사항 (4개)
│   ├── SFR/               # 시스템 기능 요구사항 (7개)
│   ├── SIR/               # 시스템 인터페이스 요구사항 (5개)
│   └── TER/               # 테스트 요구사항 (1개)
└── analysis/              # 분석 문서 (RFP 분석, 업무 배분 등)
    └── 20260331/
        ├── 업무배분_분석.md
        ├── RFP-할당회사.md
        └── 기타정보.md
```

## 폴더별 역할

| 폴더         | 용도                                                       |
| ------------ | ---------------------------------------------------------- |
| `originals/` | 원본 문서 (제안요청서, 설계내역서 등 비마크다운 파일)      |
| `RFPs/`      | 제안요청서(RFP)의 요구사항을 항목별로 분해한 마크다운 문서 |
| `analysis/`  | RFP 분석, 업무 배분, 요구사항 추적 등 분석 결과 문서       |

## 컨소시엄 구성

| 업체           | 역할                                             |
| -------------- | ------------------------------------------------ |
| 디토닉         | 빅데이터 데이터허브 플랫폼 및 LLM 메인 (PM 총괄) |
| 이노팸         | 대시보드 및 LLM 일부                             |
| 이노뎁         | 현장 장비 및 센서 설치 중심                      |
| 진우이노베이션 | 전기차 등 정보 연계 중심                         |
