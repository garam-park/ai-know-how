- [ ] 버블 클릭 시 사이드패널에 해당 클러스터 상세 표시
  - 핵심 키워드 5개 (태그 형태)

- [ ] 클러스터 수 슬라이더 변경 시 차트 재렌더링 시뮬레이션
- [ ] 병합/분할 이력 타임라인 항목 5개로 확충
