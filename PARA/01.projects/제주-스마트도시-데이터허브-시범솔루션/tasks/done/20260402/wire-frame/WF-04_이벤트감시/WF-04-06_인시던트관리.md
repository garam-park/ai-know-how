## 화면 구성

### 상단 상태별 카운트 카드
- **4개 카드** (1행 4열 그리드)
- 각 카드 구성:
  - **상태명**: "접수", "대응중", "완료", "종결"
  - **색상**: 각 상태별 전용 색상
    - 접수: 회색
    - 대응중: 파랑
    - 완료: 초록
    - 종결: 검정
  - **카운트**: 큰 숫자 (예: "3")
  - **추가 정보**: "진행중" 또는 "오늘" 등

### 뷰 전환 토글 (상단 우측)
- 버튼 2개: "칸반 보드", "테이블 뷰"
- 현재 선택 상태 강조

### 콘텐츠 영역 (2가지 뷰 선택)

#### 뷰 1: 칸반 보드 (기본)
- **레이아웃**: 4개 열 (Kanban 보드)
  - 열1: 접수 (회색 헤더)
  - 열2: 대응중 (파랑 헤더)
  - 열3: 완료 (초록 헤더)
  - 열4: 종결 (검정 헤더)

- **각 열의 카드**:
  - 이벤트 유형 아이콘 (좌상단)
  - 인시던트ID (예: "INC-2024-04-001", 소형 폰트)
  - 인시던트 유형 (예: "화재경보", 중간 폰트)
  - 주차장명 (작은 폰트, 회색)
  - 발생 시각 (작은 폰트, 회색)
  - 담당자 아바타 또는 이름 (좌하단)
  - 심각도 뱃지 (우상단, 색상)
  - 우측 아래: "상세" 클릭 가능 영역

- **카드 드래그**: 마우스로 열 간 드래그 가능 (상태 전환)

- **열별 높이**: 스크롤 가능 (많은 카드 시)

#### 뷰 2: 테이블 뷰
- **컬럼**:
  - 선택 체크박스 (다중 선택 가능)
  - 인시던트ID
  - 유형 (아이콘 + 텍스트)
  - 주차장명
  - 발생시각
  - 담당자
  - 심각도
  - 상태 (진행상황 바 또는 텍스트)
  - 액션 (드롭다운: 상태변경, 상세, 삭제 등)

- **정렬**: 컬럼 헤더 클릭으로 정렬 (오름/내림)

- **필터** (테이블 상단):
  - 상태 필터 (모두/접수/대응중/완료/종결)
  - 심각도 필터 (모두/주의/경고/위험/긴급)
  - 시간범위 필터 (오늘/7일/30일/전체)
  - 주차장 필터 (다중 선택)

- **검색**: 인시던트ID 또는 주차장명 입력

- **페이지네이션**: 10개 항목/페이지 또는 무한스크롤

### 좌측 패널 (테이블 뷰 선택 시): 상세 조회 사이드패널
- 선택된 행 클릭 시 우측에 사이드패널 표시
- **패널 내용**:
  - 인시던트 기본정보 (ID, 유형, 주차장, 발생시각)
  - 현재 상태 + 상태 변경 버튼
  - 심각도 뱃지
  - 담당자 정보 + 변경 버튼
  - 타임라인 (상태 변경 이력)
  - 비고/메모
  - "닫기" 버튼
