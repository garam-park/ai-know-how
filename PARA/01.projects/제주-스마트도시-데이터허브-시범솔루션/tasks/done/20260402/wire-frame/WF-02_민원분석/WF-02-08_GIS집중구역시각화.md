## 화면 구성

### 메인 영역 (전체 화면)
- **제주도 지도** (Leaflet/Mapbox)
  - 풀스크린 지도 뷰
  - 3개 레이어 토글 버튼 (우측 상단)

### 레이어 토글 (우측 상단)
- **체크박스 그룹**
  - 민원 집중구역 (체크 활성화, 빨강 폴리곤)
  - 집중단속구역 (체크 활성화, 파랑 폴리곤)
  - 수요부족지역 (체크 비활성화, 노랑 폴리곤)
  - 토글 시 지도 폴리곤 표시/숨김

### 지도 폴리곤 상호작용
- **폴리곤 클릭 팝업**
  - 구역명
  - 민원건수
  - 단속현황
  - 주차면수
  - 상세보기 버튼 → WF-03-02로 이동

### 우측 패널 (25%)
- **레이어별 범례**
  - 민원 집중구역: 빨강 설명
  - 집중단속구역: 파랑 설명
  - 수요부족지역: 노랑 설명

- **구역 목록 패널**
  - 정렬 옵션 (민원건수순/주차면수순/가나다순)
  - 필터 옵션 (레이어별 필터)
  - 스크롤 가능 목록
    - 각 항목: 구역명, 건수, 주차면수
    - 클릭 시 지도에서 해당 구역 강조/팝업

### 하단 영역 (20%)
- **선택 구역 상세 통계 카드**
  - 구역명: (선택된 구역)
  - 민원건수: X건
  - 주요 유형: (차트)
  - 단속현황: (처리율)
  - 주차면수: X면
