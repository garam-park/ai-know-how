## 화면 구성

### 헤더 영역 (상단)
- 제목: "공지-알림센터"
- 우측 버튼 영역:
  - "전체 읽음" 처리 버튼 (아이콘+텍스트)
  - 설정 버튼 (톱니바퀴 아이콘)

### 탭 네비게이션 (상단 - 3개 탭)
```
[전체] [시스템] [이벤트]
```
- 활성 탭 강조 표시 (언더라인 또는 배경색)
- 각 탭 우측에 미읽음 개수 배지 표시 (숫자)
  - 전체: 12건
  - 시스템: 5건
  - 이벤트: 7건

### 필터 영역 (탭 아래)
**필터 바** (한 줄 구성):
- 기간 선택: 드롭다운 (전체 / 1주일 / 1개월 / 3개월)
- 유형 선택: 드롭다운 (전체 / 공지 / 경고 / 정보 / 에러)
- 심각도 선택: 드롭다운 (전체 / Critical / Warning / Info)
- "초기화" 버튼 (필터 초기화)
- 검색창 (제목/내용 검색)

### 알림 목록 영역 (메인)
**알림 리스트** (스크롤 가능):

#### 각 알림 항목 구성:
- 좌측: 읽음/미읽음 표시 (동그란 점, 미읽음: 파란색, 읽음: 회색)
- 아이콘: 알림 유형별 아이콘 (🔔 공지, ⚠️ 경고, ℹ️ 정보, ❌ 에러)
- 제목: 알림 제목 (텍스트, 미읽음이면 볼드)
- 시각: 발생 시각 (우측, 작은 회색 텍스트, 상대 시간: "2시간 전")
- 요약: 알림 요약 텍스트 (2줄 제한, 말줄임)
- 심각도: 배지 표시 (Critical: 빨강, Warning: 주황, Info: 파랑)

#### 예시 항목:
```
● [⚠️] 주차장 C-04 센서 오류 감지                           12:32
  센서 신호 수신 두절, 수동 점검 필요                    Warning

● [🔔] 일일 보고서 작성 완료                               10:15
  지난 24시간 통계 보고서가 생성되었습니다              Info

○ [ℹ️] 시스템 점검 예정 공지                               09:00
  4월 5일 02:00~04:00 시스템 점검 예정                   Info
```

### 상세 내용 펼침 (아코디언)
- 각 항목 클릭 시 상세 내용 펼쳐짐
- 펼쳐진 상태:
  - 제목
  - 발생 시각 (YYYY-MM-DD HH:MM:SS)
  - 유형 / 심각도
  - 상세 내용 (본문, 여러 줄 가능)
  - 액션 버튼 (있는 경우: "조치하기", "상세보기" 등)
  - 닫기 버튼 (상세 내용 축소)

### 하단 영역
**페이지네이션**
- "이전" | 페이지 숫자 (1, 2, 3, ...) | "다음"
- 현재 페이지 강조 표시
- 한 페이지: 20건 표시

### 빈 상태 (알림 없을 때)
- 중앙에 아이콘 + 메시지
- "알림이 없습니다."

## 사용 컴포넌트

- WF-00-02: 공통 카드/리스트 컴포넌트
- WF-00-03: 탭, 드롭다운, 검색창
- WF-00-05: 알림/피드 컴포넌트
