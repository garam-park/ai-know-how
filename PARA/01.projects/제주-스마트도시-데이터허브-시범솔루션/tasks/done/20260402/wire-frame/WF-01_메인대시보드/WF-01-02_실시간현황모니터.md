## 화면 구성

### 레이아웃 구조 (2열 배치)

#### 좌측 영역 (약 70% 너비)
**제주도 지도 및 주차장 마커**
- 제주도 기본 지도 (오픈스트리트맵 또는 카카오맵 API)
- 공영주차장 9개소 마커 표시:
  - 위도/경도로 정확한 위치 표시
  - 마커 색상으로 상태 표시:
    - 🟢 정상 (녹색): 가용 구간 충분
    - 🟡 주의 (황색): 가용 구간 70% 이상
    - 🔴 경고 (적색): 가용 구간 70% 미만
  - 마커 클릭 시 팝업: 주차장명 + 가용률 + 현재 차량 수
  - 마커 호버 시 툴팁: 주차장 약칭 + 가용 구간

#### 우측 영역 (약 30% 너비)
**실시간 이벤트 피드 (타임라인)**
- 제목: "실시간 이벤트 피드"
- 상단에 자동 새로고침 토글, 수동 새로고침 버튼
- 타임라인 형태로 최근 20건 표시:
  - 시각(YYYY-MM-DD HH:MM:SS) | 이벤트 유형 아이콘 | 제목 | 심각도 표시
  - 사건별 배경색 구분 (Critical: 진빨강, Warning: 주황, Info: 파랑)
  - 스크롤 가능, 더 보기 버튼
- 우측에 실시간 업데이트 표시 (깜빡이는 점)

### 하단 영역
**주차장별 센서 상태 요약 카드** (2열 배치, 9개 카드)
- 각 카드 구성:
  - 주차장명 (예: "공항로 주차장", "삼도2로 주차장")
  - 주차장 ID/위치 (작은 텍스트)
  - 가용 구간 숫자: "145/200" (가용/전체)
  - 가용률 게이지: 145/200 = 72.5%
  - 센서 상태 표시: 정상/경고/오류 (아이콘 + 텍스트)
  - 마지막 업데이트 시각: "2분 전"
  - 클릭 시 상세 정보 페이지로 이동

## 사용 컴포넌트
- WF-00-01: 기본 레이아웃 (헤더, 사이드바)
- WF-00-02: 공통 카드 컴포넌트
- WF-00-04: 게이지 차트, 상태 표시
- WF-00-05: 이벤트/피드 컴포넌트
- 지도 라이브러리 (Leaflet.js 또는 Kakao Map API)
