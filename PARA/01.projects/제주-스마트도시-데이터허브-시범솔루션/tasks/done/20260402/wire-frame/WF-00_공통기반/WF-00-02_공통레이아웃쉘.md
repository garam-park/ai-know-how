## 화면 구성
전체 페이지의 기본 골격을 정의하는 공통 레이아웃 쉘. 헤더, 사이드바, 컨텐츠 영역, 푸터로 구성됨.

### GNB (Global Navigation Bar)
**위치**: 페이지 상단
**높이**: 64px (고정)
**구성 요소**:
- 로고 (좌측): 시스템 로고 + 프로젝트명 "제주 스마트도시 데이터허브"
- 메인 메뉴 (중앙): 7개 주요 메뉴 항목
  1. 대시보드
  2. 교통운영
  3. 주차관리
  4. 환경모니터링
  5. 에너지관리
  6. 민원관리
  7. 운영관리
- 알림 아이콘 (우측): 미읽음 알림 뱃지 표시
- 사용자 프로필 (우측): 사용자명, 역할 표시, 프로필 메뉴 (로그아웃 등)

### LNB (Local Navigation Bar)
**위치**: 페이지 좌측
**폭**: 기본 280px (접기 시 64px)
**구성 요소**:
- 현재 선택된 메뉴 그룹의 서브메뉴 목록
- 각 메뉴 항목에 아이콘 + 텍스트
- 메뉴 접기/펼치기 토글 버튼 (좌측 하단)
- 액티브 메뉴 항목 강조 표시
- 스크롤 가능 영역

### 컨텐츠 영역
**위치**: 메인 콘텐츠 영역 (GNB 하단, LNB 우측)
**구성**:
- Breadcrumb 네비게이션 (상단)
  - 형식: 홈 > 메뉴 > 서브메뉴 > 현재 페이지
- 페이지 제목 + 설명
- 메인 콘텐츠 (동적으로 변경되는 부분)

### 푸터
**위치**: 페이지 하단
**높이**: 60px (고정)
**구성 요소**:
- 저작권 표시: "Copyright 2024 Jeju Smart City Data Hub"
- 링크: 서비스 약관, 개인정보처리방침, 문의
- 버전 정보

### 테마 전환
**위치**: GNB 우측 사용자 프로필 메뉴 내
**기능**: 다크/라이트 테마 토글 (토글 버튼)
- 라이트 모드: 밝은 배경, 어두운 텍스트
- 다크 모드: 어두운 배경, 밝은 텍스트
