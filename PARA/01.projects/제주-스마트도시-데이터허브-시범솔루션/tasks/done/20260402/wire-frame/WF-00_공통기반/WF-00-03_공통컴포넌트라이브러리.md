## 화면 구성
재사용 가능한 공통 컴포넌트들을 한눈에 볼 수 있는 컴포넌트 카탈로그 페이지. 각 컴포넌트별로 상태 변형, 사용 가이드라인, 코드 예제를 포함.

## 컴포넌트 목록

### 1. KPI 카드 (Key Performance Indicator Card)
**설명**: 주요 지표를 시각화하는 카드
**구성**:
- 아이콘 (좌측 상단)
- 제목 (예: "일일 교통량")
- 수치 (큰 글자, Primary 색상)
- 증감률 (% 표시, 상승/하강 아이콘 포함)
- 선택 사항: 기간 또는 추가 정보
**상태**: 기본, 강조, 위험
**크기**: Small (200x120px), Medium (280x140px), Large (350x160px)

### 2. 데이터 테이블
**설명**: 데이터를 행/열 형식으로 표시
**기능**:
- 컬럼 헤더 클릭으로 정렬 (오름차순/내림차순)
- 체크박스 (행 선택, 전체 선택/해제)
- 페이징 (이전/다음, 페이지 번호, 페이지당 행 수 선택)
- 필터 기능 (각 컬럼별 필터)
- 행 클릭으로 상세보기
**상태**: 기본, 호버, 선택됨, 로딩
**기본 설정**: 페이지당 10개 항목

### 3. 모달 (Modal Dialog)
**유형**:
- 확인 모달: 제목 + 메시지 + 확인/취소 버튼
- 경고 모달: 제목 + 경고 메시지 + 위험 색상 + 확인/취소 버튼
- 폼 모달: 제목 + 입력 필드들 + 제출/취소 버튼
**공통 기능**:
- 배경 반투명 오버레이 (클릭으로 닫힘 또는 고정)
- 모달 애니메이션 (슬라이드인/페이드)
- ESC 키로 닫기

### 4. 토스트 알림 (Toast Notification)
**유형**: 4가지 상태
- 성공 (Success): 초록색, 체크 아이콘
- 정보 (Info): 파랑색, 정보 아이콘
- 경고 (Warning): 황색, 경고 아이콘
- 오류 (Error): 빨강색, X 아이콘
**위치**: 화면 우측 상단
**자동 닫기**: 3초 후 자동 사라짐
**기능**: 닫기 버튼 포함

### 5. 탭 패널 (Tab Panel)
**설명**: 탭을 통해 콘텐츠 영역을 전환
**구성**:
- 탭 헤더 (여러 탭 항목)
- 탭 콘텐츠 영역 (활성 탭만 표시)
**기능**:
- 마우스 클릭으로 탭 전환
- 키보드 (좌/우 화살표)로 탭 이동
- 액티브 탭 강조 표시 (언더라인 또는 배경색)

### 6. 필터 바 (Filter Bar)
**설명**: 데이터 필터링을 위한 바
**구성**:
- 날짜 범위 선택 (From ~ To 캘린더)
- 드롭다운 필터 (예: "상태", "카테고리")
- 검색 입력 필드
- 필터 초기화 버튼
- 필터 적용 버튼
**배치**: 가로 한 줄 (반응형으로 여러 줄 가능)

### 7. 뱃지 (Badge)
**설명**: 상태나 카테고리를 나타내는 작은 라벨
**상태별 색상 (6종)**:
- 기본 (Gray): `#6B7280`
- 정보 (Blue): `#2563EB`
- 성공 (Green): `#16A34A`
- 경고 (Yellow): `#EAB308`
- 위험 (Red): `#DC2626`
- 보라 (Purple): `#7C3AED`
**크기**: Small, Medium, Large

### 8. 프로그레스 바 (Progress Bar)
**설명**: 작업 진행 상황을 표시
**구성**:
- 배경 바 (전체 길이)
- 진행 바 (현재 진행률)
- 진행률 텍스트 (선택)
**색상**: Primary (진행 중), Success (완료), Warning (주의), Danger (오류)
**크기**: Small (4px), Medium (8px), Large (12px)

### 9. 스피너/로딩 (Loading Spinner)
**설명**: 데이터 로딩 중임을 표시
**유형**:
- 회전 스피너 (원형 로딩 인디케이터)
- 도트 스피너 (점이 움직이는 로딩)
- 바 스피너 (막대가 차오르는 로딩)
**크기**: Small (24px), Medium (40px), Large (64px)
**색상**: Primary 기본

### 10. 빈 상태 Placeholder
**설명**: 데이터가 없을 때 표시하는 UI
**구성**:
- 아이콘 (큰 회색 아이콘)
- 제목 (예: "데이터 없음")
- 설명 (예: "조건에 맞는 데이터가 없습니다")
- 선택 사항: 액션 버튼 (예: "초기화")
**사용 사례**: 검색 결과 없음, 빈 리스트

### 11. 페이지네이션 (Pagination)
**설명**: 페이지 이동 컨트롤
**구성**:
- 이전 버튼 (화살표)
- 페이지 번호 버튼들 (1, 2, 3, ...)
- 다음 버튼 (화살표)
- 선택 사항: "1-10 of 150" 같은 범위 텍스트
**기능**:
- 현재 페이지 강조
- 비활성 페이지 회색 처리

### 12. 드롭다운 메뉴 (Dropdown Menu)
**설명**: 클릭으로 옵션 목록을 보여주는 메뉴
**구성**:
- 트리거 버튼 (레이블 + 화살표 아이콘)
- 드롭다운 패널 (옵션 목록)
**기능**:
- 마우스 클릭/호버로 열기
- 바깥쪽 클릭으로 닫기
- 옵션 선택 후 자동 닫기
- 체크마크로 선택된 옵션 표시 (가능)

### 13. 툴팁 (Tooltip)
**설명**: 요소에 마우스를 올렸을 때 설명 텍스트 표시
**구성**:
- 트리거 요소 (아이콘, 텍스트 등)
- 팝오버 텍스트 (작은 박스)
**위치**: top, right, bottom, left (4가지)
**애니메이션**: 페이드인/페이드아웃
**자동 닫기**: 마우스 이동 시

### 14. 아코디언 (Accordion)
**설명**: 확장/축소 가능한 섹션
**구성**:
- 헤더 (클릭 가능, 제목 + 토글 아이콘)
- 콘텐츠 (헤더 클릭 시 확장/축소)
**기능**:
- 한 번에 한 개만 열기 (또는 여러 개 동시 열기 옵션)
- 부드러운 애니메이션 (슬라이드)
- 활성 헤더 강조 표시

## 더미 데이터
각 컴포넌트별로 대표적인 더미 데이터 포함:
- KPI 카드: 숫자, 증감률 샘플 (예: 15,234개, +5.2%)
- 테이블: 샘플 행 10개 (사용자명, 이메일, 역할, 등록일)
- 모달: 샘플 메시지 및 폼
- 드롭다운: 샘플 옵션 5-10개
- 필터 바: 샘플 날짜, 카테고리
