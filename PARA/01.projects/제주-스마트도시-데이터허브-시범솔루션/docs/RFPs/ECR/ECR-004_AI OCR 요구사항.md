## 정의

AI OCR 세부 규격

## 세부내용

- 품목 : OCR 솔루션
- 수량 : 1식
- 주요기능
  - 스캔 PDF, 이미지(ex. JPG,PNG) 등 비정형 문서를 OCR로 인식하여 텍스트 추출
  - 한글 문서(민원내용, 주소, 성명, 지명 등)에 대해 높은 인식률을 제공해야 하며, 숫자·영문 혼합 문서도 지원
  - 지정한 항목을 필드 단위(Key-Value)로 자동 추출(Structured OCR) 지원
  - 테이블 기반 서식 문서의 셀 구조를 인식하여 표 형태로 추출 지원
  - 웹페이지에서 첨부된 PDF/이미지를 다운로드 → OCR 인식 → 텍스트 추출 → 데이터 저장까지 RPA와 통합 연계가 가능한 기능 지원
  - 문서의 구조, 템플릿에 따라 인식률 향상을 위한 자체 학습 기능을 제공
