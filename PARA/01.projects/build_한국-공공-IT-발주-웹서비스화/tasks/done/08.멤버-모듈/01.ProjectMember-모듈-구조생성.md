# ProjectMember 모듈 구조생성

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~15분 |
| AI 에이전트 | ~5분 |

## What (무엇을)
`apps/backend/src/modules/project-member/` 디렉토리를 생성하고 NestJS 모듈 구조를 세팅합니다. ProjectMember 모듈의 기본 프레임워크(module, controller, service, DTO)를 완성하고 AppModule에 등록합니다.

## Why (왜)
프로젝트에 인력을 배정하고 관리하기 위한 기반입니다. 멤버 초대, 역할 및 투입비율 관리, 중복투입 감지 등 모든 인력 관리 기능의 토대입니다.

## 선행 작업
- `07.컨소시엄사-모듈/02.컨소시엘사-등록-API.md`

  import { PrismaService } from '@/prisma/prisma.service';
  import { ProjectMemberController } from './project-member.controller';
  import { ProjectMemberService } from './project-member.service';

  @Module({
    controllers: [ProjectMemberController],
    providers: [ProjectMemberService, PrismaService],
    exports: [ProjectMemberService],

  export class ProjectMemberModule {}

### Step 5: dto/ 폴더 및 기본 DTO 작성
- 대상 파일: `apps/backend/src/modules/project-member/dto/project-member.dto.ts`
- 작업 내용:
  ```typescript
  import { UserDto } from '@/modules/user/dto/user.dto';
  import { RoleDto } from '@/modules/role/dto/role.dto';

  export class ProjectMemberDto {
    id: string;
    projectId: string;
    userId: string;
    projectCompanyId: string;
    roleId: string;
    inputRate: number;
    createdAt: Date;
    updatedAt: Date;
  }

  export class ProjectMemberDetailDto extends ProjectMemberDto {
    user: UserDto;
    company: { id: string; name: string };
    role: RoleDto;
  }

  export class InviteMemberDto {
    email: string;
    projectCompanyId: string;
    roleId: string;
    inputRate?: number;
  }

  export class UpdateMemberDto {
    roleId?: string;
    inputRate?: number;
  }

  export class MemberOverlapDto {
    user: UserDto;
    projects: {
      project: { id: string; name: string };
      inputRate: number;
      startDate: Date;
      endDate: Date;
    }[];
    totalInputRate: number;
  }
  ```

## 참고
- ProjectMemberDetailDto: 멤버 상세 조회 시 user, company, role 정보 포함
- @@unique([projectId, userId]): 같은 프로젝트에 동일 사용자 중복 불가
- inputRate: 기본값 100 (단위: %)
- MemberOverlapDto: 중복투입 감지 시 사용
