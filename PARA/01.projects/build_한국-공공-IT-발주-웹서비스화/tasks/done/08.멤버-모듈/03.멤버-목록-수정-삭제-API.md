# 멤버 목록, 수정, 삭제 API

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~30분 |
| AI 에이전트 | ~10분 |

## What (무엇을)
GET `/api/v1/projects/:projectId/members` (목록), PATCH `/api/v1/projects/:projectId/members/:memberId` (수정), DELETE `/api/v1/projects/:projectId/members/:memberId` (삭제) 엔드포인트를 구현합니다. 멤버 조회, 역할/투입비율 변경, 멤버 제거를 담당합니다.

## Why (왜)
프로젝트의 모든 참여 인원을 조회하고 관리합니다. 권한에 따라 조회 범위를 제한하고(Scope), 역할과 투입비율을 유연하게 변경할 수 있어야 합니다.

### Step 1: ProjectMemberService에 findMany 메서드 구현
- 대상 파일: `apps/backend/src/modules/project-member/project-member.service.ts`
- 작업 내용:
  ```typescript
  async findMany(
    projectId: string,
    scope: 'ALL' | 'OWN_COMPANY',
    currentUserCompanyId?: string,
    limit = 10,
    offset = 0,
  ): Promise<{ members: ProjectMemberDetailDto[]; totalCount: number }> {
    // 프로젝트 존재 확인
    const project = await this.prisma.project.findFirst({
      where: { id: projectId, deletedAt: null },
    });

    if (!project) {
      throw new NotFoundException({
        code: 404000,
        message: '프로젝트를 찾을 수 없습니다.',
      });
    }

    // Scope에 따른 where 조건
    const where: any = { projectId, deletedAt: null };
    if (scope === 'OWN_COMPANY') {
      where.projectCompany = { companyId: currentUserCompanyId };
    }

    const [members, totalCount] = await Promise.all([
      this.prisma.projectMember.findMany({
        where,
        include: {
          user: true,
          projectCompany: { include: { company: true } },
          role: { include: { permissions: true } },
        },
        take: limit,
        skip: offset,
        orderBy: { createdAt: 'desc' },
      }),
      this.prisma.projectMember.count({ where }),
    ]);

    return {
      members: members.map((m) => ({
        id: m.id,
        projectId: m.projectId,
        userId: m.userId,
        projectCompanyId: m.projectCompanyId,
        roleId: m.roleId,
        inputRate: m.inputRate,
        createdAt: m.createdAt,
        updatedAt: m.updatedAt,
        user: {
          id: m.user.id,
          email: m.user.email,
          name: m.user.name,
        },
        company: {
          id: m.projectCompany.company.id,
          name: m.projectCompany.company.name,
        },
        role: {
          id: m.role.id,
          name: m.role.name,
          code: m.role.code,
        },
      })),
      totalCount,
    };
  }
  ```

### Step 2: ProjectMemberService에 update 메서드 구현
- 대상 파일: `apps/backend/src/modules/project-member/project-member.service.ts`
- 작업 내용:
  ```typescript
  async update(
    projectId: string,
    memberId: string,
    dto: UpdateMemberDto,
  ): Promise<ProjectMemberDetailDto> {
    // 멤버 존재 확인
    const member = await this.prisma.projectMember.findFirst({
      where: { id: memberId, projectId, deletedAt: null },
    });

    if (!member) {
      throw new NotFoundException({
        code: 404000,
        message: '멤버를 찾을 수 없습니다.',
      });
    }

    // roleId 변경 시 Role 유효성 확인
    if (dto.roleId) {
      const role = await this.prisma.role.findFirst({
        where: { id: dto.roleId, deletedAt: null },
      });

      if (!role) {
        throw new NotFoundException({
          code: 404000,
          message: '역할을 찾을 수 없습니다.',
        });
      }
    }

    // inputRate 범위 검증
    if (dto.inputRate !== undefined && (dto.inputRate < 0 || dto.inputRate > 100)) {
      throw new BadRequestException({
        code: 400000,
        message: '투입비율은 0~100 범위입니다.',
      });
    }

    const updated = await this.prisma.projectMember.update({
      where: { id: memberId },
      data: {
        ...(dto.roleId && { roleId: dto.roleId }),
        ...(dto.inputRate !== undefined && { inputRate: dto.inputRate }),
      },
      include: {
        user: true,
        projectCompany: { include: { company: true } },
        role: { include: { permissions: true } },
      },
    });

    return {
      id: updated.id,
      projectId: updated.projectId,
      userId: updated.userId,
      projectCompanyId: updated.projectCompanyId,
      roleId: updated.roleId,
      inputRate: updated.inputRate,
      createdAt: updated.createdAt,
      updatedAt: updated.updatedAt,
      user: {
        id: updated.user.id,
        email: updated.user.email,
        name: updated.user.name,
      },
      company: {
        id: updated.projectCompany.company.id,
        name: updated.projectCompany.company.name,
      },
      role: {
        id: updated.role.id,
        name: updated.role.name,
        code: updated.role.code,
      },
    };
  }
  ```

### Step 3: ProjectMemberService에 delete 메서드 구현
- 대상 파일: `apps/backend/src/modules/project-member/project-member.service.ts`
- 작업 내용:
  ```typescript
  async delete(projectId: string, memberId: string): Promise<void> {
    // 멤버 존재 확인
    const member = await this.prisma.projectMember.findFirst({
      where: { id: memberId, projectId, deletedAt: null },
    });

    if (!member) {
      throw new NotFoundException({
        code: 404000,
        message: '멤버를 찾을 수 없습니다.',
      });
    }

    // Soft delete
    await this.prisma.projectMember.update({
      where: { id: memberId },
      data: { deletedAt: new Date() },
    });
  }
  ```

### Step 4: ProjectMemberController에 엔드포인트 추가
- 대상 파일: `apps/backend/src/modules/project-member/project-member.controller.ts`
- 작업 내용:
  ```typescript
  import { Get, Patch, Delete, Param, Query, Body, UseGuards } from '@nestjs/common';
  import { JwtAuthGuard } from '@/auth/guards/jwt-auth.guard';
  import { PermissionGuard } from '@/auth/guards/permission.guard';
  import { RequirePermission } from '@/common/decorators/require-permission.decorator';
  import { CurrentUser } from '@/common/decorators/current-user.decorator';

  @Get()
  @UseGuards(JwtAuthGuard, PermissionGuard)
  @RequirePermission('MEMBER', 'VIEW')
  async findMany(
    @Param('projectId') projectId: string,
    @Query('scope') scope: 'ALL' | 'OWN_COMPANY' = 'ALL',
    @Query('limit') limit?: number,
    @Query('offset') offset?: number,
    @CurrentUser() user: any,
  ): Promise<any> {
    const result = await this.projectMemberService.findMany(
      projectId,
      scope,
      scope === 'OWN_COMPANY' ? user.companyId : undefined,
      limit ? parseInt(limit as any, 10) : 10,
      offset ? parseInt(offset as any, 10) : 0,
    );
    return {
      code: 200000,
      result,
    };
  }

  @Patch(':memberId')
  @UseGuards(JwtAuthGuard, PermissionGuard)
  @RequirePermission('MEMBER', 'PERMISSION')
  async update(
    @Param('projectId') projectId: string,
    @Param('memberId') memberId: string,
    @Body() dto: UpdateMemberDto,
  ): Promise<any> {
    const member = await this.projectMemberService.update(
      projectId,
      memberId,
      dto,
    );
    return {
      code: 200000,
      result: { member },
    };
  }

  @Delete(':memberId')
  @UseGuards(JwtAuthGuard, PermissionGuard)
  @RequirePermission('MEMBER', 'REMOVE')
  async delete(
    @Param('projectId') projectId: string,
    @Param('memberId') memberId: string,
  ): Promise<{
    code: number;
    message: string;
  }> {
    await this.projectMemberService.delete(projectId, memberId);
    return {
      code: 200000,
      message: '멤버가 제거되었습니다.',
    };
  }
  ```

- Soft delete: 멤버 삭제 시 데이터 추적 가능
