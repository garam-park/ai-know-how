# 프로젝트 목록 및 상세 조회 API

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~30분 |
| AI 에이전트 | ~10분 |

## What (무엇을)
GET `/api/v1/projects` (내가 속한 프로젝트 목록)과 GET `/api/v1/projects/:id` (프로젝트 상세)를 구현합니다. 사용자는 자신이 참여한 프로젝트만 조회 가능하며, 상세 조회에서 자신의 권한 목록도 반환합니다.

## Why (왜)
사용자는 본인이 관여하는 프로젝트만 볼 수 있어야 합니다. 목록에서 남은 기간, 진행률, 본인 역할 등을 빠르게 파악하고, 상세에서는 프론트 메뉴 제어를 위해 자신의 권한 목록이 필요합니다.

## 수행 가이드

### Step 1: ProjectService에 findMyProjects 메서드 구현
- 대상 파일: `apps/backend/src/modules/project/project.service.ts`
- 작업 내용:
  ```typescript
  async findMyProjects(
    userId: string,
    limit = 10,
    offset = 0,
  ): Promise<{ projects: ProjectSummaryDto[]; totalCount: number }> {
    // ProjectMember를 통해 사용자가 참여한 프로젝트 조회
    const projectMembers = await this.prisma.projectMember.findMany({
      where: { userId, deletedAt: null },
      include: {
        project: true,
        projectCompany: { include: { company: true } },
        role: true,
      },
      take: limit,
      skip: offset,
    });

    const projects = projectMembers.map((pm) => {
      const now = new Date();
      const daysLeft = Math.ceil(
        (pm.project.endDate.getTime() - now.getTime()) / (1000 * 60 * 60 * 24)
      );
      
      // 진행률 계산: (현재 - 시작) / (종료 - 시작) * 100
      const totalDays = (pm.project.endDate.getTime() - pm.project.startDate.getTime()) / (1000 * 60 * 60 * 24);
      const elapsedDays = (now.getTime() - pm.project.startDate.getTime()) / (1000 * 60 * 60 * 24);
      const progress = Math.min(100, Math.max(0, Math.round((elapsedDays / totalDays) * 100)));

      return {
        id: pm.project.id,
        name: pm.project.name,
        code: pm.project.code,
        startDate: pm.project.startDate,
        endDate: pm.project.endDate,
        daysLeft: daysLeft < 0 ? 0 : daysLeft,
        progress,
        myRole: pm.role.name,
        myCompany: pm.projectCompany.company.name,
      };
    });

    const totalCount = await this.prisma.projectMember.count({
      where: { userId, deletedAt: null },
    });

    return { projects, totalCount };
  }
  ```

### Step 2: ProjectService에 findById 및 권한 조회 메서드 구현
- 대상 파일: `apps/backend/src/modules/project/project.service.ts`
- 작업 내용:
  ```typescript
  async findById(id: string, userId: string): Promise<{
    project: ProjectDto;
    myPermissions: PermissionDto[];
  }> {
    // 사용자가 해당 프로젝트에 참여하는지 확인
    const membership = await this.prisma.projectMember.findFirst({
      where: { projectId: id, userId, deletedAt: null },
      include: { role: { include: { permissions: true } } },
    });

    if (!membership) {
      throw new ForbiddenException({
        code: 403000,
        message: '접근 권한이 없습니다.',
      });
    }

    const project = await this.prisma.project.findFirst({
      where: { id, deletedAt: null },
    });

    if (!project) {
      throw new NotFoundException({
        code: 404000,
        message: '프로젝트를 찾을 수 없습니다.',
      });
    }

    // 사용자의 권한 목록
    const myPermissions = membership.role.permissions.map((p) => ({
      resource: p.resource,
      action: p.action,
    }));

    return {
      project: {
        id: project.id,
        name: project.name,
        code: project.code,
        description: project.description,
        startDate: project.startDate,
        endDate: project.endDate,
        budget: project.budget,
        maxWbsDepth: project.maxWbsDepth,
        createdAt: project.createdAt,
        updatedAt: project.updatedAt,
      },
      myPermissions,
    };
  }
  ```

### Step 3: ProjectController에 GET 엔드포인트 추가
- 대상 파일: `apps/backend/src/modules/project/project.controller.ts`
- 작업 내용:
  ```typescript
  import { Get, Param, Query, UseGuards } from '@nestjs/common';
  import { JwtAuthGuard } from '@/auth/guards/jwt-auth.guard';
  import { CurrentUser } from '@/common/decorators/current-user.decorator';

  @Get()
  @UseGuards(JwtAuthGuard)
  async findMyProjects(
    @CurrentUser() user: any,
    @Query('limit') limit?: number,
    @Query('offset') offset?: number,
  ): Promise<any> {
    const result = await this.projectService.findMyProjects(
      user.id,
      limit ? parseInt(limit as any, 10) : 10,
      offset ? parseInt(offset as any, 10) : 0,
    );
    return {
      code: 200000,
      result,
    };
  }

  @Get(':id')
  @UseGuards(JwtAuthGuard)
  async findById(
    @Param('id') id: string,
    @CurrentUser() user: any,
  ): Promise<any> {
    const result = await this.projectService.findById(id, user.id);
    return {
      code: 200000,
      result,
    };
  }
  ```

## 참고
- daysLeft: 음수인 경우 0으로 처리 (종료된 프로젝트)
- progress: 0~100% 범위로 clamping
- 권한 목록(myPermissions): 프론트에서 메뉴/버튼 가시성 제어에 사용
- ForbiddenException: HTTP 403 상태 코드 (403000 에러 코드)
