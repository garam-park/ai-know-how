# WBS 수정/삭제 API

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~20분 |
| AI 에이전트 | ~8분 |

## What (무엇을)
PATCH /api/v1/projects/:id/wbs-nodes/:nodeId 와 DELETE /api/v1/projects/:id/wbs-nodes/:nodeId 엔드포인트를 구현합니다. 수정은 노드 정보를 변경하고, 삭제는 soft delete로 하위 노드도 함께 삭제합니다.

## Why (왜)
WBS 노드의 정보(제목, 날짜, 담당자 등)는 프로젝트 진행 중 변경되어야 하고, 노드 삭제 시 하위 모든 노드도 함께 삭제되어야 구조의 일관성을 유지할 수 있습니다. soft delete를 통해 데이터 감시 추적(audit trail)을 보존합니다.

### Step 2: WbsService에 수정/삭제 로직 추가
`apps/backend/src/modules/wbs/wbs.service.ts`:
```typescript
async updateWbsNode(
  nodeId: number,
  updateDto: UpdateWbsNodeDto,
) {
  // type 변경 불가 (CATEGORY ↔ TASK)
  const updateData: any = { ...updateDto };

  if (updateDto.startDate) {
    updateData.startDate = new Date(updateDto.startDate);
  }
  if (updateDto.endDate) {
    updateData.endDate = new Date(updateDto.endDate);
  }

  const node = await this.prisma.wbsNode.update({
    where: { id: nodeId },
    data: updateData,
    include: {
      assignee: true,
      projectCompany: true,
    },
  });

  return node;
}

async deleteWbsNode(nodeId: number) {
  // soft delete: deletedAt 설정
  const node = await this.prisma.wbsNode.findUnique({
    where: { id: nodeId },
  });

  if (!node) {
    throw new NotFoundException('Node not found');
  }

  // 하위 노드 재귀 soft delete
  await this.softDeleteRecursive(nodeId);

  // 삭제 후 상위 CATEGORY 진척률 재계산
  if (node.parentId) {
    await this.recalculateProgress(node.parentId);
  }

  return { id: nodeId, deletedAt: new Date() };
}

private async softDeleteRecursive(nodeId: number) {
  // 현재 노드 soft delete
  await this.prisma.wbsNode.update({
    where: { id: nodeId },
    data: { deletedAt: new Date() },
  });

  // 하위 노드 조회
  const children = await this.prisma.wbsNode.findMany({
    where: {
      parentId: nodeId,
      deletedAt: null,
    },
  });

  // 각 자식 노드 재귀 삭제
  for (const child of children) {
    await this.softDeleteRecursive(child.id);
  }
}
```

### Step 3: WbsController에 수정/삭제 엔드포인트 추가
`apps/backend/src/modules/wbs/wbs.controller.ts`:
```typescript
import { Patch, Delete, NotFoundException } from '@nestjs/common';

@Patch(':nodeId')
@RequirePermission(Resource.WBS, Action.EDIT)
async updateWbsNode(
  @Param('id') projectId: string,
  @Param('nodeId') nodeId: string,
  @Body() updateDto: UpdateWbsNodeDto,
) {
  const node = await this.wbsService.updateWbsNode(
    parseInt(nodeId),
    updateDto,
  );

  return {
    code: 200000,
    result: {
      node,
    },
  };
}

@Delete(':nodeId')
@RequirePermission(Resource.WBS, Action.EDIT)
async deleteWbsNode(
  @Param('id') projectId: string,
  @Param('nodeId') nodeId: string,
) {
  const result = await this.wbsService.deleteWbsNode(
    parseInt(nodeId),
  );

  return {
    code: 200000,
    result,
  };
}
```

## 참고

- 하위 노드 삭제는 재귀 함수로 구현
- 진척률 재계산은 다음 작업에서 구현하는 recalculateProgress 함수 호출
