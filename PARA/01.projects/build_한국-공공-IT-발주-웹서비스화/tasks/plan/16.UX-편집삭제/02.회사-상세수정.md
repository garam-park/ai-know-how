# 회사 상세/수정 페이지

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~35분 |
| AI 에이전트 | ~12분 |

## What (무엇을)
회사 상세 페이지(`/companies/:companyId`)를 신규 생성하고, 회사 정보(이름, 사업자번호, 주소, 전화번호) 수정 기능을 구현합니다. 회사 목록에서 클릭하면 상세 페이지로 이동합니다.

## Why (왜)
현재 회사 목록은 테이블 형태의 읽기 전용이며, 회사를 클릭해도 아무 동작이 없습니다. 사업자번호 오류 수정, 주소 변경 등 회사 정보 수정은 기본적인 마스터 데이터 관리 기능입니다.

## 선행 작업
- `15.UX-핵심플로우/01.회사-등록-폼.md` (회사가 존재해야 수정 가능)

## 수행 가이드

### Step 1: 회사 상세 페이지 컴포넌트 생성
- 대상 파일: `apps/frontend/src/pages/CompanyDetailPage.tsx` (신규)
- 작업 내용:
  ```typescript
  import { useEffect, useState } from 'react';
  import { useParams, useNavigate } from 'react-router-dom';
  import { api } from '../services/api';

  interface Company {
    id: number;
    name: string;
    bizNo?: string;
    address?: string;
    tel?: string;
    createdAt: string;
  }

  export default function CompanyDetailPage() {
    const { companyId } = useParams();
    const navigate = useNavigate();
    const [company, setCompany] = useState<Company | null>(null);
    const [editing, setEditing] = useState(false);
    const [form, setForm] = useState({ name: '', bizNo: '', address: '', tel: '' });
    const [error, setError] = useState('');

    useEffect(() => {
      if (!companyId) return;
      api.get<{ company: Company }>(`/companies/${companyId}`)
        .then((res) => {
          setCompany(res.result.company);
          setForm({
            name: res.result.company.name,
            bizNo: res.result.company.bizNo || '',
            address: res.result.company.address || '',
            tel: res.result.company.tel || '',
          });
        })
        .catch(() => navigate('/companies'));
    }, [companyId]);

    const handleSave = async () => {
      setError('');
      try {
        const res = await api.patch<{ company: Company }>(`/companies/${companyId}`, {
          name: form.name,
          bizNo: form.bizNo || undefined,
          address: form.address || undefined,
          tel: form.tel || undefined,
        });
        setCompany(res.result.company);
        setEditing(false);
      } catch (err: any) {
        setError(err?.message || '수정에 실패했습니다.');
      }
    };

    if (!company) return <div>로딩 중...</div>;

    return (
      <div style={{ maxWidth: 600, margin: '0 auto' }}>
        <h1 style={{ fontSize: 24, marginBottom: 24 }}>회사 상세</h1>
        {/* 편집 모드: input 필드 / 읽기 모드: 텍스트 표시 */}
        {/* 수정/저장/취소 버튼 */}
      </div>
    );
  }
  ```
