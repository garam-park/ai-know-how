# 컨소시엄 관리 화면

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~40분 |
| AI 에이전트 | ~15분 |

## What (무엇을)
/projects/:id 내 컨소시엄 탭에서 프로젝트에 참여하는 회사들을 트리 구조(조직도)로 표시합니다. 권한에 따라 회사 추가/수정/삭제 기능을 제공합니다.

## Why (왜)
컨소시엄 구조를 시각적으로 확인하고 관리하는 것이 프로젝트 운영의 핵심입니다.

### Step 1: 컨소시엄 API 서비스 생성
`apps/frontend/src/services/consortiumService.ts` 파일을 생성합니다:

```typescript
import apiClient from './api';

export interface CompanyNode {
  id: string;
  name: string;
  code?: string;
  parentId?: string | null;
  depth: number;
  children?: CompanyNode[];
}

export const consortiumService = {
  getCompaniesTree: async (projectId: string): Promise<CompanyNode[]> => {
    const response = await apiClient.get(
      `/projects/${projectId}/companies/tree`
    );
    return response.data;
  },

  addCompany: async (
    projectId: string,
    data: {
      name: string;
      code?: string;
      parentId?: string;
    }
  ): Promise<CompanyNode> => {
    const response = await apiClient.post(
      `/projects/${projectId}/companies`,
      data
    );
    return response.data;
  },

  updateCompany: async (
    projectId: string,
    companyId: string,
    data: { name?: string; code?: string }
  ): Promise<CompanyNode> => {
    const response = await apiClient.patch(
      `/projects/${projectId}/companies/${companyId}`,
      data
    );
    return response.data;
  },

  deleteCompany: async (
    projectId: string,
    companyId: string
  ): Promise<void> => {
    await apiClient.delete(`/projects/${projectId}/companies/${companyId}`);
  },
};
```

### Step 2: ConsortiumPage 컴포넌트 생성
`apps/frontend/src/pages/projects/ConsortiumPage.tsx` 파일을 생성합니다:

```typescript
import React, { useEffect, useState } from 'react';
import { useProjectContext } from '../../hooks/useProjectContext';
import { useAuth } from '../../hooks/useAuth';
import {
  consortiumService,
  CompanyNode,
} from '../../services/consortiumService';
import TreeNode from '../../components/TreeNode';
import Button from '../../components/Button';
import Modal from '../../components/Modal';
import Input from '../../components/Input';
import Loading from '../../components/Loading';
import Toast from '../../components/Toast';
import './ConsortiumPage.css';

const ConsortiumPage: React.FC = () => {
  const { project } = useProjectContext();
  const { user } = useAuth();

  const [companies, setCompanies] = useState<CompanyNode[]>([]);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);
  const [toast, setToast] = useState<{
    message: string;
    type: 'success' | 'error';
  } | null>(null);

  const [isAddModalOpen, setIsAddModalOpen] = useState(false);
  const [addFormData, setAddFormData] = useState({
    name: '',
    code: '',
    parentId: '',
  });

  const canEdit = user?.role === 'ADMIN' || user?.role === 'PM';

  useEffect(() => {
    fetchCompanies();
  }, [project.id]);

  const fetchCompanies = async () => {
    try {
      setLoading(true);
      const data = await consortiumService.getCompaniesTree(project.id);
      setCompanies(data);
    } catch (err) {
      setError(
        err instanceof Error ? err.message : '컨소시엄 정보를 불러올 수 없습니다'
      );
    } finally {
      setLoading(false);
    }
  };

  const handleAddCompany = async (e: React.FormEvent<HTMLFormElement>) => {
    e.preventDefault();

    if (!addFormData.name.trim()) {
      setToast({ message: '회사명은 필수입니다', type: 'error' });
      return;
    }

    try {
      await consortiumService.addCompany(project.id, {
        name: addFormData.name,
        code: addFormData.code || undefined,
        parentId: addFormData.parentId || undefined,
      });

      setToast({ message: '회사가 추가되었습니다', type: 'success' });
      setAddFormData({ name: '', code: '', parentId: '' });
      setIsAddModalOpen(false);
      await fetchCompanies();
    } catch (err) {
      setToast({
        message: err instanceof Error ? err.message : '회사 추가 실패',
        type: 'error',
      });
    }
  };

  const handleDeleteCompany = async (companyId: string) => {
    if (!window.confirm('이 회사를 삭제하시겠습니까?')) {
      return;
    }

    try {
      await consortiumService.deleteCompany(project.id, companyId);
      setToast({ message: '회사가 삭제되었습니다', type: 'success' });
      await fetchCompanies();
    } catch (err) {
      setToast({
        message: err instanceof Error ? err.message : '회사 삭제 실패',
        type: 'error',
      });
    }
  };

  if (loading) {
    return <Loading message="컨소시엄 로딩 중..." />;
  }

  if (error) {
    return <div className="consortium__error">{error}</div>;
  }

  return (
    <div className="consortium">
      <div className="consortium__header">
        <h2>컨소시엄 관리</h2>
        {canEdit && (
          <Button
            variant="primary"
            onClick={() => setIsAddModalOpen(true)}
          >
            + 회사 추가
          </Button>
        )}
      </div>

      {companies.length === 0 ? (
        <div className="consortium__empty">
          <p>등록된 회사가 없습니다</p>
        </div>
      ) : (
        <div className="consortium__tree">
          {companies.map((company) => (
            <TreeNode
              key={company.id}
              node={company}
              onDelete={canEdit ? handleDeleteCompany : undefined}
              depth={0}
            />
          ))}
        </div>
      )}

      <Modal
        isOpen={isAddModalOpen}
        title="회사 추가"
        onClose={() => setIsAddModalOpen(false)}
        size="medium"
      >
        <form onSubmit={handleAddCompany} className="consortium__form">
          <div className="consortium__field">
            <Input
              label="회사명 *"
              type="text"
              value={addFormData.name}
              onChange={(e) =>
                setAddFormData((prev) => ({
                  ...prev,
                  name: e.target.value,
                }))
              }
              placeholder="회사명을 입력하세요"
            />
          </div>

          <div className="consortium__field">
            <Input
              label="회사 코드"
              type="text"
              value={addFormData.code}
              onChange={(e) =>
                setAddFormData((prev) => ({
                  ...prev,
                  code: e.target.value,
                }))
              }
              placeholder="회사 코드를 입력하세요"
            />
          </div>

          <div className="consortium__field">
            <label className="consortium__label">상위 회사</label>
            <select
              value={addFormData.parentId}
              onChange={(e) =>
                setAddFormData((prev) => ({
                  ...prev,
                  parentId: e.target.value,
                }))
              }
              className="consortium__select"
            >
              <option value="">없음 (최상위)</option>
              {companies.map((company) => (
                <option key={company.id} value={company.id}>
                  {company.name}
                </option>
              ))}
            </select>
          </div>

          <div className="consortium__footer">
            <Button
              variant="secondary"
              onClick={() => setIsAddModalOpen(false)}
            >
              취소
            </Button>
            <Button variant="primary" type="submit">
              추가
            </Button>
          </div>
        </form>
      </Modal>

      {toast && (
        <Toast
          message={toast.message}
          type={toast.type}
          onClose={() => setToast(null)}
        />
      )}
    </div>
  );
};

export default ConsortiumPage;
```

### Step 3: TreeNode 컴포넌트 생성
`apps/frontend/src/components/TreeNode.tsx` 파일을 생성합니다:

```typescript
import React, { useState } from 'react';
import { CompanyNode } from '../services/consortiumService';
import './TreeNode.css';

interface TreeNodeProps {
  node: CompanyNode;
  depth: number;
  onDelete?: (nodeId: string) => void;
}

const TreeNode: React.FC<TreeNodeProps> = ({ node, depth, onDelete }) => {
  const [isExpanded, setIsExpanded] = useState(true);

  const hasChildren = node.children && node.children.length > 0;

  return (
    <div className="tree-node">
      <div className="tree-node__item" style={{ paddingLeft: `${depth * 24}px` }}>
        {hasChildren && (
          <button
            className={`tree-node__toggle ${
              isExpanded ? 'tree-node__toggle--expanded' : ''
            }`}
            onClick={() => setIsExpanded(!isExpanded)}
          >
            ▶
          </button>
        )}
        {!hasChildren && <span className="tree-node__toggle-spacer" />}

        <span className="tree-node__icon">
          {hasChildren ? '📁' : '🏢'}
        </span>

        <span className="tree-node__label">{node.name}</span>
        {node.code && <span className="tree-node__code">({node.code})</span>}

        {onDelete && (
          <button
            className="tree-node__delete"
            onClick={() => onDelete(node.id)}
            aria-label="삭제"
          >
            ✕
          </button>
        )}
      </div>

      {hasChildren && isExpanded && (
        <div className="tree-node__children">
          {node.children!.map((child) => (
            <TreeNode
              key={child.id}
              node={child}
              depth={depth + 1}
              onDelete={onDelete}
            />
          ))}
        </div>
      )}
    </div>
  );
};

export default TreeNode;
```

## 완료 기준
- [ ] 컨소시엄 조직도가 트리 구조로 표시
- [ ] 각 노드에 회사명과 코드 표시
- [ ] 상위 회사는 폴더 아이콘, 하위 회사 포함 시 확장/축소 가능
- [ ] PM/ADMIN만 "회사 추가" 버튼 표시
- [ ] 회사 추가 모달에서 상위 회사 선택 가능
- [ ] 회사 삭제 버튼 (확인 후 삭제)
- [ ] 회사 추가/삭제 후 트리 자동 새로고침
- [ ] 권한 없는 사용자는 CRUD 버튼 미표시

## 참고
- 트리 구조 렌더링: 재귀 컴포넌트로 depth 제어
- 확장/축소: useState로 상태 관리
- 아이콘: 이모지 또는 아이콘 라이브러리 사용
- 깊이별 들여쓰기: padding-left 계산
