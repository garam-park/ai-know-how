# 멤버 관리 화면

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~40분 |
| AI 에이전트 | ~15분 |

## What (무엇을)
/projects/:id 내 멤버 탭에서 프로젝트의 멤버 목록을 테이블로 표시합니다. 새 멤버 초대, 역할/투입비율 수정, 중복투입 경고를 포함합니다.

## Why (왜)
프로젝트 인력 관리는 성공적인 사업 추진의 핵심입니다. 중복투입 감지로 인력 배치 위험을 사전에 방지할 수 있습니다.

### Step 1: 멤버 API 서비스 생성
`apps/frontend/src/services/memberService.ts` 파일을 생성합니다:

```typescript
import apiClient from './api';

export interface MemberDto {
  id: string;
  email: string;
  name: string;
  company: string;
  role: string;
  allocationRate: number;
  joinedDate: string;
  isDuplicateAllocation?: boolean;
}

export interface DuplicateWarning {
  memberId: string;
  memberName: string;
  projects: {
    projectId: string;
    projectName: string;
    allocationRate: number;
  }[];
  totalAllocationRate: number;
}

export const memberService = {
  getMembers: async (projectId: string): Promise<MemberDto[]> => {
    const response = await apiClient.get(`/projects/${projectId}/members`);
    return response.data;
  },

  inviteMember: async (
    projectId: string,
    data: {
      email: string;
      companyId: string;
      role: string;
      allocationRate: number;
    }
  ): Promise<MemberDto> => {
    const response = await apiClient.post(
      `/projects/${projectId}/members/invite`,
      data
    );
    return response.data;
  },

  updateMember: async (
    projectId: string,
    memberId: string,
    data: {
      role?: string;
      allocationRate?: number;
    }
  ): Promise<MemberDto> => {
    const response = await apiClient.patch(
      `/projects/${projectId}/members/${memberId}`,
      data
    );
    return response.data;
  },

  removeMember: async (
    projectId: string,
    memberId: string
  ): Promise<void> => {
    await apiClient.delete(`/projects/${projectId}/members/${memberId}`);
  },

  checkDuplicateAllocation: async (
    projectId: string
  ): Promise<DuplicateWarning[]> => {
    const response = await apiClient.get(
      `/projects/${projectId}/members/duplicates`
    );
    return response.data;
  },
};
```

### Step 2: MembersPage 컴포넌트 생성
`apps/frontend/src/pages/projects/MembersPage.tsx` 파일을 생성합니다:

```typescript
import React, { useEffect, useState } from 'react';
import { useProjectContext } from '../../hooks/useProjectContext';
import { useAuth } from '../../hooks/useAuth';
import {
  memberService,
  MemberDto,
  DuplicateWarning,
} from '../../services/memberService';
import { consortiumService, CompanyNode } from '../../services/consortiumService';
import Table from '../../components/Table';
import Button from '../../components/Button';
import Modal from '../../components/Modal';
import Input from '../../components/Input';
import Badge from '../../components/Badge';
import Loading from '../../components/Loading';
import Toast from '../../components/Toast';
import './MembersPage.css';

const MembersPage: React.FC = () => {
  const { project } = useProjectContext();
  const { user } = useAuth();

  const [members, setMembers] = useState<MemberDto[]>([]);
  const [companies, setCompanies] = useState<CompanyNode[]>([]);
  const [duplicateWarnings, setDuplicateWarnings] = useState<DuplicateWarning[]>([]);
  const [loading, setLoading] = useState(true);
  const [toast, setToast] = useState<{
    message: string;
    type: 'success' | 'error';
  } | null>(null);

  const [isInviteModalOpen, setIsInviteModalOpen] = useState(false);
  const [inviteFormData, setInviteFormData] = useState({
    email: '',
    companyId: '',
    role: 'STAFF',
    allocationRate: 100,
  });

  const canManageMembers = user?.role === 'ADMIN' || user?.role === 'PM';

  useEffect(() => {
    fetchData();
  }, [project.id]);

  const fetchData = async () => {
    try {
      setLoading(true);
      const [membersData, companiesData, warningsData] = await Promise.all([
        memberService.getMembers(project.id),
        consortiumService.getCompaniesTree(project.id),
        memberService.checkDuplicateAllocation(project.id),
      ]);

      setMembers(membersData);
      setCompanies(companiesData);
      setDuplicateWarnings(warningsData);
    } catch (err) {
      setToast({
        message: err instanceof Error ? err.message : '멤버 정보를 불러올 수 없습니다',
        type: 'error',
      });
    } finally {
      setLoading(false);
    }
  };

  const handleInviteMember = async (e: React.FormEvent<HTMLFormElement>) => {
    e.preventDefault();

    if (!inviteFormData.email || !inviteFormData.companyId) {
      setToast({ message: '필수 필드를 입력하세요', type: 'error' });
      return;
    }

    try {
      await memberService.inviteMember(project.id, {
        email: inviteFormData.email,
        companyId: inviteFormData.companyId,
        role: inviteFormData.role,
        allocationRate: inviteFormData.allocationRate,
      });

      setToast({ message: '멤버 초대가 완료되었습니다', type: 'success' });
      setInviteFormData({
        email: '',
        companyId: '',
        role: 'STAFF',
        allocationRate: 100,
      });
      setIsInviteModalOpen(false);
      await fetchData();
    } catch (err) {
      setToast({
        message: err instanceof Error ? err.message : '멤버 초대 실패',
        type: 'error',
      });
    }
  };

  const handleRemoveMember = async (memberId: string) => {
    if (!window.confirm('이 멤버를 삭제하시겠습니까?')) {
      return;
    }

    try {
      await memberService.removeMember(project.id, memberId);
      setToast({ message: '멤버가 삭제되었습니다', type: 'success' });
      await fetchData();
    } catch (err) {
      setToast({
        message: err instanceof Error ? err.message : '멤버 삭제 실패',
        type: 'error',
      });
    }
  };

  if (loading) {
    return <Loading message="멤버 로딩 중..." />;
  }

  const columns = [
    {
      key: 'name' as const,
      label: '이름',
      render: (value: string, row: MemberDto) => (
        <div className="member-cell">
          <span>{value}</span>
          {row.isDuplicateAllocation && (
            <Badge variant="warning" size="small">
              중복투입
            </Badge>
          )}
        </div>
      ),
    },
    { key: 'company' as const, label: '소속 회사', width: '150px' },
    { key: 'role' as const, label: '역할', width: '100px' },
    {
      key: 'allocationRate' as const,
      label: '투입비율',
      width: '80px',
      render: (value: number) => `${value}%`,
    },
    {
      key: 'joinedDate' as const,
      label: '참여일',
      width: '100px',
      render: (value: string) => new Date(value).toLocaleDateString('ko-KR'),
    },
    {
      key: 'id' as const,
      label: '작업',
      width: '80px',
      render: (_: string, row: MemberDto) =>
        canManageMembers && (
          <button
            className="member-delete-button"
            onClick={() => handleRemoveMember(row.id)}
          >
            삭제
          </button>
        ),
    },
  ];

  return (
    <div className="members">
      <div className="members__header">
        <h2>멤버 관리</h2>
        {canManageMembers && (
          <Button
            variant="primary"
            onClick={() => setIsInviteModalOpen(true)}
          >
            + 멤버 초대
          </Button>
        )}
      </div>

      {duplicateWarnings.length > 0 && (
        <div className="members__warnings">
          <h3>중복투입 경고</h3>
          {duplicateWarnings.map((warning) => (
            <div key={warning.memberId} className="members__warning-item">
              <strong>{warning.memberName}</strong>: 총 투입비율{' '}
              {warning.totalAllocationRate}% (
              {warning.projects.map((p) => `${p.projectName} ${p.allocationRate}%`).join(', ')})
            </div>
          ))}
        </div>
      )}

      {members.length === 0 ? (
        <div className="members__empty">
          <p>아직 멤버가 없습니다</p>
        </div>
      ) : (
        <Table columns={columns} data={members} />
      )}

      <Modal
        isOpen={isInviteModalOpen}
        title="멤버 초대"
        onClose={() => setIsInviteModalOpen(false)}
        size="medium"
      >
        <form onSubmit={handleInviteMember} className="members__form">
          <div className="members__field">
            <Input
              label="이메일 *"
              type="email"
              value={inviteFormData.email}
              onChange={(e) =>
                setInviteFormData((prev) => ({
                  ...prev,
                  email: e.target.value,
                }))
              }
              placeholder="user@example.com"
            />
          </div>

          <div className="members__field">
            <label className="members__label">소속 회사 *</label>
            <select
              value={inviteFormData.companyId}
              onChange={(e) =>
                setInviteFormData((prev) => ({
                  ...prev,
                  companyId: e.target.value,
                }))
              }
              className="members__select"
            >
              <option value="">선택하세요</option>
              {companies.map((company) => (
                <option key={company.id} value={company.id}>
                  {company.name}
                </option>
              ))}
            </select>
          </div>

          <div className="members__field">
            <label className="members__label">역할</label>
            <select
              value={inviteFormData.role}
              onChange={(e) =>
                setInviteFormData((prev) => ({
                  ...prev,
                  role: e.target.value,
                }))
              }
              className="members__select"
            >
              <option value="STAFF">실무자</option>
              <option value="MANAGER">관리자</option>
              <option value="LEAD">리드</option>
            </select>
          </div>

          <div className="members__field">
            <Input
              label="투입비율 (%)"
              type="number"
              min="0"
              max="100"
              value={inviteFormData.allocationRate}
              onChange={(e) =>
                setInviteFormData((prev) => ({
                  ...prev,
                  allocationRate: parseInt(e.target.value, 10),
                }))
              }
            />
          </div>

          <div className="members__footer">
            <Button
              variant="secondary"
              onClick={() => setIsInviteModalOpen(false)}
            >
              취소
            </Button>
            <Button variant="primary" type="submit">
              초대
            </Button>
          </div>
        </form>
      </Modal>

      {toast && (
        <Toast
          message={toast.message}
          type={toast.type}
          onClose={() => setToast(null)}
        />
      )}
    </div>
  );
};

export default MembersPage;
```

- [ ] 중복투입 경고가 상단에 표시

- [ ] 멤버 삭제 버튼 (확인 후 삭제)
- [ ] 초대/삭제 후 목록 자동 새로고침
- [ ] 중복투입 정보 함께 새로고침

## 참고
- 투입비율: 0~100% 범위 입력 제한
- 중복투입: 여러 프로젝트에 100% 이상 투입 시 경고
- 테이블의 "작업" 열에는 권한 있는 사용자만 삭제 버튼 표시
- 멤버 수정(역할, 투입비율)은 추후 인라인 편집 기능으로 확장 가능
