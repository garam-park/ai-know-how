# 회사 등록 API

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~20분 |
| AI 에이전트 | ~8분 |

## What (무엇을)
POST `/api/v1/companies` 엔드포인트를 구현합니다. 새로운 회사를 시스템에 등록합니다. 회사명은 필수, 사업자번호, 주소, 전화번호는 선택 입력입니다.

## Why (왜)
새 참여사나 하청사를 프로젝트에 등록하기 전에 먼저 회사 마스터 데이터에 등록해야 합니다. 회사 등록 API는 회사 마스터 관리의 핵심 기능입니다.

## 수행 가이드

### Step 1: CreateCompanyDto 정의 확인 및 검증 추가
- 대상 파일: `apps/backend/src/modules/company/dto/company.dto.ts`
- 작업 내용:
  ```typescript
  import { IsString, IsOptional, MaxLength } from 'class-validator';

  export class CreateCompanyDto {
    @IsString()
    @MaxLength(255)
    name: string;

    @IsOptional()
    @IsString()
    @MaxLength(20)
    bizNo?: string;

    @IsOptional()
    @IsString()
    @MaxLength(500)
    address?: string;

    @IsOptional()
    @IsString()
    @MaxLength(20)
    tel?: string;
  }
  ```

### Step 2: CompanyService에 create 메서드 구현
- 대상 파일: `apps/backend/src/modules/company/company.service.ts`
- 작업 내용:
  ```typescript
  async create(dto: CreateCompanyDto): Promise<CompanyDto> {
    // 사업자번호 중복 체크
    if (dto.bizNo) {
      const existing = await this.prisma.company.findFirst({
        where: { bizNo: dto.bizNo, deletedAt: null },
      });
      if (existing) {
        throw new ConflictException({
          code: 409000,
          message: '이미 등록된 사업자번호입니다.',
        });
      }
    }

    const company = await this.prisma.company.create({
      data: {
        name: dto.name,
        bizNo: dto.bizNo,
        address: dto.address,
        tel: dto.tel,
      },
    });

    return {
      id: company.id,
      name: company.name,
      bizNo: company.bizNo,
      address: company.address,
      tel: company.tel,
      ownerId: company.ownerId,
      createdAt: company.createdAt,
      updatedAt: company.updatedAt,
    };
  }
  ```

### Step 3: CompanyController에 POST 엔드포인트 추가
- 대상 파일: `apps/backend/src/modules/company/company.controller.ts`
- 작업 내용:
  ```typescript
  import { Post, Body } from '@nestjs/common';
  import { CreateCompanyDto } from './dto/company.dto';

  @Post()
  async create(@Body() dto: CreateCompanyDto): Promise<{
    code: number;
    result: { company: CompanyDto };
  }> {
    const company = await this.companyService.create(dto);
    return {
      code: 201000,
      result: { company },
    };
  }
  ```
