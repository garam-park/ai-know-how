# 회사 등록 폼 추가

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~25분 |
| AI 에이전트 | ~10분 |

## What (무엇을)
`CompanyListPage`에 회사 등록 폼(모달 또는 인라인)을 추가하여 사용자가 직접 회사를 생성할 수 있도록 합니다. 회사명(필수), 사업자번호, 주소, 전화번호를 입력받습니다.

## Why (왜)
현재 회사 목록은 조회만 가능하고 등록 기능이 없습니다. 프로젝트 생성 전에 소속 회사를 등록해야 하는데 UI가 없어 핵심 플로우가 시작부터 막힙니다. 회사 등록은 전체 워크플로우의 첫 번째 단계입니다.

## 선행 작업
- 없음 (백엔드 POST `/api/v1/companies` 이미 구현 완료)

## 수행 가이드

### Step 1: 회사 등록 모달 컴포넌트 생성
- 대상 파일: `apps/frontend/src/components/CompanyCreateModal.tsx` (신규)
- 작업 내용:
  ```typescript
  import { useState } from 'react';
  import { api } from '../services/api';

  interface Props {
    open: boolean;
    onClose: () => void;
    onCreated: () => void;
  }

  export default function CompanyCreateModal({ open, onClose, onCreated }: Props) {
    const [form, setForm] = useState({ name: '', bizNo: '', address: '', tel: '' });
    const [error, setError] = useState('');
    const [submitting, setSubmitting] = useState(false);

    if (!open) return null;

    const handleSubmit = async (e: React.FormEvent) => {
      e.preventDefault();
      setError('');
      setSubmitting(true);
      try {
        await api.post('/companies', {
          name: form.name,
          bizNo: form.bizNo || undefined,
          address: form.address || undefined,
          tel: form.tel || undefined,
        });
        setForm({ name: '', bizNo: '', address: '', tel: '' });
        onCreated();
        onClose();
      } catch (err: any) {
        setError(err?.message || '회사 등록에 실패했습니다.');
      } finally {
        setSubmitting(false);
      }
    };

    const set = (key: string) => (e: React.ChangeEvent<HTMLInputElement>) =>
      setForm((prev) => ({ ...prev, [key]: e.target.value }));

    return (
      <div style={{ position: 'fixed', inset: 0, background: 'rgba(0,0,0,0.4)', display: 'flex', alignItems: 'center', justifyContent: 'center', zIndex: 1000 }}>
        <div style={{ background: '#fff', borderRadius: 8, padding: 24, width: 480, maxWidth: '90vw' }}>
          <h2 style={{ fontSize: 18, marginBottom: 16 }}>회사 등록</h2>
          <form onSubmit={handleSubmit}>
            {/* name(필수), bizNo, address, tel 입력 필드 */}
            {/* 등록 / 취소 버튼 */}
          </form>
        </div>
      </div>
    );
  }
  ```

### Step 2: CompanyListPage에 등록 버튼 및 모달 연동

  // JSX: 제목 옆에 버튼 추가
