# 빈 상태 가이드 메시지

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~25분 |
| AI 에이전트 | ~10분 |

## What (무엇을)
각 목록/탭에서 데이터가 없을 때 단순히 "없습니다" 대신, 다음 단계를 안내하는 가이드 메시지와 액션 버튼을 표시합니다. 사용자가 무엇을 해야 하는지 명확히 알 수 있도록 합니다.

## Why (왜)
현재 빈 상태 메시지는 "등록된 회사가 없습니다.", "멤버가 없습니다." 등 사실만 전달합니다. 신규 사용자가 시스템을 처음 접했을 때 어떤 순서로 무엇을 해야 하는지 알 수 없어 이탈합니다. 빈 상태 가이드는 온보딩의 핵심입니다.

## 선행 작업
- `15.UX-핵심플로우/01.회사-등록-폼.md` ~ `15.UX-핵심플로우/05.WBS-CRUD-UI.md` (액션 버튼이 동작하려면 해당 기능이 구현되어 있어야 함)

## 수행 가이드

### Step 1: EmptyState 공통 컴포넌트 생성
- 대상 파일: `apps/frontend/src/components/EmptyState.tsx` (신규)
- 작업 내용:
  ```typescript
  import { ReactNode } from 'react';

  interface Props {
    icon?: string;          // 이모지 or 아이콘 문자
    title: string;
    description: string;
    action?: ReactNode;     // CTA 버튼
  }

  export default function EmptyState({ icon = '📋', title, description, action }: Props) {
    return (
      <div style={{ padding: '48px 24px', textAlign: 'center', color: '#718096' }}>
        <div style={{ fontSize: 40, marginBottom: 12 }}>{icon}</div>
        <h3 style={{ fontSize: 16, fontWeight: 600, color: '#4a5568', marginBottom: 8 }}>{title}</h3>
        <p style={{ fontSize: 14, marginBottom: 16, maxWidth: 360, margin: '0 auto 16px' }}>{description}</p>
        {action}
      </div>
    );
  }
  ```

- 대상 파일: `apps/frontend/src/pages/ProjectListPage.tsx`

