## 2. 공통 이슈 패턴

### 2.1 네비게이션 링크 (30+ 파일)

대다수 파일에서 이전/다음 네비게이션에 파일 번호가 누락되어 있습니다.
- 현재: `이전: Supabase Studio 이해`
- 권장: `이전: 04-A-06 Supabase Studio 이해`

### 2.2 플레이스홀더 명확성 (15+ 파일)

도메인, IP, 비밀번호 등 사용자가 직접 입력해야 하는 값의 대입 방법이 불명확합니다.
- `yourdomain.com`, `your-password` 등에 대해 "이 값을 자신의 값으로 변경" 안내 부족

### 2.3 FE 비유 일관성

일부 파일은 FE 비유가 풍부하지만(04-A 그룹), 다른 파일에서는 비유 없이 서버 용어만 사용합니다.
대상 독자(FE 개발자)를 고려하면 모든 새 개념에 최소 1개의 FE 비유가 필요합니다.

### 2.4 실습 파일의 사전조건 누락

일부 실습 파일에서 필수 선행 조건(로그인, 패키지 설치 등)이 명시되지 않아 
독립적으로 읽을 때 중간에 막힐 수 있습니다.
