## 학습 가이드

### 10분 단위 파일 유형 정의

| 유형 | 설명 | 예상 소요 시간 |
|---|---|---|
| 개념 | 이론 읽기 + 구조 이해 | 10분 |
| 실습 | 직접 명령어 입력 & 결과 확인 | 15~20분 |
| 참고 | 트러블슈팅 / 체크리스트 | 필요할 때 |

### 권장 학습 방식

- **그룹 단위**로 목표 설정 ("이번 주는 05-C 백업/복구 그룹 완료")
- 실습 파일은 **반드시 서버에서 직접 실행**하며 진행
- 막히는 부분은 해당 `참고` 파일 먼저 확인 후 구글링

### FE 개발자를 위한 핵심 팁

| 상황 | 조언 |
|---|---|
| SSL 설정이 막막할 때 | Caddy 사용 시 `tls` 한 줄로 자동 HTTPS — Nginx 건너뛰어도 됨 |
| 비용 최소화 | Hetzner CX22 (월 ~₩6,000)로 충분 |
| 데이터 날릴까봐 겁날 때 | Phase 5 백업 그룹(05-C)을 가장 먼저 완성할 것 |
| 커뮤니티 도움 | Supabase 공식 Discord `#self-hosting` 채널 적극 활용 |
