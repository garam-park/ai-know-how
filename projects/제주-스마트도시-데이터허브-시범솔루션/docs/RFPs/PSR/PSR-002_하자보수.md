# PSR-002 하자보수

| 항목                  | 내용              |
| --------------------- | ----------------- |
| **요구사항 분류**     | 사업지원 요구사항 |
| **요구사항 고유번호** | PSR-002           |
| **요구사항 명칭**     | 하자보수          |

## 정의

하자보수

## 세부내용

- 관계법령에 따라 무상 하자보수기간은 본 사업이 종료된 날(준공일)로부터 1년으로 함
- 구축 사업 진행에 하자가 발생할 경우 대책을 수립하여 제출하고 신속하게 하자를 처리할 수 있는 방안을 제시하여야 함

## 산출정보

기술지원 계획서
