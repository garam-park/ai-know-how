# PSR-004 책임 및 안전대책

| 항목                  | 내용              |
| --------------------- | ----------------- |
| **요구사항 분류**     | 사업지원 요구사항 |
| **요구사항 고유번호** | PSR-004           |
| **요구사항 명칭**     | 책임 및 안전대책  |

## 정의

책임 및 안전대책에 관한 사항

## 세부내용

- 본 사업을 수행함에 있어 제안사의 책임으로 발생되는 모든 사고와 그로 인한 손해에 대해서는 제안사가 변상조치를 하여야 함
- 테스트 시 발주기관의 보고 또는 승인 없이 자체적으로 실시하여 실제 운영이 중단되어 발생하는 민원에 대해서는 전적으로 제안사가 책임짐.
- 제안사는 안전책임자를 지정하여 제반 안전규정 및 법규를 준수하여야 하며 안전사고가 발생하지 않도록 최선의 노력을 다 하여야 함.

## 산출정보

제약사항(COR)
