# SIR-001 사용자 인터페이스

| 항목                  | 내용                |
| --------------------- | ------------------- |
| **요구사항 분류**     | 인터페이스 요구사항 |
| **요구사항 고유번호** | SIR-001             |
| **요구사항 명칭**     | 사용자 인터페이스   |

## 정의

사용자 인터페이스 구성요건

## 세부내용

- 화면UI표준 및 설계 가이드를 수립함
  - 웹 표준을 준수하여 UI 설계를 하여야 함
  - UI표준 및 설계가이드는 웹 컨텐츠 접근성 지침과 웹 접근성 향상을 위한 국가표준기술 가이드라인을 준수
  - 직관적인 인터페이스를 제공하도록 화면 구성
  - 업무특성을 반영한 표준UI를 개발자에게 제공
  - 정보의 접근성, 가독성 등을 고려한 디자인을 적용하여야 함
- Non Active-X 기반의 Cross 브라우징을 지원하여야 함
- 정보기술에 대한 미숙련자도 쉽게 사용가능하고 직관적인 사용자 인터페이스를 구현해야 함
- 사용자의 수행 활동에 대한 확인 메시지 제공
  - 입력항목에 대해 필수, 선택 등 항목별 정확한 안내
  - 입력사항에 대한 체크 및 경고, 오류 메시지
  - 삭제, 저장 등 활동시 확인 메시지 등
- 메뉴 구성은 업무의 연속성 고려하여 구성되어야 함
- 디지털 정부서비스 UI/UX 가이드라인 및 전자정부 웹사이트 품질관리 가이드 준수
