## 정의

UI/UX 적용

## 세부내용

- 통일성 있는 디자인과 UI/UX로 구성
- 사용자가 쉽게 익숙해질 수 있도록 규칙적이고 일원화된 UI와 용어 사용
- 확장성, 안정성 등을 고려한 유연한 플랫폼 적용
- 웹기반으로 개발하여야 하며, 데이터 활용도를 높이기 위하여 사용자 편의성을 고려한 설계 및 구성
- 웹 접근성 준수
  - 인터넷 웹 콘텐츠 접근성 지침
  - 웹 접근성 향상을 위한 국가표준기술 가이드라인 등
