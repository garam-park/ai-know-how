# COR-001 기본규정 준수에 관한 사항

| 항목                  | 내용                      |
| --------------------- | ------------------------- |
| **요구사항 분류**     | 제약사항                  |
| **요구사항 고유번호** | COR-001                   |
| **요구사항 명칭**     | 기본규정 준수에 관한 사항 |

## 정의

기본규정 준수

## 세부내용

- 적용 지침을 고려하여 시스템 개발 준수
- 정보시스템 구축 운영 지침(행정안전부 고시)
- 전자정부 웹사이트 품질관리 지침(행정자치부 고시)
- 전자정부 웹 호환성 준수지침(행정자치부)
- 홈페이지 SW(웹)개발보안 가이드(행정자치부·한국인터넷진흥원)
- 웹 응용프로그램 개발보안 가이드(행정자치부)
- 한국형 웹 콘텐츠 접근성 지침
- 웹 접근성 향상을 위한 국가표준 기술 가이드 라인(한국지능정보사회진흥원)
- 웹 접근성 품질인증 심사가이드(한국지능정보사회진흥원)
- 공공기관 개인정보보호 기본지침
- 표준개인정보보호지침(행정자치부 예규)
- 전자정부 웹 표준 준수 지침(행정자치부)
- 데이터 품질관리지침 v2.1(한국데이터베이스진흥원)
- 전자정부 개발·운영자를 위한 소프트웨어 개발보안가이드(행정자치부)
- 행정기관 등 웹사이트 운영 가이드라인 (행정자치부)
- 웹 접근성, 웹 호환성, 웹 표준을 준수한 행정자치부웹사이트 운영 표준 개발 가이드
- 개발생산성 향상을 위해 재사용 가능한 공통요소를 도출하고 재사용을 위한 개발 표준가이드를 제공
