# RFP 요구사항 분류 코드 정리

> 제주 스마트도시 데이터허브 시범솔루션 발굴사업 제안요청서 기준
> 행정안전부 정보시스템 감리 기준에 따른 요구사항 분류 체계

## 코드 요약

| 코드 | 영문 풀네임                              | 한글명                     | 항목 수 |
| ---- | ---------------------------------------- | -------------------------- | ------- |
| COR  | Compliance Requirement                   | 준수 요구사항              | 5       |
| DAR  | Data Requirement                         | 데이터 요구사항            | 8       |
| ECR  | Equipment/Computing Resource Requirement | 장비·컴퓨팅 자원 요구사항  | 4       |
| PER  | Performance Requirement                  | 성능 요구사항              | 2       |
| PMR  | Project Management Requirement           | 프로젝트 관리 요구사항     | 5       |
| PSR  | Project Support Requirement              | 프로젝트 지원 요구사항     | 4       |
| QUR  | Quality Requirement                      | 품질 요구사항              | 4       |
| SFR  | System Functional Requirement            | 시스템 기능 요구사항       | 7       |
| SIR  | System Interface Requirement             | 시스템 인터페이스 요구사항 | 5       |
| TER  | Test Requirement                         | 테스트 요구사항            | 1       |

## 코드별 상세

### COR — 준수 요구사항 (Compliance Requirement)

법규, 규정, 표준 등 사업 수행 시 반드시 준수해야 하는 사항을 정의한다.

| 번호    | 항목명                                    |
| ------- | ----------------------------------------- |
| COR-001 | 기본규정 준수에 관한 사항                 |
| COR-002 | 지적 재산권 및 저작권 관련 법규·규정 준수 |
| COR-003 | 개인정보보호법 준수                       |
| COR-004 | SW사업정보 저장소 데이터 작성 및 제출     |
| COR-005 | 계약목적물의 지적재산권 귀속 및 보호      |

### DAR — 데이터 요구사항 (Data Requirement)

데이터 표준, 구조 설계, 검증, 관리 체계, 연계·수집·정제 등 데이터 관련 전반을 다룬다.

| 번호    | 항목명                             |
| ------- | ---------------------------------- |
| DAR-001 | 데이터 표준 수립 및 상위표준 준수  |
| DAR-002 | 데이터 일반사항                    |
| DAR-003 | 데이터 구조 설계 방안              |
| DAR-004 | 데이터 구조 검증                   |
| DAR-005 | 데이터 관리 체계 수립              |
| DAR-006 | 데이터 개방 및 메타데이터 관리체계 |
| DAR-007 | 데이터 연계/수집/정제              |
| DAR-008 | 스마트도시 데이터 표준 모델        |

### ECR — 장비·컴퓨팅 자원 요구사항 (Equipment/Computing Resource Requirement)

하드웨어 도입, 솔루션 구축 등 인프라 장비에 대한 요구사항을 정의한다.

| 번호    | 항목명                 |
| ------- | ---------------------- |
| ECR-001 | AI 서버(GPU) 요구사항  |
| ECR-002 | 망연계 시스템 요구사항 |
| ECR-003 | AI RPA 솔루션 요구사항 |
| ECR-004 | AI OCR 요구사항        |

### PER — 성능 요구사항 (Performance Requirement)

시스템의 성능 기준(응답시간, 처리량 등)을 정의한다.

| 번호    | 항목명           |
| ------- | ---------------- |
| PER-001 | 안정적 성능 보장 |
| PER-002 | 서비스 응답 시간 |

### PMR — 프로젝트 관리 요구사항 (Project Management Requirement)

사업 수행 체계, 보고, 산출물 관리, 보안 등 프로젝트 운영 전반에 관한 사항을 정의한다.

| 번호    | 항목명                                      |
| ------- | ------------------------------------------- |
| PMR-001 | 사업수행 및 관리                            |
| PMR-002 | 사업관리 및 현황보고                        |
| PMR-003 | 보고 및 산출물 관리                         |
| PMR-004 | 사업수행 보안대책                           |
| PMR-005 | 소프트웨어 개발보안(시큐어코딩) 가이드 준수 |

### PSR — 프로젝트 지원 요구사항 (Project Support Requirement)

기술지원, 교육, 하자보수, 보안 등 사업 완료 이후 지원에 관한 사항을 정의한다.

| 번호    | 항목명                    |
| ------- | ------------------------- |
| PSR-001 | 기술지원 및 교육훈련 계획 |
| PSR-002 | 하자보수                  |
| PSR-003 | 기밀보안 계획             |
| PSR-004 | 책임 및 안전대책          |

### QUR — 품질 요구사항 (Quality Requirement)

품질 보증, 기능 정확성, 장애 대응 등 시스템 품질 관리 기준을 정의한다.

| 번호    | 항목명           |
| ------- | ---------------- |
| QUR-001 | 품질 보증 방안   |
| QUR-002 | 품질 요구사항    |
| QUR-003 | 기능구현 정확성  |
| QUR-004 | 장애대응 및 처리 |

### SFR — 시스템 기능 요구사항 (System Functional Requirement)

시스템이 구현해야 할 핵심 기능을 정의한다. 본 사업의 주요 개발 범위에 해당한다.

| 번호    | 항목명                                   |
| ------- | ---------------------------------------- |
| SFR-001 | 데이터 수집·연계 체계 구축               |
| SFR-002 | AI 기반 민원데이터 전처리 모듈           |
| SFR-003 | AI 주차민원 분석 대응 기능               |
| SFR-004 | AI 기반 실시간 공영주차장 감시 대응 기능 |
| SFR-005 | 공영주차장 스마트 안전 AI 분석 솔루션    |
| SFR-006 | 데이터 카탈로그 모듈                     |
| SFR-007 | 오픈소스 방식 개발 및 확산               |

### SIR — 시스템 인터페이스 요구사항 (System Interface Requirement)

사용자 인터페이스, 웹 표준, UI/UX 등 시스템 인터페이스 관련 사항을 정의한다.

| 번호    | 항목명            |
| ------- | ----------------- |
| SIR-001 | 사용자 인터페이스 |
| SIR-002 | 사용자 편의성     |
| SIR-003 | 연계 공통사항     |
| SIR-004 | 웹 표준 및 호환   |
| SIR-005 | UI/UX             |

### TER — 테스트 요구사항 (Test Requirement)

시스템 검증 및 테스트 수행 기준을 정의한다.

| 번호    | 항목명          |
| ------- | --------------- |
| TER-001 | 테스트 공통사항 |
