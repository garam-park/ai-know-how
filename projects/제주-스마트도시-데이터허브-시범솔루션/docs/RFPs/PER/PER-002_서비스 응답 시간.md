# PER-002 서비스 응답 시간

| 항목                  | 내용             |
| --------------------- | ---------------- |
| **요구사항 분류**     | 성능 요구사항    |
| **요구사항 고유번호** | PER-002          |
| **요구사항 명칭**     | 서비스 응답 시간 |

## 정의

목표시스템의 처리속도 및 시간에 대한 사항

## 세부내용

- AI 서비스는 사용자의 질의로부터 시스템이 응답하는 시간은 5초 이내이어야 함
- 응답시간이 5초 이상 걸릴 경우, 사용자에게 지연을 알리는 메시지를 제공해야 함
