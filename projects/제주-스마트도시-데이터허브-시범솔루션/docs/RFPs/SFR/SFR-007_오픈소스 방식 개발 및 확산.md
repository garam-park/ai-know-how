# SFR-007 오픈소스 방식 개발 및 확산

| 항목                  | 내용                       |
| --------------------- | -------------------------- |
| **요구사항 분류**     | 기능 요구사항              |
| **요구사항 고유번호** | SFR-007                    |
| **요구사항 명칭**     | 오픈소스 방식 개발 및 확산 |

## 정의

확산성과 범용성에 대한 방안

## 세부내용

- 스마트도시 데이터허브 표준 API(NGSI-LD)를 기반으로 제주 시범솔루션의 모듈화 및 패키징하여 별도의 개발 없이 다양한 지자체에 적용 가능 할 수 있도록 구성해야 함.
- 시범솔루션은 NGSI-LD 기반으로 개발되며, 관련 코드 및 문서화는 오픈소스로 공개되어야 함.
  - AI 기반 민원 데이터 전처리 모듈
  - AI 기반 민원 분석 대응 솔루션 기능
  - AI 기반 실시간 공영주차장 감시 대응 기능
  - 공영주차장 스마트 안전 AI 분석 솔루션
  - 데이터 카탈로그 모듈
- GitHub 등 공용 저장소를 통해 소스코드, API 문서, 데이터모델 정의서를 오픈소스로 배포할 수 있어야 함.
- 기능별 모듈은 타 지자체에서 재사용할 수 있도록 라이선스, 버전, 활용 예제 등을 포함한 문서와 함께 제공되어야 함.
- 본 시범솔루션은 전북특별자치도와의 실증 협력 환경에서도 적용·검증이 가능하도록 구성되어야 함

## 산출정보

소스코드, 오픈소스 목록, API 명세서, 라이선스 명시서
