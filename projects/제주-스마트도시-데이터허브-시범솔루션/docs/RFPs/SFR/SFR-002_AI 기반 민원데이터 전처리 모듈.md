# SFR-002 AI 기반 민원데이터 전처리 모듈

| 항목                  | 내용                           |
| --------------------- | ------------------------------ |
| **요구사항 분류**     | 기능 요구사항                  |
| **요구사항 고유번호** | SFR-002                        |
| **요구사항 명칭**     | AI 기반 민원데이터 전처리 모듈 |

## 정의

AI기반 민원정보 수집·전처리 관리 요건

## 세부내용

- 모든 민원·행정 데이터는 NGSI-LD 기반의 표준 데이터 모델로 변환·저장할 수 있는 방안을 제시해야 함
- 데이터 변환·등록 과정은 ETL 프로세스를 기반으로 운영할 수 있도록 구체적 방법을 제시해야 함
- 생활 언어(사투리 등) 비표준 표현에 대한 표준화 처리 방안을 제시해야 함
- 개인정보(전화번호·이름 등) 자동 마스킹 처리 방안을 제시해야 함
- 제주도청 120만덕콜센터의 음성 민원 데이터를 자동 수집하고, STT 기술을 통해 텍스트로 변환·저장·분석할 수 있는 처리 구조를 제시하여야 함
- 음성 인식 성능은 WER, CER 등 정량 지표로 측정 가능하도록 설계되어야 하며, 측정 방법과 목표 수준을 제시하여야 함
- 지명, 도로명, 기관명, 날짜, 교통·안전 키워드 등 주요 개체명을 추출할 수 있는 기능을 제시해야 함
- 수집된 민원 데이터는 불법주정차, 충전구역 점유, 보행자 안전, 시설불편, 화재 위험, 주차장 안전 등 도메인 특화 분류체계를 기반으로 자동 분류할 수 있는 방안을 제시하여야 함
- 감성·감정 분석을 통해 민원 유형을 자동 탐지할 수 있는 방안을 제시해야 함
