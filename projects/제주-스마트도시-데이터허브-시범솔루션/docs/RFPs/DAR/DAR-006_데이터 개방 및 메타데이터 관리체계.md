# DAR-006 데이터 개방 및 메타데이터 관리체계

| 항목                  | 내용                               |
| --------------------- | ---------------------------------- |
| **요구사항 분류**     | 데이터 요구사항                    |
| **요구사항 고유번호** | DAR-006                            |
| **요구사항 명칭**     | 데이터 개방 및 메타데이터 관리체계 |

## 정의

개방데이터 서비스 연속성 확보 및 메타데이터 현행화

## 세부내용

- 메타데이터 현행화
  - 목표시스템의 메타데이터 표준 관리 항목이 데이터 카탈로그 모듈에 등록 및 현행화 되도록 지원하여야 함
- 데이터 개방
  - 목표시스템의 데이터가 공공데이터 포털 등을 통해 파일 또는 API 등의 형태로 개방될 수 있도록 지원하여야 함
