# DAR-008 스마트도시 데이터 표준 모델

| 항목                  | 내용                        |
| --------------------- | --------------------------- |
| **요구사항 분류**     | 데이터 요구사항             |
| **요구사항 고유번호** | DAR-008                     |
| **요구사항 명칭**     | 스마트도시 데이터 표준 모델 |

## 정의

스마트도시 데이터 표준 모델 마련 방안 요구사항

## 세부내용

- 수집·처리된 데이터는 NGSI-LD 기반 표준 데이터 모델에 따라 구조화되어야 하며, 데이터 카탈로그에 등록되어야 함
- 도시데이터 상호운용성을 위한 TTA 표준(TTAK.KO-10.1331-Part4/R1)을 반영해야 하며 전문기관과의 협의를 통해 표준화 추진 및 제안을 수행해야 함
- 표준화 과정에서 솔루션 구현 및 동작에 필수적인 데이터(Mandatory)와 부가 기능에 필요한 데이터(Optional)를 구분하여 반영할 수 있어야 함
- 국내 단체표준화기구(한국정보통신기술협회, 스마트도시표준화포럼 등)와의 협의를 통해 시범솔루션 데이터 모델 표준화를 수행해야 함
