```
Phase 1 (1~2주차) — 기반 + 핵심 대시보드
├── WF-00 공통 기반 (최우선)
├── WF-01 메인 대시보드
└── WF-02 민원 분석 대시보드 (일부)

Phase 2 (3~4주차) — 분석·감시 화면
├── WF-02 민원 분석 대시보드 (나머지)
├── WF-03 정책 시뮬레이션
└── WF-04 이벤트 감시 대시보드

Phase 3 (5~6주차) — LLM·AI·보고서
├── WF-05 LLM 분석 결과 시각화
├── WF-06 AI Assistant
└── WF-07 보고서 자동 생성

Phase 4 (7~8주차) — 데이터 관리·시스템
├── WF-08 데이터 수집·전처리 관리
├── WF-09 데이터 카탈로그
└── WF-10 시스템 관리
```

---

## 기술 스택 제안 (Mid-fi HTML PoC)

- **HTML/CSS**: Tailwind CSS (빠른 스타일링)
- **차트**: Chart.js 또는 Apache ECharts
- **지도**: Leaflet + OpenStreetMap
- **아이콘**: Lucide Icons 또는 Heroicons
- **프레임워크**: 순수 HTML 또는 React (JSX) — 컴포넌트 재사용 고려 시 React 추천
- **더미 데이터**: JSON 파일로 분리하여 관리
