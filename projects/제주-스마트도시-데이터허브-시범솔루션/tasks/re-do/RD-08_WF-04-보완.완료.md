## 수정 사항

### WF-04-04 센서별상태모니터

- [ ] RD-02의 `addClusteredMarkers()` 적용하여 센서 위치 마커 클러스터링
- [ ] 센서 상태별 마커 색상 분류 (정상=초록, 점검=노랑, 이상=빨강, 오프라인=회색)
- [ ] 실시간 상태 업데이트 시뮬레이션 (20초 간격, 랜덤 1~2개 센서 상태 변경)
- [ ] 마커 클릭 시 상세 팝업 (센서ID, 위치, 최근값, 최종점검일)
- [ ] 상태별 카운터 KPI 카드 (정상 42 / 점검 3 / 이상 1 / 오프라인 2)

### WF-04-08 통합관제연계

- [ ] 연계 시스템 상태 패널 보강 (4개 시스템: CCTV관제, 소방서, 경찰서, 제주시청)
  - 각 시스템별: 연결상태 아이콘(초록/빨강), 최근 동기화 시점, 데이터 건수
- [ ] 실시간 모니터링 시뮬레이션 (30초 간격, 동기화 시점 갱신)
- [ ] 연계 이벤트 로그 테이블 (최근 10건: 시점, 시스템명, 이벤트유형, 상태)
- [ ] 수동 동기화 버튼 ("지금 동기화" → 로딩 스피너 → 완료 토스트)
