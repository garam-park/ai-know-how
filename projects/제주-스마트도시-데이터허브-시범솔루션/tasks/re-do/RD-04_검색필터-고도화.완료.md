## 배경

검색 자동완성, 복합 조건 필터, 테이블 정렬/필터 기능이 부분적으로만 구현. WF-08~10 (17개 화면)이 공통적으로 이 패턴의 부족으로 PARTIAL 판정.

## 수정 사항

### components.js 공통 함수 추가

- [ ] **테이블 정렬 고도화** — `initTableSort(tableId)` 개선
  - 숫자/문자열/날짜 자동 감지 정렬
  - 정렬 방향 아이콘 표시 (▲/▼)
  - 3번째 클릭 시 정렬 초기화

- [ ] **테이블 필터 함수** — `initTableFilter(tableId, filterInputId)`
  - 텍스트 입력 → 전체 열 대상 실시간 검색
  - 매칭 행만 표시, 건수 카운터 업데이트
  - debounce 300ms 적용

- [ ] **검색 자동완성** — `initAutocomplete(inputId, suggestions)`
  - 드롭다운 추천 목록 (최대 8개)
  - 키보드 방향키 탐색, Enter 선택
  - 최근 검색 이력 5건 표시 (세션 기반)

- [ ] **복합 조건 필터 UI** — `initAdvancedFilter(containerId, fields)`
  - 필드별 조건 추가 (AND/OR 조합)
  - 조건 태그 형태 표시, X 버튼으로 개별 제거
  - "초기화" 버튼으로 전체 제거

### components.css 스타일 추가

- [ ] **자동완성 드롭다운 스타일** — `.autocomplete-dropdown`, `.autocomplete-item`
- [ ] **필터 태그 스타일** — `.filter-tag`, `.filter-tag-remove`
- [ ] **정렬 아이콘 스타일** — `.sort-icon`, `.sort-asc`, `.sort-desc`
