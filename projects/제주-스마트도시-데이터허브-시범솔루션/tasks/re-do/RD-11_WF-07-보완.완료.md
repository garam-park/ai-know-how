## 수정 사항

### WF-07-01 보고서목록-관리

- [ ] 일괄내보내기 기능 구현 (체크박스 선택 → "선택 내보내기" 버튼 활성화)
- [ ] 체크박스 전체선택/해제 토글
- [ ] 내보내기 클릭 시 "3건 내보내기 준비 중..." 토스트

### WF-07-02 보고서미리보기

- [ ] 우측 메타정보 패널 보강
  - 생성일, 대상기간, 데이터 소스 목록, 차트 수(3), 표 수(2), 총 페이지(12)
- [ ] 줌 슬라이더 동작 (50%~200%, 중앙 문서 영역 CSS transform scale 적용)
- [ ] 페이지 네비게이션 ("← 이전 / 3 of 12 / 다음 →") UI 동작

### WF-07-03 보고서편집-승인

- [ ] Rich Text 에디터 툴바 보강 (굵게/기울임/밑줄/목록/들여쓰기/되돌리기)
  - `document.execCommand` 기반 간단 구현
- [ ] "AI 재생성" 버튼 클릭 시 해당 섹션 로딩(2초) → 텍스트 교체 시뮬레이션
- [ ] 자동저장 표시기 ("자동 저장됨 14:32:15") 30초 간격 갱신
- [ ] 댓글 패널 "해결됨" 토글 (클릭 시 취소선 + 배경 회색)

### WF-07-04 보고서템플릿관리

- [ ] 카드 썸네일 영역에 간략한 섹션 구성 미리보기 (표지/요약/본문 박스 레이아웃)
- [ ] "편집" 클릭 시 모달 → 섹션 목록 드래그 대신 ↑/↓ 재정렬 버튼
- [ ] "복제" 클릭 시 "(사본)" 접미사 붙은 새 카드 추가 시뮬레이션
- [ ] "삭제" 클릭 시 확인 모달 → 카드 제거 애니메이션
