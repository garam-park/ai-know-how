## 화면 구성

### 지도 중심 레이아웃
- **좌측 (70%)**: 제주도 지도 + 히트맵 오버레이
  - 그리드별 색상 분포(히트맵)
    - 빨강: 초과수요 (공급 < 수요)
    - 주황: 부분수요 (약간 부족)
    - 노랑: 중간 (거의 균형)
    - 초록: 여유공급 (공급 > 수요)
  - 각 그리드 클릭 시 상세정보 로드
  - 줌/패닝 컨트롤

- **우측 (30%)**: 상세 정보 패널
  - 선택 그리드 기본정보
    - 그리드명/ID
    - 수요량 (일일 예상)
    - 공급량 (현재 주차면수)
    - 갭 (수요-공급)
    - 혼잡도 게이지 (0~100%)
  - 시간대별 추이 스파크라인 차트

- **상단 토글**: 레이어 전환 버튼
  - "수요도", "공급도", "혼잡도" 3개 토글
  - 선택된 지표에 따라 히트맵 색상 변경

- **하단**: 전체 그리드 요약 테이블
  - 컬럼: 순번, 그리드명, 수요량, 공급량, 갭, 혼잡도(%), 상태
  - 정렬 기능(컬럼 헤더 클릭)
  - 페이지네이션 (10개 항목/페이지)
