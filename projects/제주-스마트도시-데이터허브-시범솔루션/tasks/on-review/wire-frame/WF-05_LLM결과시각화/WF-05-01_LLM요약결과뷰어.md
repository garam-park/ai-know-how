## 화면 구성

### 상단 영역
- **요약 유형 탭**: 12개 탭 (문단요약, 문서요약, 다문서요약, 타임라인요약, 주제별요약, 부서별요약, 시민용요약, 긴급요약, 비교요약, 추세요약, 핵심어요약, 맞춤요약)
- **필터 영역**: 기간 필터(FROM/TO), 키워드 검색 바

### 메인 콘텐츠 영역
- **요약 결과 카드 목록** (그리드 또는 리스트)
  - 카드 구성:
    - 원문 미리보기 (썸네일 또는 텍스트 preview)
    - 요약문
    - 생성일시
    - 신뢰도 점수 (0-100%)
    - 작용 버튼 (상세보기, 내보내기, 공유)

### 상세 사이드패널
- 선택된 요약의 원문/요약문 비교
- 생성 설정 정보
