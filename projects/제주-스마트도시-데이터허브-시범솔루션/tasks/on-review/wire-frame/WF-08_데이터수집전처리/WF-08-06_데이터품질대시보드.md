## 화면 구성
### 상단 품질 지표 카드 (5개)
각 카드에 게이지 차트 포함:
| 지표명 | 범위 | 설명 |
|--------|------|------|
| 완전성 | 0~100% | 필수 필드 채움 정도 |
| 정확성 | 0~100% | 데이터 오류율 역계산 |
| 일관성 | 0~100% | 데이터 형식 일관성 |
| 적시성 | 0~100% | 시간 내 수집 비율 |
| 표준준수율 | 0~100% | NGSI-LD 표준 준수도 |

- 각 카드에 게이지 차트 시각화
- 현재 지표값 큰 폰트로 표시
- 목표값 또는 이전 기간 대비 변화율 표시

### 중앙 일별 품질 추이 라인차트
- X축: 날짜 (최근 30일)
- Y축: 지표값 (0~100%)
- 5개 라인 (각 지표별 색상)
- 범례: 5개 지표 표시
- 마우스 호버 시 날짜별 상세 수치 툴팁

### 하단 좌측 품질 이슈 목록
| 항목 | 설명 |
|------|------|
| 데이터소스 | 민원포털, 만덕콜센터 등 |
| 이슈유형 | 누락, 중복, 형식오류, 오류값 등 |
| 영향범위 | 건수 (예: 123건) |
| 심각도 | 높음(빨강), 중간(주황), 낮음(초록) 뱃지 |

- 테이블 정렬 가능 (심각도, 영향범위)
- 행 클릭 시 상세 정보 표시

### 하단 우측 NGSI-LD 표준 준수율 파이차트
- 준수(초록) vs 미준수(빨강) 비율 표시
- 백분율 레이블
- 클릭 시 준수 항목 상세 목록 표시
