## 화면 구성
### 상단 우측 데이터셋 등록 버튼
- "데이터셋 등록" 버튼 → WF-09-02 데이터셋 상세 편집 화면으로 이동

### 중앙 검색 영역
- 큰 검색 바 (가로 폭 전체 또는 80%)
- 자연어 검색 지원 (안내 텍스트)
- Placeholder: "찾고 싶은 데이터를 자연어로 검색하세요"
- 검색 아이콘 또는 "검색" 버튼
- 최근 검색어 드롭다운 (선택사항)

### 검색 바 아래 인기 태그 칩
- 태그 칩 나열 (예: 민원, 교통, 안전, 기상, 통계, 실시간)
- 클릭 시 해당 태그로 검색 수행
- 태그별 사용빈도 표시 (선택사항)

### 하단 3개 섹션

#### 1. 인기 데이터셋 카드
- 제목: "인기 데이터셋"
- 4개 카드 표시 (가로 스크롤 또는 고정)
- 각 카드 구성:
  - 데이터셋명
  - 제공기관
  - 간단한 설명 (1줄)
  - 다운로드 수 또는 조회수
  - 링크 → WF-09-02 상세 페이지

#### 2. 최근 등록 목록
- 제목: "최근 등록"
- 리스트 형식 (5건 표시)
- 각 항목: 데이터셋명, 제공기관, 등록일, 링크 → WF-09-02

#### 3. 카테고리 브라우징
- 제목: "카테고리별 탐색"
- 아코디언 형태 (3개 분류):
  1. **조직별** (아코디언)
     - 부서명 리스트
     - 클릭 시 해당 부서 데이터셋 필터링

  2. **분야별** (아코디언)
     - 분야명 리스트 (교통, 안전, 민원, 기상, 통계 등)
     - 클릭 시 해당 분야 검색

  3. **주제별** (아코디언)
     - 주제명 리스트 (실시간, 통계, 지형, 날씨 등)
     - 클릭 시 해당 주제 검색
