## 화면 구성

### 좌측 영역 (약 65%)
- **주차장 멀티뷰 그리드**: 3x3 그리드 (9개 주차장)
- 각 셀 구성:
  - 주차장 썸네일 이미지 (CCTV 스냅샷, 360x270px)
  - 주차장명 (하단 오버레이 텍스트)
  - 상태 뱃지 (좌상단 코너)
    - 색상: 정상(초록)/주의(노랑)/경고(주황)/긴급(빨강)
    - 텍스트: 상태명 + 활성 이벤트 수 (예: "주의 (2)")
  - 호버 시: 밝기 약간 증가, 상세보기 버튼 표시

- **클릭 동작**: 셀 클릭 시 WF-04-02(이벤트 상세 팝업) 호출

### 우측 영역 (약 35%)
- **실시간 이벤트 피드**
  - 제목: "실시간 이벤트 (최근 30건)"
  - 타임라인 형식 (수직 스크롤)
  - 각 이벤트 항목:
    - 좌측: 심각도 색상 점 (4px, 원형)
    - 중앙: 이벤트 정보
      - 시각 (상대시간 또는 절대시간, 예: "3분전" 또는 "14:25:30")
      - 이벤트명 (예: "비정상 음향 탐지")
      - 주차장명 (회색 텍스트)
      - 간단한 설명 (한 줄, 예: "신도심 A / 화재경보 의심")
    - 우측: 액션 버튼 (화살표 또는 "상세")

  - **색상 코드** (심각도):
    - 빨강: 긴급 (화재/안전 위협)
    - 주황: 위험 (비정상 사건)
    - 노랑: 경고 (주의 필요)
    - 초록: 정상 (복구/정상화)

  - 스크롤: 최대 30건 표시 (초과 시 스크롤 바)

### 하단 영역
- **전체 위험도 요약 바**
  - 레이블: "전체 위험도"
  - 9개 주차장별 위험도 세그먼트 바
    - 각 세그먼트: 주차장명 (작은 텍스트) + 위험도(%) 표시
    - 색상: 주차장 상태에 따라 (정상/주의/경고/긴급)
  - 전체 평균 위험도 (%)

### 화면 상단 (헤더 영역, WF-00-04 기반)
- 페이지 제목: "실시간 감시 콘솔"
- 새로고침 버튼 (실시간 업데이트 토글)
- 조회 기간 필터 (기본: 실시간, 선택: 최근 1시간/6시간/24시간)
