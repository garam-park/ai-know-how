## 화면 구성

### 상단 영역
- **기간 선택 옵션**
  - 버튼 그룹: 일 / 주 / 월 / 분기 / 연
  - 선택된 옵션에 따라 차트 X축 단위 변경

### 메인 영역 - 큰 라인차트
- **타이틀**: 기간별 민원 추세
- **라인 구성** (유형별 색상 구분)
  - 불법주차: 빨강
  - 보행안전: 초록
  - 시설불편: 파랑
  - 충전구역점유: 노랑
  - 화재위험: 주황
  - 주차안전: 보라
- **이상치 포인트** 하이라이트 (원형 마커로 표시)
- **범례**: 차트 상단 또는 우측

### 하단 좌측 (50%)
- **전년 동기 대비 비교 차트**
  - 막대 차트
  - 금년 vs 전년 나란히 비교
  - 범례: 금년(강조색), 전년(옅은색)

### 하단 우측 (50%)
- **요일별/시간대별 히트맵**
  - 가로: 요일 (월~일)
  - 세로: 시간대 (0시~23시)
  - 셀 색상: 민원 건수 기반 히트맵 색상
  - 셀 내 텍스트: 건수 숫자

### 통계 카드 영역
- **통계 요약** (4개 카드)
  - 평균: X건/일
  - 최대: X건/일
  - 최소: X건/일
  - 표준편차: X
