## 화면 구성

### 좌측 패널 (20%)
- **필터 영역**
  - 기간 선택 (시작일~종료일)
  - 민원유형 멀티셀렉트
  - 심각도 슬라이더 (1~5)
  - 적용/초기화 버튼

### 중앙 맵 영역 (80%)
- **제주도 지도** (Leaflet/Mapbox)
  - 전체 화면 지도 뷰
  - 민원 밀집도 히트맵 레이어 (색상 그래디언트)
  - 불법주차 집중구역 폴리곤 표시 (반투명 빨강)
  - 폴리곤 클릭 시 팝업 표시
    - 구역명
    - 민원 건수
    - 주요 민원 유형 및 비율
    - 상세보기 버튼

### 상단 우측
- **히트맵 범례**
  - 밀도 5단계 색상 표시 (보라→파랑→초록→노랑→빨강)
  - 각 단계별 민원건수 범위 표시

### 범례 정보
- 매우 낮음: 0~10건 (보라)
- 낮음: 11~50건 (파랑)
- 중간: 51~100건 (초록)
- 높음: 101~200건 (노랑)
- 매우 높음: 201+건 (빨강)
