## 화면 구성

### 패널 레이아웃
- **우측 사이드패널** (WF-06-01 챗 인터페이스 우측)
  - AI 응답과 동시 표시
  - 드래그하여 크기 조절 가능
  - 접기/펼치기 버튼

### 상단 헤더
- **패널 제목**: "근거 문서"
- **새로고침 버튼** (새로운 검색 실행)

### 문서 목록 영역
- **문서 카드** (리스트 형태)
  - 출처 유형 아이콘 (조례/정책문서/민원회신/통계자료)
  - 문서 제목
  - 출처 정보 (부서, 발행처)
  - 관련도 점수 바 (%)
  - 발행일
  - 클릭하여 상세보기 (미리보기 하단 팽창)

### 문서 미리보기 영역
- **선택된 문서의 본문 일부** (클릭된 카드)
  - 관련 부분 하이라이트 (노란색 배경)
  - 원문 인용 (따옴표)
  - 스크롤 가능

### 하단 액션 영역
- **"더 찾기" 버튼** (추가 검색)
  - 클릭 시 WF-05-04 검색 인터페이스로 이동
  - 또는 패널 내 추가 결과 로드

- **피드백 버튼 그룹**
  - "도움됨" 버튼 (좋아요 아이콘)
  - "안됨" 버튼 (싫어요 아이콘)
  - 피드백 제출 후 상태 표시 (체크 아이콘)

## 사용 컴포넌트
