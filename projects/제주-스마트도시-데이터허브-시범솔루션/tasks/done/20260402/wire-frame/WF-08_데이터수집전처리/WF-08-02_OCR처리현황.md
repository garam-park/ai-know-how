## 화면 구성
### 상단 KPI 카드 (4개)
| 카드명 | 내용 |
|--------|------|
| 일간처리건수 | 숫자 (예: 5,234건) |
| 평균신뢰도 | 백분율 (예: 94.2%) |
| 수동검증대기 | 숫자 (예: 128건) |
| 실패건수 | 숫자 (예: 42건) - 빨간색 강조 |

### 메인 처리 결과 목록 (테이블)
| 항목 | 설명 |
|------|------|
| 문서ID | 고유 ID (클릭 시 상세 미리보기) |
| 유형 | 스캔PDF / 이미지 / 팩스 (태그) |
| 인식일시 | YYYY-MM-DD HH:MM:SS |
| 신뢰도 | 0~100% 프로그레스 바 표시 |
| 상태 | 완료(초록), 검증대기(주황), 실패(빨강) 뱃지 |

- **신뢰도 낮은 건 하이라이트**: 신뢰도 70% 이하인 행의 배경색 연한 노란색
- 행 클릭 시 인식 결과 미리보기 패널 표시

### 인식 결과 미리보기 (우측 또는 모달)
- 좌측: 원본 이미지 또는 PDF 미리보기
- 우측: 추출 텍스트 (스크롤 가능)
- 좌우 비교 뷰로 동기화 스크롤 (선택사항)
- 신뢰도 낮은 영역 박스 강조

### 하단 필드별 인식률 통계
- 가로 바차트 또는 테이블
- 필드명별 인식 성공률 표시 (예: 문서제목 98%, 본문 92%, 날짜 88%)
