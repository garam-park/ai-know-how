## 화면 구성
### 상단 봇 상태 카드 (4개)
| 카드명 | 내용 |
|--------|------|
| 활성봇 수 | 숫자 (예: 6대) |
| 실행중 | 숫자 (예: 3대) |
| 대기 | 숫자 (예: 2대) |
| 에러 | 숫자 (예: 1대) - 빨간색 강조 |

### 중앙 스케줄 캘린더 뷰
- 캘린더 형식 (월별 또는 주별 뷰 선택 가능)
- 각 일자에 봇의 실행 일정 표시
- 간트차트 형태로 각 봇의 일정 바 표시
- 색상: 봇별로 다른 색상
- 예정/실행중/완료/실패 상태별 색상 구분

### 실행 로그 테이블
| 항목 | 설명 |
|------|------|
| 봇ID | BOT-001, BOT-002 등 |
| 대상사이트 | 웹사이트명 (예: 민원포털) |
| 시작시각 | YYYY-MM-DD HH:MM |
| 종료시각 | YYYY-MM-DD HH:MM |
| 수집건수 | 정수 |
| 상태 | 성공(초록), 부분성공(주황), 실패(빨강) 뱃지 |
| 소요시간 | HH:MM:SS |

- Captcha 발생 건 별도 표시: 행 좌측에 "🔐 CAPTCHA" 라벨
- 테이블 정렬/필터링 가능
- 페이지네이션 또는 무한 스크롤

### 상단 우측 수동 실행 버튼
- "봇 실행" 버튼 → 모달 (실행할 봇 선택 및 옵션 설정)
