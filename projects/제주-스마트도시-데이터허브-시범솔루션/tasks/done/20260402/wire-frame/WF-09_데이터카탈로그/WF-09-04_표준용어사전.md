## 화면 구성
### 상단 검색 및 인덱싱 영역
- 검색 바: 용어 검색 (자동완성 지원)
- 알파벳 인덱스: A ~ Z (클릭 시 해당 문자로 시작하는 용어로 스크롤)
- 가나다 인덱스: ㄱ ~ ㅎ (클릭 시 해당 문자로 시작하는 용어로 스크롤)
- "신규 용어 후보 등록" 버튼 → 등록 모달

### 용어 목록 테이블
| 항목 | 설명 |
|------|------|
| 용어명 | 한글명 (클릭 시 상세 사이드패널) |
| 영문명 | English name |
| 정의 | 용어의 의미 정의 |
| 동의어 | 유사 용어 태그 |
| 도메인 | 영역 분류 (교통, 안전, 민원 등) |
| 상태 | 승인(초록), 후보(주황), 폐기(회색) 뱃지 |

- 테이블 정렬 가능 (용어명, 상태)
- 필터링: 상태별 필터 (승인된 용어만, 후보 포함, 폐기된 용어 포함)
- 페이지네이션 또는 무한 스크롤

### 상세 사이드패널 (우측)
- 행 또는 용어명 클릭 시 표시
- 용어 상세 정보:
  - 용어명
  - 영문명
  - 정의 (여러 문단)
  - 동의어 목록
  - 도메인
  - 상태
  - 생성일 / 수정일

- **관련 데이터셋 링크**: "다음 데이터셋에서 사용됨"
  - 데이터셋명 리스트 (클릭 시 WF-09-02로 이동)

- **사용 빈도**: "이 용어는 X개 데이터셋, Y번 사용됨"

### 신규 용어 후보 등록 모달
- 폼 필드:
  - 용어명 * (Text)
  - 영문명 (Text)
  - 정의 * (Textarea)
  - 근거 * (Textarea, 예: "RFP SFR-006 참조")
  - 도메인 (Dropdown)

- "등록" 및 "취소" 버튼

### 관리자용 버튼 (관리자만 표시)
- 승인 / 반려 버튼 (사이드패널 하단)
- 반려 시 모달: 반려 사유 입력
