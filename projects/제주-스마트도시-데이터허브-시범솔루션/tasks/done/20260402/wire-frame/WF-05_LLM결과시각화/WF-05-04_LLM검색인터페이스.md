## 화면 구성

### 중앙 검색 영역
- **검색 바** (대형)
  - Placeholder: "민원 내용이나 정책을 자연어로 검색하세요"
  - 입력 필드
  - 검색 버튼 / 엔터 키 지원

- **검색 모드 토글**
  - 벡터검색 (의미 기반)
  - 키워드검색 (정확 매칭)
  - 하이브리드 (병합 검색)

### 좌측 사이드패널 (필터)
- **패싯 필터** (접을 수 있는 구조)
  - 유형 (문제, 제안, 칭찬, 민원 등)
  - 기간 (달력 또는 프리셋)
  - 부서 (체크박스)
  - 관련도 (슬라이더)

### 검색 결과 목록 (중앙 우측)
- **검색 이력 드롭다운** (상단)
  - 최근 5개 검색어 표시

- **결과 카드** (리스트)
  - 관련도 점수 바 (%)
  - 제목
  - 요약 (2-3줄)
  - 하이라이트된 매칭 문구
  - 출처 정보
  - 작용 버튼 (상세보기, 공유)

### 옵션 영역
- **리랭킹 옵션 토글**
  - 활성화 시 결과 재정렬 (관련도 + 신뢰도 + 최신성)
