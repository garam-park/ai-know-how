|------|-----|

## 수정 사항

### WF-03-03 정책효과예측결과

- [ ] Before/After 비교 영역을 좌우 분할 지도 2개로 구현 (div 기반 플레이스홀더)
- [ ] 중앙 슬라이더로 Before/After 비율 조절 가능 (CSS clip-path 활용)
- [ ] 효과지표 카드 6개 완전 구현 (민원감소율, 주차가용률, 혼잡도변화, 비용효율, 시민만족도, ROI)
- [ ] 각 카드에 변화량 화살표 + 색상 코딩 (개선=초록, 악화=빨강) 확인

### WF-03-04 시나리오비교

- [ ] RD-01의 `createRadarChart()` 적용하여 6축 레이더 차트 구현
  - 축: 비용효율, 만족도, 시공용이성, 유지비절감, 환경개선, 지속성
- [ ] 비교 테이블 15개 지표 전체 채우기 (현재 기본만 구현)
- [ ] 셀 색상 코딩 강화 (최우수=진초록, 중간=노랑, 최하=회색)
- [ ] AI 추천 카드에 신뢰도 바 (0~100%) 추가

### WF-03-05 요금제-공유주차추천

- [ ] RD-02의 `addPolygonLayer()` 적용하여 요금제 추천 강도별 폴리곤 렌더링
  - 색상: 진빨강(80~100%) → 주황(60~79%) → 노랑(40~59%) → 회색(추천 안함)
- [ ] RD-02의 `addClusteredMarkers()` 적용하여 공유주차 마커 클러스터링
- [ ] RD-02의 `addLayerControl()` 적용하여 2개 레이어 토글
- [ ] 추천 상세 모달 구현 (추천 근거 텍스트 + 예상 효과 수치)

## 완료 기준

- [ ] 3개 화면 모두 수정 완료
- [ ] 레이더 차트, 폴리곤 레이어 정상 렌더링
- [ ] 각 화면 재평가 시 종합판정 PASS 달성
