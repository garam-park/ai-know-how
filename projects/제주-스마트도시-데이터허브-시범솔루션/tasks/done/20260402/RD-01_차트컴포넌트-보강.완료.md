## 수정 사항

### charts.js 공통 차트 라이브러리

- [ ] **트리맵 차트 함수 추가** — `createTreemap(containerId, data, options)`
  - 사각형 크기 비례 표현, 색상 그라데이션, 호버 시 값 표시
  - WF-05-02 LLM분류현황에서 사용

- [ ] **레이더 차트 헬퍼 추가** — `createRadarChart(containerId, labels, datasets, options)`
  - 6축 이상 지원, 다중 데이터셋 오버레이, 범례 토글
  - WF-03-04 시나리오비교, WF-03-06 투자우선순위에서 사용

- [ ] **버블 차트 헬퍼 추가** — `createBubbleChart(containerId, data, options)`
  - 크기·색상 매핑, 클릭 이벤트 콜백, 범례
  - WF-05-03 클러스터링 시각화에서 사용

- [ ] **게이지 차트 함수 추가** — `createGauge(containerId, value, max, options)`
  - 반원형 게이지, 색상 구간(정상/주의/위험), 애니메이션
  - WF-01-02 실시간현황모니터, WF-03-02 혼잡분석에서 사용

- [ ] **범례 토글 공통 기능** — 모든 차트에 범례 클릭 시 데이터셋 표시/숨김 적용

- [ ] **차트 리사이즈 대응** — `window.resize` 이벤트에 debounce 적용, 컨테이너 크기 자동 추적
