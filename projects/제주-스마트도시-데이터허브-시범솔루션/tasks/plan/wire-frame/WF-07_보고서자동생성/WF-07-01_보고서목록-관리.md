## 화면 구성
### 상단 탭 영역
- 탭: 전체 / 월간 / 분기 / 수시
- 각 탭에서 해당 유형의 보고서만 필터링

### 필터 영역
- 기간 선택 (날짜 범위)
- 보고서 유형 (다중선택)
- 상태 필터 (생성중/검토중/승인/확정)
- 부서 필터 (다중선택)

### 메인 데이터 테이블
| 항목 | 설명 |
|------|------|
| 번호 | 순서 번호 |
| 보고서명 | 제목 (클릭 시 WF-07-02로 이동) |
| 유형 | 월간/분기/수시 배지 |
| 대상기간 | YYYY-MM 형식 |
| 생성일 | YYYY-MM-DD HH:MM |
| 상태 | 생성중(파란색), 검토중(주황색), 승인(초록색), 확정(회색) 뱃지 |
| 담당부서 | 부서명 |
| 작성자 | 사용자명 |

### 상단 우측 버튼 영역
- "새 보고서 생성" 버튼 → 클릭 시 템플릿 선택 모달 표시

### 일괄 작업
- 행 선택 체크박스
- 일괄 내보내기 (PDF/DOCX 형식 선택)
