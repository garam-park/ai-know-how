## 화면 구성
### 상단 워크플로우 상태 표시
- 단계별 진행 표시 (5단계): 작성 → 1차검토 → 수정 → 2차검토 → 최종승인
- 각 단계별 담당자 표시
- 현재 단계 강조 (색상/체크마크)
- 각 단계 클릭 시 해당 단계 정보 표시

### 좌측 보고서 편집 영역
- 리치 텍스트 에디터 (글자 서식, 목록, 테이블 삽입)
- 섹션별 편집 가능:
  - [ ] 제목 섹션
  - [ ] 요약 섹션
  - [ ] 본문 섹션
  - [ ] 차트 섹션
  - [ ] 테이블 섹션
  - [ ] 결론 섹션
- 섹션 헤더에 "AI 재생성" 버튼 (클릭 시 해당 섹션 자동 재작성)
- 변경 사항 자동 저장 표시 (상태 아이콘)

### 우측 코멘트/피드백 패널 (300px 고정 폭)
- 코멘트 스레드 목록 (타임라인 형태)
- 각 코멘트: 부서명, 작성자, 작성일시, 의견 내용
- 부서별 색상 구분 (태그)
- 해결됨/미해결 상태 표시
- "코멘트 추가" 버튼 → 입력 모달
