## 화면 구성
### 좌측 목차 네비게이션 (220px 고정 폭)
- 스크롤 가능한 목차 리스트
- 각 섹션 클릭 시 중앙 본문에서 해당 섹션으로 스크롤
- 현재 섹션 하이라이트
- 구성: 제목 / 요약 / 본문 / 차트 / 테이블 / 결론

### 중앙 본문 영역 (A4 스타일)
- 흰색 배경, 그림자 효과로 A4 페이지처럼 표시
- 페이지 시뮬레이션 (여백: 좌우 40px, 상하 50px)
- 섹션별 구성:
  - **제목 페이지**: 보고서명, 기관명, 작성일, 대상기간
  - **요약 페이지**: Executive Summary (핵심 내용 3-5줄)
  - **본문**: 구분선으로 섹션 분리
  - **차트 섹션**: 내장 차트 (placeholder)
  - **테이블 섹션**: 데이터 테이블
  - **결론 섹션**: 요약 및 권고사항

### 우측 메타정보 패널 (280px 고정 폭)
