## 화면 구성
Chart.js 기반의 다양한 차트 샘플을 보여주는 컴포넌트 페이지. 각 차트별로 더미 데이터, 반응형 리사이즈, 인터랙션을 포함.

## 차트 타입 및 구성

### 1. 라인 차트 (Line Chart)
**설명**: 시계열 데이터의 추세를 표시하는 차트
**사용 사례**: 일일 교통량, 시간별 에너지 사용량, 주간 방문자 추세
**구성**:
- X축: 시간 (00:00 ~ 23:00) 또는 날짜 (2024-01-01 ~ 2024-01-31)
- Y축: 수치 (0 ~ 최대값, 자동 스케일)
- 여러 라인 가능 (색상 구분)
- 라인 스타일: 실선, 점선 (구분 가능)
- 데이터 포인트: 원형 마커 (호버 시 값 표시)
**기능**:
- 범례 (범례 클릭으로 라인 표시/숨김)
- 그리드 라인
- 툴팁 (호버 시 정확한 값 표시)
- 부드러운 곡선 보간

### 2. 바 차트 (Bar Chart)
**설명**: 카테고리별 수치를 비교하는 차트
**유형**:
- 수직 바 차트 (Vertical Bar): 카테고리를 X축에 배치
- 수평 바 차트 (Horizontal Bar): 카테고리를 Y축에 배치
- 스택 바 차트 (Stacked Bar): 여러 값을 하나의 바에 누적

**사용 사례**:
- 지역별 주차장 수, 월별 민원 건수
- 스택: 카테고리별 상태 분포 (완료/진행중/보류)

**구성**:
- 바 (사각형): 각 데이터를 나타냄
- 라벨: 각 바 위에 값 표시
- 색상: 카테고리별 다른 색상
- 애니메이션: 바가 점진적으로 차오르는 효과

**기능**:
- 범례
- 그리드 라인
- 호버 시 강조 (밝기 증가)
- 값 라벨 표시

### 3. 파이/도넛 차트 (Pie/Doughnut Chart)
**설명**: 전체에서 각 부분이 차지하는 비율을 표시
**유형**:
- 파이 차트: 원형, 중앙 채워짐
- 도넛 차트: 원형, 중앙 비움 (중앙에 텍스트 표시 가능)

**사용 사례**:
- 교통 혼잡도 분포, 에너지 사용 구성비
- 운영 상태 비율 (정상/주의/위험)

**구성**:
- 섹션 (각 데이터 포인트): 다른 색상
- 라벨: 각 섹션의 비율 또는 값 표시
- 범례: 우측 또는 하단에 항목 표시
- 애니메이션: 섹션이 회전하며 그려지는 효과

**기능**:
- 호버 시 섹션 강조
- 클릭으로 섹션 선택 (선택 상태 시각화)
- 도넛 차트 중앙에 합계 또는 주요 수치 표시

### 4. 히트맵 (Heatmap)
**설명**: 2차원 데이터를 색상 강도로 표시 (시간 x 지역)
**사용 사례**: 시간대별 교통 혼잡도, 지역별 민원 밀도
**구성**:
- 행 (Y축): 시간대 (00:00~23:00, 또는 요일)
- 열 (X축): 지역 또는 장소
- 색상 강도: 값의 크기를 나타냄 (연한색 ~ 진한색)
- 색상 스케일: 3~5단계 (저 ~ 고)

**색상 예**:
- 연한 파랑 → 진한 빨강 (저 ~ 고)
- 흰색 → 검정색 (저 ~ 고)

**기능**:
- 툴팁 (셀 호버 시 정확한 값)
- 범례 (색상과 값의 매핑 표시)

### 5. 게이지 차트 (Gauge Chart)
**설명**: 목표에 대한 달성률 또는 위험도를 표시하는 반원형 차트
**사용 사례**:
- 에너지 사용률 (0~100%)
- 시스템 가동률 (0~100%)
- 주차장 수용률 (0~100%)

**구성**:
- 반원형 배경 (전체)
- 색상 구간:
  - 정상 (녹색): 0~66%
  - 주의 (황색): 66~85%
  - 위험 (빨강색): 85~100%
- 바늘 (현재 값 가리킴)
- 중앙 텍스트: 현재 값 (%), 단위

**기능**:
- 부드러운 애니메이션 (바늘이 회전)
- 단계별 색상 변화

### 6. 레이더 차트 (Radar Chart)
**설명**: 다축 데이터를 비교하는 차트 (5개 이상의 지표)
**사용 사례**: 지역별 종합 평가, 다중 지표 비교
**구성**:
- 축 (5~8개): 각 지표
- 영역 (다각형): 각 데이터 시리즈를 나타냄
- 색상: 시리즈별 다른 색상 (반투명)
- 격자선: 축과 원형 그리드

**기능**:
- 여러 시리즈 비교 (최대 3~4개)
- 범례
- 툴팁 (호버 시 값 표시)

### 7. 버블 차트 (Bubble Chart)
**설명**: 3개 차원의 데이터를 표시 (X, Y 좌표 + 버블 크기)
**사용 사례**: 주차장 분석 (위치, 수용율, 회전율), 지역별 운영 지표
**구성**:
- X축: 첫 번째 지표 (예: 수용 용량)
- Y축: 두 번째 지표 (예: 현재 이용률)
- 버블 크기: 세 번째 지표 (예: 일일 이용 건수)
- 버블 색상: 카테고리 또는 상태

**기능**:
- 호버 시 버블 강조 + 툴팁
- 범례 (색상/크기 범위)
- 버블 애니메이션

### 8. 트리맵 (Treemap)
**설명**: 계층적 데이터를 직사각형으로 표시 (비율 반영)
**사용 사례**: 전체 예산 중 부문별 비율, 지역별 면적 비율
**구성**:
- 직사각형: 각 데이터, 크기는 값에 비례
- 색상: 카테고리 또는 부모 노드
- 레이블: 각 사각형에 항목명 + 값 표시
- 계층 표현: 안으로 깊어질수록 작은 사각형

**기능**:
- 클릭으로 드릴다운 (상세 계층으로 이동)
- 뒤로 이동 버튼
- 현재 경로 표시 (breadcrumb)
- 호버 시 사각형 강조

## 공통 기능

### 반응형 리사이즈
- 컨테이너 크기 변경 시 자동 리사이즈
- 최소 너비: 300px, 최소 높이: 250px
- 큰 화면: 최대 너비 1000px, 최대 높이 600px

### 범례 (Legend)
- 위치: 우측 또는 하단 (선택 가능)
- 클릭으로 데이터 시리즈 표시/숨김 (라인, 바, 레이더 차트)
- 색상 표시 (정사각형 또는 원)

### 툴팁 (Tooltip)
- 호버 시 나타남
- 내용: 데이터 라벨 + 값 + 단위
- 배경: 반투명 검정
- 글자: 흰색, 작은 크기

### 애니메이션
- 초기 로드: 부드러운 애니메이션 (0.5~1초)
- 호버: 색상 변화, 밝기 증가
- 데이터 업데이트: 값 변화에 따른 리드로우

## 더미 데이터

### 라인 차트
- 일일 교통량 (시간별): 00:00~23:00, 500~3500 대/시간
- 주간 방문자: 월~일, 1000~5000명

### 바 차트
- 지역별 주차장 수: 제주시/서귀포시/북제주/남제주 (20~50개)
- 월별 민원: 1월~12월 (10~100건)

### 파이/도넛 차트
- 운영 상태: 정상(60%), 주의(25%), 위험(15%)
- 에너지 구성: 태양광(35%), 풍력(25%), 그리드(40%)

### 히트맵
- 시간별/지역별 교통 혼잡도 (0~100)
- 행: 00:00, 06:00, 12:00, 18:00
- 열: 제주시, 서귀포시, 북제주, 남제주

### 게이지 차트
- 에너지 사용률: 72%
- 주차장 수용률: 85%

### 레이더 차트
- 5개 지표: 운영성(80), 안정성(85), 효율성(70), 만족도(75), 보안(90)

### 버블 차트
- 주차장 분석:
  - X축(용량): 100~500대
  - Y축(이용률): 20~95%
  - 버블 크기(일일 건수): 200~1000건

### 트리맵
- 예산 배분:
  - 교통(40%) → 도로(25%), 주차(15%)
  - 에너지(30%)
  - 환경(30%)
