## 화면 구성
로그인 페이지 및 역할 선택 모달을 포함한 인증 진입점. PoC (Proof of Concept) 특성상 비밀번호 없이 역할 선택만으로 시스템에 진입하는 방식 구현.

### 로그인 페이지 레이아웃

**전체 구성**:
- 전체 화면 배경: 그라데이션 또는 단색 (어두운 파랑, #2563EB)
- 중앙 정렬 로그인 박스 (흰색, 그림자 적용)

### 로그인 박스 구성

**상단 영역**:
- 시스템 로고: 상단 중앙 (100x100px)
- 프로젝트명: "제주 스마트도시 데이터허브"
- 부제목: "Jeju Smart City Data Hub"
- 설명 텍스트: "스마트 도시 운영 및 관리를 위한 통합 데이터 플랫폼"

**폼 영역**:

#### 아이디 입력 필드
- 레이블: "아이디"
- 플레이스홀더 텍스트: "사용자 아이디를 입력하세요"
- 아이콘: 사용자 아이콘 (좌측)
- 입력 유효성: 영문 및 숫자만 허용
- 가로 너비: 100%, 최대 400px
- 높이: 44px

#### 비밀번호 입력 필드 (선택 - PoC에서는 스킵 가능)
- 레이블: "비밀번호"
- 플레이스홀더 텍스트: "비밀번호를 입력하세요"
- 아이콘: 자물쇠 아이콘 (좌측)
- 표시/숨김 토글: 우측 버튼 (눈 아이콘)
- 입력 유효성: 8자 이상
- 높이: 44px

#### 로그인 버튼
- 텍스트: "로그인"
- 스타일: Primary 색상 (#2563EB), 흰색 텍스트
- 크기: 너비 100%, 높이 44px
- 호버 상태: 배경색 진해짐 (#1D4ED8)
- 활성 상태: 누르는 효과 (약간 내려감)
- 비활성화: 회색, 커서 not-allowed

#### 추가 링크
- "비밀번호 찾기" 링크: 하단 우측, 작은 텍스트, 회색, 클릭 시 비밀번호 재설정 모달 열기
- "회원가입" 링크: 비밀번호 찾기 좌측, 필요 시만 표시 (PoC에서는 생략 가능)

### PoC 특별 사항

**비밀번호 불필요**:
- 입력 필드: 비밀번호 필드 제거 또는 더미 입력 ("demo" 고정)
- 검증: 아이디 존재 여부만 확인

**아이디 목록** (더미):
- admin
- operator
- viewer

**로그인 버튼 클릭 흐름**:
1. 입력된 아이디 검증
2. "역할 선택" 모달 열기 (아이디별 할당된 역할 표시)
3. 역할 선택 후 대시보드로 이동

### 역할 선택 모달 (Role Selection Modal)

**모달 기본 구조**:
- 배경: 반투명 검정 (z-index: 40)
- 박스: 흰색, 중앙 정렬, 그림자
- 너비: 400px (모바일: 90vw)
- 높이: auto

**모달 내용**:

#### 헤더
- 제목: "역할 선택"
- 설명: "로그인할 역할을 선택하세요. 역할에 따라 보이는 메뉴가 다릅니다."
- 닫기 버튼: X 아이콘 (우측 상단)

#### 역할 선택 옵션 (3가지)

**1. 관리자 (Administrator)**
- 아이콘: 관리 아이콘 (톱니바퀴)
- 제목: "관리자"
- 설명: "전체 시스템 관리 및 설정 권한. 모든 메뉴 및 기능 접근 가능"
- 선택 라디오 버튼
- 강조: Primary 색상 테두리 (선택됨)
- 메뉴 미리보기:
  - 대시보드
  - 교통운영
  - 주차관리
  - 환경모니터링
  - 에너지관리
  - 민원관리
  - 운영관리

**2. 운영자 (Operator)**
- 아이콘: 운영 아이콘 (차트)
- 제목: "운영자"
- 설명: "시스템 운영 및 데이터 모니터링 권한. 제한된 메뉴 접근"
- 선택 라디오 버튼
- 메뉴 미리보기:
  - 대시보드
  - 교통운영
  - 주차관리
  - 환경모니터링
  - 에너지관리

**3. 열람자 (Viewer)**
- 아이콘: 보기 아이콘 (눈)
- 제목: "열람자"
- 설명: "데이터 조회 및 보고서 확인만 가능. 수정 권한 없음"
- 선택 라디오 버튼
- 메뉴 미리보기:
  - 대시보드
  - 환경모니터링
  - 에너지관리

#### 역할 선택 박스 스타일
- 너비: 100%, 높이: auto
- 패딩: 16px
- 마진: 12px (상하)
- 테두리: 1px 회색, 2px Primary 색상 (선택됨)
- 배경: 연한 회색 (호버), 흰색 (기본)
- 커서: pointer
- 트랜지션: 0.2초 (색상 변화 부드럽게)

#### 하단 버튼
- "진입" 버튼:
  - Primary 색상, 흰색 텍스트
  - 너비: 100%, 높이: 44px
  - 마진 상단: 20px
- "취소" 버튼 (선택):
  - Secondary 색상 (회색), 검정 텍스트
  - 너비: 48% (좌측)

### 로그인 페이지 추가 요소

**배경 그래픽** (선택):
- 배경: 그라데이션 (밝은 파랑 → 진한 파랑)
- 배경 패턴 또는 도형 (부스트 트랩, 선택)

**푸터** (로그인 페이지):

- 링크: "서비스약관" | "개인정보처리방침"
- 버전: "v1.0"
- 위치: 화면 하단
