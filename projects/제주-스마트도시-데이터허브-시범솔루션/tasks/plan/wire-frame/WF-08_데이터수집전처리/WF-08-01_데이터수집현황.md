## 화면 구성
### 상단 수집 소스별 상태 카드 (수평 스크롤 또는 6개 고정)
각 카드 포함 항목:
- 소스 아이콘
- 소스명 (민원포털, 만덕콜센터, 통합관제, 기상청API, 112/119, 전기차충전)
- 최종수집시각 (HH:MM:SS)
- 일간건수 (숫자, 큰 폰트)
- 상태뱃지 (정상/지연/실패 - 색상 구분)

### 중앙 시간대별 수집량 스택 바차트
- X축: 시간 (00시 ~ 23시, 또는 날짜)
- Y축: 수집 건수
- 소스별 색상으로 스택 바 표시
- 범례: 6개 소스 표시
- 마우스 호버 시 해당 시간대 상세 수집량 툴팁

### 하단 수집 로그 테이블
| 항목 | 설명 |
|------|------|
| 시각 | HH:MM:SS |
| 소스 | 민원포털/만덕콜센터 등 |
| 건수 | 정수 |
| 상태 | 성공/실패 뱃지 |
| 에러메시지 | "Connection timeout", "Invalid format" 등 |

- 테이블 정렬 가능 (시각, 소스, 건수, 상태)
- 페이지네이션 또는 무한 스크롤
- **에러 건은 상단 고정**: 빨간색 배경, "⚠️" 아이콘으로 눈에 띄게 표시
- 검색 바: 소스명, 에러메시지로 필터링
