## 화면 구성
### 상단 KPI 카드 (4개)
| 카드명 | 내용 |
|--------|------|
| 일간변환건수 | 숫자 (예: 3,456건) |
| 평균WER | 백분율 (Word Error Rate, 예: 8.5%) |
| 평균CER | 백분율 (Character Error Rate, 예: 5.2%) |
| PII마스킹건수 | 숫자 (예: 234건) |

### 메인 변환 결과 목록 (테이블)
| 항목 | 설명 |
|------|------|
| 음성ID | 고유 ID (클릭 시 상세 뷰) |
| 통화일시 | YYYY-MM-DD HH:MM:SS |
| 길이 | MM:SS 형식 |
| 화자수 | 정수 (예: 2명) |
| WER | 백분율 (0~100%) |
| 상태 | 완료(초록), 처리중(파란색), 실패(빨강) 뱃지 |

- 행 클릭 시 상세 정보 패널 또는 모달 표시

### 상세 뷰 (우측 패널 또는 모달)
- **음성 파형 Placeholder**: SVG 또는 이미지로 파형 표시
- **변환 텍스트**: 스크롤 가능한 텍스트 영역
- **화자별 구분 표시**: 다른 색상이나 들여쓰기로 표시
  - 예: [화자 1] "안녕하세요"
  - [화자 2] "네, 인사합니다"
- **PII 마스킹 하이라이트**: 주민번호, 전화번호 등이 마스킹된 부분 표시 (노란색 배경)

### 하단 사투리 인식 정확도 통계
- 지역별 인식 정확도 바차트 또는 테이블
- 예: 경남 사투리 92%, 강원 사투리 88%, 경기 사투리 95%
