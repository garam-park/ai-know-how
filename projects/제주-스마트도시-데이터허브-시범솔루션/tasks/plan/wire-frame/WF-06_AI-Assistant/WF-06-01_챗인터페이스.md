## 화면 구성

### 좌측 사이드패널 - 대화 세션 목록
- **새 대화 버튼** (+ 아이콘)
- **이전 대화 이력** (리스트)
  - 각 항목: 대화 제목 (자동 생성), 작성일시, 마지막 메시지 미리보기
  - 클릭하여 대화 로드
  - 삭제 버튼 (휴지통 아이콘)

### 중앙 채팅 영역
- **메시지 버블** (스크롤 가능)
  - 사용자 메시지: 우측 정렬, 파란색 배경
  - AI 메시지: 좌측 정렬, 회색 배경
  - 각 메시지: 내용, 타임스탐프, 아바타/아이콘
  - 메시지 내 인라인 차트/테이블 렌더링 (placeholder 영역)

### 하단 입력 영역
- **추천 질문 칩** (4개) - 입력 바 위
  - "이번 달 민원 요약해줘"
  - "한림읍 주차 현황"
  - "불법주차 정책 추천"
  - "공영주차장 안전현황"

- **입력 바** (하단)
  - 텍스트 입력 필드 (placeholder: "메시지를 입력하세요...")
  - 음성 입력 버튼 (마이크 아이콘)
  - 첨부 버튼 (클립 아이콘)
  - 전송 버튼 (종이비행기 아이콘)

## 사용 컴포넌트
