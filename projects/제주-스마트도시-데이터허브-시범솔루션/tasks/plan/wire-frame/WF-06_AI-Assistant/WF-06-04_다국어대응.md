## 화면 구성

### 상단 언어 선택 영역
- **언어 선택 드롭다운** (상단 우측)
  - 한국어 (기본값)
  - English
  - 中文 (Simplified Chinese)
  - 日本語 (Japanese)
  - 선택 시 전체 UI 및 콘텐츠 번역

### 메인 콘텐츠 영역
- **WF-06-01 챗 인터페이스와 동일 레이아웃**
  - 선택 언어로 UI 텍스트 표시 (버튼, 플레이스홀더, 레이블)
  - AI 응답이 선택 언어로 표시
  - 추천 질문 칩이 선택 언어로 번역됨

### FAQ 섹션 (메인 아래 또는 탭)
- **자주 묻는 질문 10개** (선택 언어로 표시)
  - 각 FAQ 항목:
    - 질문 (제목)
    - 답변 (펼칠 수 있는 구조)
    - 클릭하면 챗에 질문 추가 버튼

### 메시지 별 번역 옵션
- **각 메시지 하단**
  - "원문/번역문 토글" 버튼 (토글 또는 드롭다운)
  - 원문: 원래 언어로 표시
  - 번역문: 선택 언어로 표시
  - 둘 다 표시 옵션 (나란히 표시)

- **번역 품질 피드백 버튼** (각 메시지 하단)
  - 피드백 아이콘 (별 또는 평점)
  - 클릭하여 번역 품질 평가 (1~5점)
  - 제출 시 피드백 저장 알림
