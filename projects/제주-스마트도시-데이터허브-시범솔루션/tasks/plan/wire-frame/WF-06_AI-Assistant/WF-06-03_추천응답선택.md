## 화면 구성

### 패널/모달 구조
- **모달 또는 우측 사이드패널** (WF-06-01과 함께 표시)
- 제목: "추천 응답"
- 닫기 버튼 (X 아이콘)

### 상단 유형 탭
- **3개 탭** (라디오 또는 탭 네비게이션)
  - 민원 응답
  - 정책 안내
  - 공문/공지

### 응답 형식 토글 (탭 하단)
- **응답 형식 선택** (토글 버튼 또는 드롭다운)
  - 요약형 (1-2문장)
  - 상세형 (3-5문장)
  - 음성 스크립트 (TTS 친화적)

### 메인 콘텐츠 영역
- **AI 추천 응답 후보 3개** (카드 형태)
  - 각 카드 구성:
    - 응답 제목 (자동 생성)
    - 요약문 (1줄 preview)
    - 상세 펼치기 버튼 (화살표 아이콘)
    - 선택 라디오 버튼

- **펼친 상태** (각 카드)
  - 전체 응답 텍스트
  - 수정 가능 에디터 (인라인)
  - "접기" 버튼

### 선택 후 편집 영역 (우측 또는 하단)
- **선택한 응답 편집 영역**
  - 제목 입력
  - 본문 텍스트 에디터 (Rich Text)
  - 실시간 글자 수 카운터
  - 형식 스타일 버튼 (굵기, 이탤릭, 색상)

### 하단 액션 영역
- **"적용" 버튼** (주요 액션)
  - 선택한 응답을 WF-06-01 채팅으로 반영
  - 버튼 클릭 시 모달 닫기
- **"취소" 버튼** (회색)
