## 화면 구성

### 좌측 지도 영역 (70%)
- 제주도 지도 기반 레이어 표시
- **레이어 1**: 요금제 추천 지역 (폴리곤)
  - 색상으로 추천 강도 표시
    - 진한 빨강: 매우 강함 (80~100%)
    - 주황: 강함 (60~80%)
    - 연한 주황: 보통 (40~60%)
    - 연한 노랑: 약함 (20~40%)
    - 회색: 미추천 (0~20%)
  - 마우스 오버 시 지역명 + 추천강도(%) 표시

- **레이어 2**: 공유주차 추천 지역 (별도 마커 또는 기호)
  - 마커 아이콘(공유주차 전용 아이콘)
  - 색상: 초록색(강함), 연초록(약함)
  - 클러스터링 기능 (줌 레벨에 따라 그룹화)

- **범례**: 우측 상단 또는 우측에 고정
  - 요금제 추천: 색상 범위 설명
  - 공유주차 추천: 아이콘 설명
  - 토글 버튼: 각 레이어 온/오프 전환

- **줌/패닝**: 표준 컨트롤

### 우측 패널 (30%)
- **상단**: "추천 현황 요약"
  - 요금제 추천 지역 수
  - 공유주차 추천 지역 수
  - 전체 커버리지 (%)

- **중앙**: 추천 지역 목록
  - 테이블 형식 (스크롤 가능)
  - 컬럼: 순번, 지역명, 추천유형(요금제/공유주차), 추천강도(%), 우선순위
  - 정렬: 강도 내림차순 (기본)
  - 행 클릭 시 상세 근거 팝업 호출

- **하단**: 필터/검색
  - 추천유형 필터 (요금제만/공유주차만/전체)
  - 강도 범위 필터 (슬라이더 또는 셀렉트)
  - 검색 입력란 (지역명)

### 모달/팝업: 상세 근거
- 팝업 제목: "[지역명] 추천 상세"
- 내용:
  - 추천 유형
  - 추천 강도(%)
  - 우선순위 (1~5위)
  - 추천 사유 (텍스트, 3~5줄)
    - 예: "민원 빈도가 월평균 45건으로 높은 수준. 피크시간 혼잡도 85% 달성 예상. 수요 대비 공급 부족 심각(갭: 280대)."
  - 예상 효과 (텍스트)
    - 예: "요금제 적용 시 민원 35% 감소, 혼잡도 26% 개선 기대."
  - 닫기 버튼
