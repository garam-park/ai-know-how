## 화면 구성

### 상단 시나리오 선택 영역
- 드롭다운 셀렉트 3개 (좌/중/우)
  - 각 드롭다운: "시나리오 선택" 레이블 + 저장된 시나리오 목록
  - 선택된 시나리오명 표시
  - 각 시나리오별 태그: 정책유형 아이콘 (주차/요금/노변 등)

### 중앙 비교 테이블 영역
- **행**: 지표 항목 (15개 이상)
  - 민원 감소율
  - 주차 가용률
  - 혼잡도 변화
  - 비용 효율
  - 시민 만족도
  - ROI
  - 투자 규모
  - 시공 기간
  - 유지보수비
  - 에너지 절감
  - 환경 개선도
  - 교통 흐름 개선
  - 신청 처리율
  - 민원 해결율
  - 종합 효율성

- **열**: 시나리오 1, 시나리오 2, 시나리오 3
- **셀**: 수치 + 진행 바(0~100%)
- **색상 코딩**: 셀 배경색으로 우열 표시
  - 최고: 진한 초록
  - 중간: 노랑
  - 최저: 연한 회색

### 좌측 레이더 차트 영역 (선택사항)
- 시나리오별 다축 비교 차트
- 축: 주요 6개 지표
  - 비용 효율 (0~100)
  - 시민 만족도 (0~10)
  - 시공 용이성 (0~10)
  - 유지비 절감 (0~100%)
  - 환경 개선 (0~10)
  - 장기성 (0~10)
- 각 시나리오별 색상 다르게 표시

### 하단 AI 추천 영역
- 카드: "AI 추천 시나리오"
  - 추천 순위 (1순위, 2순위, 3순위)
  - 추천 시나리오명
  - 추천 신뢰도 바(0~100%)
  - 추천 근거 텍스트 (3~4줄)
    - 예: "주차 가용률 72% 달성으로 이용자 만족도 최고. 투자 대비 효율성 156%로 가장 우수함."
