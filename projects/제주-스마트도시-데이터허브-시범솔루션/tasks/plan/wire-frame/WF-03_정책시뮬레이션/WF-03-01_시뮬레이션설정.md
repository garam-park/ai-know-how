## 화면 구성

### 레이아웃 분할
- **좌측 (60%)**: 제주도 지도 영역
  - 생활권 그리드 오버레이(최대 30개 구역)
  - 그리드별 클릭으로 영역 선택 가능
  - 선택된 영역은 색상 하이라이트(초록색 테두리)
  - 줌 컨트롤(+/- 버튼)

- **우측 (40%)**: 설정 패널
  - 시나리오명 입력란(텍스트)
  - 정책 유형 선택(라디오버튼 또는 셀렉트)
    - 주차장 확장
    - 노변정비
    - 요금제 적용
    - 공유주차
  - 파라미터 설정 영역
    - 슬라이더 1: 증설 면수 (0~500면)
    - 슬라이더 2: 요금 단가 (₩0~₩5,000)
    - 슬라이더 3: 적용 기간 (1~24개월)
  - 비교 시나리오 추가 버튼(최대 3개 선택 가능)
  - 하단: "시뮬레이션 실행" 주요 버튼(파란색)

- **하단**: 이전 시나리오 이력 목록
  - 테이블 형식: 시나리오명, 정책유형, 적용 날짜, 상태(완료/실행중)
  - 클릭으로 이전 시나리오 불러오기 가능
