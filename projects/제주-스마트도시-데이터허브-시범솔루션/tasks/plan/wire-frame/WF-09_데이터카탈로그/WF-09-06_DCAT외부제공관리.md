## 화면 구성
### 상단 제공 형식 선택 탭 (3개)
1. **JSON-LD** (기본 선택)
2. **Turtle**
3. **DCAT-AP-KR**

### 메인 데이터셋 선택 영역
- 제목: "내보낼 데이터셋 선택"
- 체크박스 목록 또는 테이블
  - [ ] 데이터셋명 1
  - [ ] 데이터셋명 2
  - [ ] 데이터셋명 3
  - ... (여러 데이터셋)

- 각 행에: 데이터셋명, 선택 상태, 내보내기 형식 표시
- "전체 선택" / "전체 해제" 버튼
- 검색 바로 필터링 가능

### 미리보기 영역
- 선택한 형식의 RDF 코드 표시
- 문법 강조 (Syntax Highlighting)
- 스크롤 가능
- "코드 복사" 버튼

예시 (JSON-LD):
```json
{
  "@context": "https://www.w3.org/ns/dcat",
  "@type": "dcat:Dataset",
  "@id": "http://data-hub.jeju.go.kr/dataset/traffic-info",
  "dcat:title": "실시간 도로 교통 정보",
  "dcat:description": "제주시 전역 도로의 교통 정보",
  "dcat:keyword": ["교통", "실시간"],
  ...
}
```

### 우측 API 엔드포인트 설정 패널
- **제목**: "외부 제공 설정"
- **형식별 API 엔드포인트**: 수정 불가 (읽기 전용, 복사 버튼)
  ```
  https://api.data-hub.jeju.go.kr/catalog/export/jsonld
  ```

- **인증키**: 생성된 인증 키 표시 (마스킹, "재발급" 버튼)

- **갱신주기**: Dropdown
  - 실시간
  - 일간
  - 주간
  - 월간

- "설정 저장" 버튼

### 하단 제공 이력 로그 테이블
| 시각 | 요청자 | 데이터셋 수 | 형식 | 상태 | 행위 |
|------|--------|-----------|------|------|------|
| 2026-03-25 14:32 | API-KEY-001 | 5 | JSON-LD | 성공 | 다운로드 |
| 2026-03-24 10:15 | 외부시스템 | 3 | Turtle | 성공 | 조회 |

- 테이블 정렬 가능
- 페이지네이션 또는 무한 스크롤
