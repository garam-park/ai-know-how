## 화면 구성
### 상단 버튼 영역
- "AI 자동 추출" 버튼 → 인터페이스 정의서/데이터 모델에서 메타데이터 자동 추출

### AI 추출 결과 비교 (Diff 뷰)
- 좌측: 기존 메타데이터 값 (회색 배경)
- 우측: AI 추출 제안값 (초록색 배경)
- 각 필드별로 변화 표시
- "적용" 버튼으로 개별 필드 선택 적용
- "모두 적용" 버튼으로 한번에 적용

### 메인 폼 영역
메타데이터 입력 폼 (필수항목에 * 표시):

| 필드명 | 타입 | 예시 값 |
|--------|------|--------|
| 데이터셋명 * | Text | "실시간 도로 교통 정보" |
| 설명 * | Textarea | "제주시 전역 도로의..." |
| 키워드 | Tag Input | "교통, 실시간, 도로" |
| 제공기관 * | Dropdown | "제주시청" |
| 담당부서 * | Dropdown | "교통과" |
| 라이선스 | Dropdown | "CC BY 4.0" |
| 공개수준 * | Dropdown | "공개" (공개/제한공개/비공개) |
| 갱신주기 | Dropdown | "실시간" (실시간/일간/주간/월간/미정) |

### 필드별 검증 표시
- 필수항목이 비어있으면 빨간 테두리 + "필수항목입니다" 메시지
- 선택적 필드는 회색 테두리

### 하단 변경 이력 아코디언
- 아코디언 형태로 "변경 이력" 표시
- 각 버전별 변경 내용:
  - 변경일시
  - 변경자
  - 변경 필드
  - 변경 전/후 값

### 하단 버튼
- "저장" 버튼 (모달: 변경 내용 요약)
- "검토요청" 버튼 (모달: 검토자 선택)
- "취소" 버튼 (뒤로가기)
