## 화면 구성
### 좌측 엔티티 타입 트리 뷰 (220px 고정 폭)
- 트리 구조로 엔티티 타입 나열
- 접기/펼치기 아이콘 (▶ / ▼)
- 엔티티 타입 목록 (예: Device, Building, Road, Traffic, Person 등)
- 선택된 엔티티 하이라이트 (배경색)
- 검색 바로 필터링 가능

### 중앙 엔티티 상세 정보
선택한 엔티티 타입의 상세 표시:
- **타입명**: "Device" (큰 폰트)
- **설명**: 엔티티 타입의 설명

#### 속성 목록 테이블
| 속성명 | 타입 | NGSI-LD 맵핑 | 설명 |
|--------|------|------------|------|
| id | String | @id | 고유 식별자 |
| type | String | @type | 엔티티 타입 |
| name | String | schema:name | 이름 |
| location | GeoProperty | geo:location | 지리적 위치 |
| temperature | Property | custom:temperature | 온도 |

- 테이블 정렬 가능
- 속성 클릭 시 상세 정보 표시

#### 관계 목록 (Relations)
- 관계명, 대상 엔티티 타입, 설명

### 우측 관계 시각화 (노드-엣지 그래프)
- 노드: 엔티티 타입 (원 또는 사각형)
- 엣지: 엔티티 간 관계 (화살표)
- 선택된 엔티티 강조 표시
- 마우스 호버 시 관계 상세 정보 툴팁
- 줌/팬 기능 (마우스 휠, 드래그)

### 하단 속성 상세 테이블
전체 속성 목록 (NGSI-LD 표준 준수율 포함):
| 속성명 | 타입 | NGSI-LD 맵핑 | 설명 | 필수 |
|--------|------|------------|------|-----|
| temperature | Property | custom:temperature | 온도 | Yes |

### AI 자동 탐색 결과 섹션
- "데이터허브에서 발견된 후보 모델" 제목
- 후보 모델 목록 (카드 또는 테이블)
  - 모델명
  - 유사도 점수 (percentage)
  - "확인" 링크 (클릭 시 해당 모델로 이동)
