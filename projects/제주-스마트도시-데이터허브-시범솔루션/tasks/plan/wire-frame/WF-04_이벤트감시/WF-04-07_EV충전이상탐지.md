## 화면 구성

### 상단 충전기 상태 요약 카드
- **4개 카드** (1행 4열 또는 2행 2열)
- 각 카드 구성:
  - **상태 라벨**: "총수", "사용중", "대기", "이상"
  - **큰 숫자**: 해당 상태의 충전기 수
    - 총수: 80
    - 사용중: 32
    - 대기: 45
    - 이상: 3
  - **색상**:
    - 총수: 검정
    - 사용중: 파랑
    - 대기: 초록
    - 이상: 빨강

### 지도 영역 (중앙, 60%)
- **지도 베이스**: 제주도 지도 (WF-00-05)
- **충전기 위치 마커**:
  - 마커 아이콘: EV 충전 전용 아이콘 (번개 모양 또는 충전 심볼)
  - **마커 색상으로 상태 표시**:
    - 파란색: 사용중 (현재 충전 진행)
    - 초록색: 대기 (충전 준비 완료, 유휴 상태)
    - 빨강색: 이상 (오류/알람 발생)
  - 마우스 오버: 충전기ID + 상태 팝업
  - 클릭: 우측 패널에 상세정보 로드

- **마커 클러스터**: 줌 레벨에 따라 자동 그룹화
  - 클러스터 숫자 표시 (예: "12")
  - 줌인 시 개별 마커로 분산

### 우측 패널 (약 25%)
- **선택 충전기 상세정보**:
  - 충전기ID (예: "CHG-001-01")
  - 위치 (상세 주소)
  - 현재 상태 (사용중/대기/이상)
  - 최후 상태변경 시각
  - 배터리 온도 (센서가 있는 경우)

### 하단 이상 이벤트 목록 (약 35%)
- **테이블 형식** (높이 고정, 스크롤 가능)
- **컬럼**:
  - 충전기ID (예: "CHG-001-02")
  - 위치 (주차장명 + 위치)
  - 이상유형 (다음 중 택 1):
    - 과충전 (Overcharging)
    - 충전중단 (Charging Interrupted)
    - 비인가사용 (Unauthorized Use)
    - 장기점유 (Long Occupancy)
    - 온도이상 (Temperature Anomaly)
    - 전력공급이상 (Power Supply Error)
  - 발생시각 (예: "14:25:30" 또는 "5분전")
  - 현재상태 (미확인/확인/대응중/해결)
  - 액션 버튼 ("상세" 또는 "해결")

- **색상 코딩**: 이상유형별 행 배경색
  - 과충전: 밝은 빨강
  - 충전중단: 밝은 주황
  - 비인가사용: 밝은 빨강
  - 장기점유: 밝은 노랑
  - 온도이상: 밝은 주황
  - 전력공급이상: 밝은 회색

### 좌측 필터 패널 (15%, 선택사항)
- **필터 버튼/섹션**:
  - "이상 유형"
    - [ ] 모두
    - [ ] 과충전
    - [ ] 충전중단
    - [ ] 비인가사용
    - [ ] 장기점유
    - [ ] 온도이상
    - [ ] 전력공급이상

  - "상태"
    - [ ] 모두
    - [ ] 미확인
    - [ ] 확인
    - [ ] 대응중
    - [ ] 해결

  - "시간범위"
    - ( ) 오늘
    - ( ) 7일
    - ( ) 30일
    - ( ) 전체

  - "적용" 버튼

### 상단 검색/정렬 (선택사항)
- 검색 입력란: 충전기ID 또는 위치 검색
- 정렬 드롭다운: 발생시각(최신순)/이상유형/위치
