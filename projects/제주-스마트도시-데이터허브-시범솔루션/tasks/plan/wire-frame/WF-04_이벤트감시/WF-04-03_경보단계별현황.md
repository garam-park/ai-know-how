## 화면 구성

### 상단 경보 단계별 요약 카드 영역
- **4개 카드** (1행 4열 그리드)
- 각 카드 구성:
  - **단계명**: "주의", "경고", "위험", "긴급"
  - **색상**: 각 단계별 전용 색상
    - 주의: 노랑
    - 경고: 주황
    - 위험: 빨강
    - 긴급: 진한 빨강
  - **활성 이벤트 수**: 큰 숫자 (예: "3")
  - **부가 텍스트**: "현재 활성 중" 또는 이벤트 요약

- **카드 레이아웃**:
  ```
  [색상 상자] [단계명]
  [큰 숫자]
  [설명]
  ```

### 중앙 경보 타임라인
- **형식**: 수평 타임라인 (시간축)
- **시간 범위**: 최근 24시간 또는 선택 범위
- **구성 요소**:
  - 가로 시간축 (시작~종료 시간 표시)
  - 시간 간격 마크 (1시간 또는 3시간 단위)
  - 이벤트 점 (작은 원, 타임라인 위에 배치)
    - X축: 발생 시각
    - Y축 (스택): 이벤트 겹침 방지
    - 색상: 이벤트 단계별 (주의/경고/위험/긴급)
  - 마우스 오버 시: 이벤트명 + 시각 + 주차장 팝업 표시
  - 클릭 시: 해당 이벤트 상세 조회

- **범례**: 타임라인 하단 또는 좌측

### 하단 단계별 이벤트 목록
- **탭 구조**: 4개 탭 ("주의", "경고", "위험", "긴급")
- **각 탭 콘텐츠**: 데이터 테이블
  - **컬럼**:
    - 시각 (상대 또는 절대, 예: "14:25:30")
    - 이벤트명 (예: "장기점유 감지")
    - 주차장명
    - 상태 (미확인/확인/대응중/해결)
    - 액션 버튼 ("상세")

  - **테이블 속성**:
    - 정렬 기능 (컬럼 헤더 클릭)
    - 페이지네이션 또는 무한스크롤
    - 행 색상: 상태에 따라 (미확인: 강조, 해결: 흐린색)

### 우측 통계 패널 (선택사항)
- **제목**: "통계"
- **항목**:
  - 일평균 경보 횟수: "12.4건/일"
  - 평균 대응시간: "8분 15초"
  - 오탐률: "12.3%"
  - 해결율: "98.7%"

- 각 항목에 추이 화살표 (전일 대비)
