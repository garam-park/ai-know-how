## 화면 구성

### 상단 연계 상태 표시 영역
- **상태 인디케이터** (좌측):
  - 원형 상태 아이콘 (크기: 32x32px)
    - 초록색 (원): 연결됨
    - 빨강색 (원): 끊김
  - 상태 텍스트 (중간):
    - "연결됨" 또는 "끊김" (크고 굵은 폰트)
  - 추가 정보:
    - "마지막 동기화 시각: 2024-04-01 14:25:30"
    - 또는 "연결 끊김 시각: 2024-04-01 14:20:00 (약 5분 전)"

- **우측 액션 버튼**:
  - "재연결" (끊김 상태일 때만 활성화)
  - "수동 동기화" (항상 활성화, 초록색)
  - "상태 갱신" (회색)

### 중앙 전송 로그 테이블
- **테이블 헤더**:
  - 전송시각
  - 이벤트ID
  - 이벤트유형 (짧은 텍스트, 예: "화재경보")
  - 전송상태 (성공/실패/재시도)
  - 응답코드 (HTTP 상태 코드, 예: "200", "503", "000")
  - 응답메시지 (짧은 텍스트, 예: "OK", "Service Unavailable", "No Response")
  - 재시도횟수 (숫자)

- **테이블 행 색상 코딩**:
  - 성공 (200): 밝은 초록
  - 실패 (4xx/5xx): 밝은 빨강
  - 재시도 (진행중): 밝은 노랑
  - 대기 (아직 미전송): 밝은 회색

- **테이블 속성**:
  - 최근 100건 표시 (스크롤 가능)
  - 정렬: 기본 시간 역순 (최신부터)
  - 정렬 가능 (컬럼 헤더 클릭)
  - 페이지네이션: 10개 항목/페이지

- **조회 기간 필터** (테이블 상단):
  - 드롭다운: "오늘", "7일", "30일", "전체"
  - 기본값: "오늘"

### 우측 패널: 전송 통계
- **통계 항목** (카드 형식):
  - 총 전송 건수 (오늘): "245건"
  - 성공 건수: "238건" (색상: 초록)
  - 실패 건수: "5건" (색상: 빨강)
  - 재시도 중: "2건" (색상: 노랑)
  - 성공률: "97.1%" (진행바)
  - 평균 응답시간: "0.5초"

- **오류 요약** (상단 고정):
  - "503 Service Unavailable: 3건"
  - "504 Gateway Timeout: 2건"
  - 등 최근 오류 TOP 5

### 하단 그래프 영역
- **2개 차트** (1행 2열)

#### 차트 1: 일별 전송 건수 (막대 차트)
- X축: 날짜 (최근 7일)
- Y축: 전송 건수
- 막대 색상:
  - 초록: 성공
  - 빨강: 실패
  - 노랑: 재시도 (스택)
- 범례: 성공/실패/재시도

#### 차트 2: 실패율 추이 (라인 차트)
- X축: 날짜 (최근 7일)
- Y축: 실패율 (%)
- 라인 색상: 빨강
- 데이터 포인트에 값 표시
- 평균선 (점선, 회색): "평균 2.3%"

### 하단 수동 재전송 영역
- **버튼**: "선택된 이벤트 재전송" (파란색)
- **용도**: 테이블에서 실패 건 선택 후 재전송
- **결과 메시지** (인라인):
  - "재전송 완료: 3건" (성공 시 초록)
  - "재전송 실패: 1건 (상세보기)" (실패 시 빨강)
