## 화면 구성

### 상단 인시던트 정보 요약 카드
- **카드 레이아웃** (가로):
  - 좌측: 인시던트 아이콘 + 유형
  - 중앙: 주요 정보
    - 인시던트ID (예: "INC-2024-04-001")
    - 유형 (예: "화재경보 신호")
    - 주차장명
    - 발생시각
  - 우측: 심각도 뱃지 + 상태 (미확인/확인/대응중/해결)

### 중앙 메인 영역: 단계별 체크리스트
- **레이아웃**: 상단에서 하단으로 단계 나열 (수직)
- **6개 단계** (각 단계별 카드 또는 섹션):

#### 1단계: 상황인지
- 체크 항목 3~4개:
  - [ ] 인시던트 알람 확인
  - [ ] 관련 센서/영상 수집
  - [ ] 주차장 현황 파악
  - [ ] 초동 대응팀 소집 (선택)
- 담당자 배정 드롭다운 (이름 선택)
- 완료 시각 기록란 (자동 입력 또는 수동)

#### 2단계: 초기대응
- 체크 항목:
  - [ ] 현장 접근성 확인
  - [ ] 안전 조치 (진입 제한, 경고 표시)
  - [ ] 추가 센서 활성화
  - [ ] 영상 저장 시작
  - [ ] 관련자 연락 시작
- 담당자 배정
- 완료 시각

#### 3단계: 유관기관연락
- 체크 항목:
  - [ ] 소방서 통보
  - [ ] 경찰청 통보
  - [ ] 보건당국 통보 (필요시)
  - [ ] 관리사 별도 지시
  - [ ] 언론 대응 준비
- 담당자 배정
- 완료 시각
- 연락처/담당자 텍스트 입력란

#### 4단계: 현장조치
- 체크 항목:
  - [ ] 화재/위험물 제거
  - [ ] 부상자 응급처치
  - [ ] 장비 안정화
  - [ ] 주변 주민 대피 완료
  - [ ] 증거물 보관
- 담당자 배정
- 완료 시각

#### 5단계: 상황종료
- 체크 항목:
  - [ ] 현장 안정 확인
  - [ ] 사후 검사 실시
  - [ ] 영상/데이터 보관
  - [ ] 주변 정상화 확인
  - [ ] 일시 폐장/운영 재개 결정
- 담당자 배정
- 완료 시각

#### 6단계: 사후보고
- 체크 항목:
  - [ ] 사건 보고서 작성
  - [ ] 원인 분석 완료
  - [ ] 개선 조치 계획 수립
  - [ ] 보험청구 준비
  - [ ] 시스템 점검 및 복구
- 담당자 배정
- 완료 시각

### 진행 상황 표시
- **페이지 상단 또는 좌측**: 진행률 프로그레스 바 또는 숫자
  - 예: "3/6 단계 완료 (50%)"
  - 완료된 단계는 초록색 체크, 진행중은 파랑색, 미시작은 회색

### 하단 비고/메모 영역
- **텍스트 입력란** (큼, 다중 줄 가능)
- 라벨: "비고/메모"
- 플레이스홀더: "현장 상황, 추가 조치 사항 등을 기록하세요."
- 작성자/시간 자동 기록

### 우측 액션 버튼 (고정)
- 버튼 레이아웃 (세로):
  - "임시저장" (회색)
  - "제출" (파란색, 모든 필수 항목 완료 시 활성화)
  - "종료" (주황색, 인시던트 종료 처리)
  - "인쇄" (회색, 보고서 인쇄)
