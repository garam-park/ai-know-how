## 화면 구성

### 상단 영역 - 필터 바
- **기간 필터**
  - 시작일 ~ 종료일 (캘린더 피커)

- **지역 필터**
  - 드롭다운 (시/군/구 선택)

- **민원유형 필터**
  - 멀티셀렉트 (불법주차, 보행안전, 시설불편, 충전구역점유, 화재위험, 주차안전)

- **상태 필터**
  - 드롭다운 (전체/접수/처리중/완료/미처리)

- **채널 필터**
  - 멀티셀렉트 (모바일앱, 웹, 콜센터, 방문)

- **키워드 검색**
  - 텍스트 입력창 (제목/내용 검색)

- **조회/초기화 버튼**
  - 버튼 그룹

### 메인 영역 - 데이터 테이블
**테이블 헤더 컨트롤**
- 전체 선택 체크박스
- 일괄 처리 드롭다운
  - 옵션: 배정/상태변경/내보내기
  - 버튼 활성화: 1개 이상 선택 시

**테이블 컬럼**
| # | 선택 | 접수번호 | 접수일 | 채널 | 지역 | 유형 | 제목 | 상태 | 감성 | 담당자 |

**테이블 데이터**
- 50건의 민원 데이터 표시
- 체크박스: 각 행 선택
- 감성도: 감성 점수에 따른 색상 (부정-빨강, 중립-회색, 긍정-초록)
- 상태 뱃지: 색상 구분 (접수-파랑, 처리중-주황, 완료-초록, 미처리-빨강)
- 행 클릭 시 WF-02-05 상세 페이지로 이동

### 하단 영역 - 페이지네이션
- 페이지 정보: "1-50 / 총 XXX건"
- 페이지 버튼 (이전/1/2/3.../다음)
- 페이지당 항목 수 선택 (10/25/50/100)
