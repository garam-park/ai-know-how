## 화면 구성

### 상단 영역
- **필터 바**: 수평 필터 바 레이아웃
  - 기간 선택 (시작일~종료일)
  - 지역 드롭다운 (시/군/구 선택)
  - 민원유형 멀티셀렉트
  - 상태 필터 (전체/접수/처리중/완료/미처리)
  - 조회/초기화 버튼

### KPI 카드 영역 (2x2 그리드)
- **KPI 카드 1**: 총 접수건수 (예: 1,234건, 전주 대비 +12%)
- **KPI 카드 2**: 처리 완료 (예: 890건, 72%)
- **KPI 카드 3**: 처리중 (예: 234건, 19%)
- **KPI 카드 4**: 미처리 (예: 110건, 9%)

### 중앙 좌측 (50%)
- **시간대별 민원 발생 라인차트**
  - X축: 24시간 시간대
  - Y축: 민원 건수
  - 유형별 색상 구분 범례

### 중앙 우측 (50%)
- **지역별 민원 비율 도넛차트**
  - 주요 지역 TOP 5 표시
  - 기타 항목 포함
  - 범례와 백분율 표시

### 하단 영역
- **최근 민원 미니 테이블**
  - 컬럼: 번호, 접수일, 지역, 유형, 제목(축약), 상태
  - 5건 데이터 표시
  - "더보기" 링크로 WF-02-09 연결
