## 화면 구성

### 상단 영역 (좌측 33%)
- **감성 분포 파이차트**
  - 긍정: 12% (초록)
  - 중립: 35% (회색)
  - 부정: 53% (빨강)
  - 범례 및 백분율 표시

### 상단 영역 (중앙 33%)
- **부정 민원 트렌드 라인차트**
  - X축: 일별 (최근 30일)
  - Y축: 부정 민원 건수
  - 날짜별 추이 표시
  - 최고점 하이라이트

### 상단 영역 (우측 33%)
- **감정 강도 분포 히스토그램**
  - X축: 감정 강도 점수 (0~10)
  - Y축: 민원 건수
  - 막대 그래프
  - 5점 중심으로 좌우 분포

### 하단 영역 (좌측 50%)
- **워드클라우드**
  - 부정 민원의 핵심 키워드
  - 빈도가 높을수록 큰 글자로 표시
  - 색상: 부정도에 따라 색상 표시
  - 상위 20개 키워드

### 하단 영역 (우측 50%)
- **감성 점수 상/하위 민원 목록**
  - 좌측: 긍정 점수 상위 5건
  - 우측: 부정 점수 상위 5건
  - 각 항목: 점수, 제목, 접수일, 클릭 시 상세로 이동

## 사용 컴포넌트
- 파이차트 (Chart.js/D3.js)
- 라인차트 (Chart.js/D3.js)
- 히스토그램 (Chart.js/D3.js)
- 워드클라우드 (word-cloud 라이브러리)
- 목록 컴포넌트
- 공통 레이아웃 (WF-00-02)
