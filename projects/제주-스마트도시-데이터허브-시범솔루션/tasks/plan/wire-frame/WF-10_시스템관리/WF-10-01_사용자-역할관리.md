## 탭 1: 사용자 관리

### 상단 버튼
- "사용자 추가" 버튼 → 추가 모달

### 사용자 테이블
| 항목 | 설명 |
|------|------|
| 이름 | 사용자 이름 |
| 이메일 | 이메일 주소 |
| 부서 | 부서명 |
| 역할 | 관리자 / 운영자 / 열람자 (태그) |
| 상태 | 활성(초록) / 비활성(회색) 뱃지 |
| 최종로그인 | YYYY-MM-DD HH:MM |
| 액션 | 편집, 비활성화/활성화, 삭제 버튼 |

- 테이블 정렬 가능 (이름, 부서, 상태, 최종로그인)
- 필터링: 부서별, 역할별, 상태별
- 페이지네이션 또는 무한 스크롤

### 사용자 추가 모달
- 폼 필드:
  - 이름 * (Text)
  - 이메일 * (Email)
  - 부서 * (Dropdown)
  - 역할 * (Dropdown: 관리자/운영자/열람자)
  - 임시 비밀번호 생성 (체크박스, 기본 체크)

- "추가" 및 "취소" 버튼
- 추가 후 확인 메시지 (임시 비밀번호 표시/복사 기능)

---

## 탭 2: 역할 관리

### 역할 목록
- 역할명: 관리자 / 운영자 / 열람자
- 각 역할별 카드 또는 테이블 행 표시

### 역할별 권한 매트릭스 (그리드)
행: 메뉴 (대시보드, 데이터 수집, 데이터 카탈로그, 보고서, 시스템 관리, 감사)
열: 역할 (관리자, 운영자, 열람자)

|  | 조회 | 수정 | 삭제 |
|---|------|------|------|
| 대시보드 | ☑ | ☑ | ☑ (관리자만) |
| 데이터 수집 | ☑ | ☑ (운영자 이상) | ☑ (관리자만) |
| 데이터 카탈로그 | ☑ | ☐ | ☐ |
| 보고서 | ☑ | ☑ | ☑ (관리자만) |
| 시스템 관리 | ☐ | ☐ | ☐ (관리자만) |
| 감사 | ☑ | ☐ | ☐ (관리자만) |

- 체크박스 그리드 형태
- 권한 수정 가능
- "저장" 버튼 (역할별)
