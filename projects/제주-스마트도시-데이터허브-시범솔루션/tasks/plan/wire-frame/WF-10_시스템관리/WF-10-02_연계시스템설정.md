## 화면 구성
### 연계 시스템 카드 목록 (그리드 또는 리스트)
각 카드 구성:
- 시스템 로고 또는 아이콘
- 시스템명 (클릭 시 설정 패널 열기)
- 연결상태: 연결됨(초록), 오류(빨강), 미연결(회색) 뱃지
- 연계 방식: API / DB / 배치 (태그)
- 최종동기화: "2026-03-25 14:32" 또는 "아직 동기화 안 됨"
- 액션: 설정, 테스트 버튼

### 연계 시스템 종류 (7개)
1. 민원포털 (API)
2. 만덕콜센터 (DB 연계)
3. 통합관제 (API)
4. 기상청 (API)
5. 112/119 (배치)
6. 전기차충전 (API)
7. 스마트시티플랫폼 (API)

### 시스템 설정 패널 (우측 또는 모달)
카드 또는 "설정" 버튼 클릭 시 표시:

| 항목 | 타입 | 예시 값 |
|------|------|--------|
| 시스템명 | Text (읽기전용) | 민원포털 |
| URL * | URL | https://civil-complaints.jeju.go.kr/api/v1 |
| 인증 방식 * | Dropdown | API Key / OAuth 2.0 / Basic Auth |
| API Key / 사용자명 | Text | api_key_1234567890 |
| 비밀번호 | Password | ****** |
| 수집주기 * | Dropdown | 실시간 / 5분 / 1시간 / 1일 |
| 타임아웃 (초) | Number | 30 |
| 재시도 횟수 | Number | 3 |
| 재시도 간격 (초) | Number | 60 |
| 활성 | Toggle | On/Off |

### 상단 "연결 테스트" 버튼
- 클릭 시 현재 설정으로 연결 테스트
- 결과 메시지: "연결 성공" / "연결 실패: 타임아웃" 등

### 하단 전체 연계 현황 요약
표 형식:
| 시스템명 | 상태 | 최종동기화 | 수집주기 | 마지막 에러 |
|---------|------|----------|---------|-----------|
| 민원포털 | 연결 | 2026-03-25 14:32 | 실시간 | - |
| 기상청 | 오류 | 2026-03-25 13:15 | 1시간 | "Connection timeout" |

- 필터링: 상태별 (연결됨/오류/미연결)
- 정렬 가능
