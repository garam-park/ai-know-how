## 화면 구성
### 상단 필터 영역
| 필터항목 | 타입 | 예시 |
|---------|------|------|
| 기간 | 날짜범위 | 2026-03-01 ~ 2026-03-25 |
| 사용자 | Dropdown (다중선택) | 김수현, 박영민 |
| 행위유형 | Checkbox (다중선택) | ☑ 로그인, ☑ 조회, ☑ 수정, ☑ 삭제, ☑ 내보내기 |
| IP | Text | 192.168. (와일드카드 검색 지원) |

- "검색" 버튼
- "필터 초기화" 버튼

### 상단 우측 버튼
- "CSV 내보내기" 버튼 (선택된 필터 기준으로 내보내기)

### 감사 로그 테이블
| 항목 | 설명 |
|------|------|
| 시각 | YYYY-MM-DD HH:MM:SS |
| 사용자 | 사용자명 |
| IP | IP 주소 (예: 192.168.1.100) |
| 행위 | 로그인 / 조회 / 수정 / 삭제 / 내보내기 |
| 대상 | 대상 메뉴/데이터셋명 (예: "데이터 카탈로그 > 실시간 도로 정보") |
| 상세 | 상세 내용 (클릭 시 확장) |
| 결과 | 성공(초록) / 실패(빨강) 뱃지 |

### 상세 보기 (행 클릭 또는 확장)
- 행동에 대한 상세 정보:
  - 요청 데이터
  - 응답 상태
  - 에러 메시지 (있을 경우)
  - 실행 시간 (ms)

### 이상 접근 하이라이트
특정 조건에서 행의 배경색을 강조:
- **비정상 시간**: 자정~새벽 (00:00~06:00) 접근
- **반복 실패**: 5회 이상 연속 실패
- **대량 내보내기**: 1회에 100건 이상 내보내기

하이라이트 행: 살짝 노란색 또는 주황색 배경

### 페이지네이션 또는 무한 스크롤
- 대량 데이터 처리 고려 (가상 스크롤 권장)
- 페이지 크기 선택: 10개, 50개, 100개
