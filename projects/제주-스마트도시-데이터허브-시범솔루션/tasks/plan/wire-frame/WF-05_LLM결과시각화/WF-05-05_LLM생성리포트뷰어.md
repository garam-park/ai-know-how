## 화면 구성

### 상단 메타 영역
- **보고서 제목** (편집 불가 또는 편집 가능)
- **생성일** (자동 기입)
- **대상 기간** (시작일 ~ 종료일)
- **톤/스타일 선택 드롭다운**
  - 공식 톤 (formal)
  - 친절 톤 (friendly)
  - 간결 톤 (concise)

### 메인 콘텐츠 영역 (좌측)
- **보고서 본문** (HTML 렌더링)
  - 제목, 소제목, 본문 단락
  - 차트/그래프 삽입 영역
  - 테이블 렌더링
  - PII 마스킹 부분은 강조 표시 (하이라이트/배경색)

### 좌측 네비게이션 (보조)
- **목차** (사이드 네비게이션)
  - 보고서 섹션 목록
  - 클릭하여 해당 섹션으로 스크롤

### 우측 사이드패널
- **A/B 버전 비교**
  - 스타일 변형 탭
  - 예: "원본", "요약 강화", "시각화 강화"
  - 각 버전 미리보기
  - 선택하여 본문 교체

### 하단 액션 영역
- **내보내기 버튼**
  - PDF 다운로드
  - DOCX 다운로드
- **편집 모드 전환** (토글 버튼)

## 사용 컴포넌트

- 버튼 그룹 (내보내기)
