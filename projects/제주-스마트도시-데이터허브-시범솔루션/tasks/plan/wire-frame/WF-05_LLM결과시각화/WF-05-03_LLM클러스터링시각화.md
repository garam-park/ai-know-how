## 화면 구성

### 상단 영역
- **군집 수 조절 슬라이더** (2~20)
- **기간 필터** (FROM/TO 달력)

### 메인 콘텐츠 영역
- **버블차트** (2D 임베딩 시각화)
  - 각 버블 = 군집
  - 버블 크기 = 민원 건수
  - 버블 색상 = 주제 카테고리
  - 버블 위치 = 유사도 기반 2D 임베딩 좌표

### 군집 상세 사이드패널 (클릭 시 활성화)
- **핵심 키워드** 5개 (태그 형태)
- **대표 민원** 3건 (제목, 요약, 작성일)
- **관련 군집** 링크 (클릭하여 이동 가능)

### 하단 영역
- **군집 병합/분할 이력 타임라인**
  - 변경 일시
  - 변경 내용
  - 관련 통계
