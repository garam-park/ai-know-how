# WbsNode 모듈 구조 생성

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~15분 |
| AI 에이전트 | ~5분 |

## What (무엇을)
`apps/backend/src/modules/wbs/` 디렉토리를 생성하고 WbsModule을 AppModule에 등록합니다. WBS(Work Breakdown Structure)는 프로젝트의 작업을 계층적으로 구조화하는 핵심 모듈입니다.

## Why (왜)
WBS(일정 구조)는 프로젝트 관리의 기초이며, 진척률 추적과 역할 할당의 중심입니다. 모듈 구조를 먼저 정의하면 이후 API 구현이 체계적으로 진행됩니다.

## 선행 작업
- `04.권한-시스템/03.PermissionGuard-구현.md`

## 수행 가이드

### Step 1: 디렉토리 구조 생성
```bash
mkdir -p apps/backend/src/modules/wbs/{dto,entities}
```

### Step 2: WbsNode 엔티티/DTO 파일 생성
- `apps/backend/src/modules/wbs/entities/wbs-node.entity.ts`: Prisma 스키마 참조 (id, projectId, parentId, type, depth, order, code, title, description, startDate, endDate, progress, weight, assigneeId, projectCompanyId, timestamps, deletedAt)
- `apps/backend/src/modules/wbs/dto/wbs-node.dto.ts`: WbsNodeDto, WbsNodeTreeDto 정의
  - WbsNodeDto: 기본 노드 정보
  - WbsNodeTreeDto: children 배열 포함 (트리 표시용)

### Step 3: 서비스, 컨트롤러, 모듈 파일 생성

**`apps/backend/src/modules/wbs/wbs.service.ts`:**
```typescript
import { Injectable } from '@nestjs/common';
import { PrismaService } from 'src/prisma/prisma.service';

@Injectable()
export class WbsService {
  constructor(private prisma: PrismaService) {}

  async getWbsNode(nodeId: number) {
    return this.prisma.wbsNode.findUnique({
      where: { id: nodeId },
    });
  }
}
```

**`apps/backend/src/modules/wbs/wbs.controller.ts`:**
```typescript
import { Controller } from '@nestjs/common';
import { WbsService } from './wbs.service';

@Controller('api/v1/projects/:projectId/wbs-nodes')
export class WbsController {
  constructor(private wbsService: WbsService) {}
}
```

**`apps/backend/src/modules/wbs/wbs.module.ts`:**
```typescript
import { Module } from '@nestjs/common';
import { WbsService } from './wbs.service';
import { WbsController } from './wbs.controller';

@Module({
  controllers: [WbsController],
  providers: [WbsService],
  exports: [WbsService],
})
export class WbsModule {}
```

### Step 4: AppModule에 WbsModule 등록
`apps/backend/src/app.module.ts`의 imports 배열에 추가:
```typescript
import { WbsModule } from './modules/wbs/wbs.module';

@Module({
  imports: [
    // ...
    WbsModule,
  ],
})
export class AppModule {}
```

### Step 5: 빌드 검증
```bash
cd apps/backend
npm run build
```
