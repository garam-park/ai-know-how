# 프로젝트 생성 API

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~30분 |
| AI 에이전트 | ~10분 |

## What (무엇을)
POST `/api/v1/projects` 엔드포인트를 구현합니다. 새로운 프로젝트(사업)를 생성합니다. 프로젝트 생성 시 생성자의 회사를 자동으로 주관사(PRIME)로, 생성자를 주관사PM으로 등록합니다.

## Why (왜)
프로젝트 생성은 모든 협력 관리 기능의 시작점입니다. 프로젝트가 생성되면 자동으로 생성자가 주관사PM 역할로 배정되어 초기 권한 관리 구조가 수립됩니다.

### Step 1: CreateProjectDto 검증 규칙 추가
- 대상 파일: `apps/backend/src/modules/project/dto/project.dto.ts`
- 작업 내용:
  ```typescript
  import { IsString, IsOptional, IsDate, IsNumber, MaxLength } from 'class-validator';
  import { Type } from 'class-transformer';

  export class CreateProjectDto {
    @IsString()
    @MaxLength(255)
    name: string;

    @IsOptional()
    @IsString()
    @MaxLength(50)
    code?: string;

    @IsOptional()
    @IsString()
    description?: string;

    @IsDate()
    @Type(() => Date)
    startDate: Date;

    @IsDate()
    @Type(() => Date)
    endDate: Date;

    @IsOptional()
    @IsNumber()
    budget?: number;

    @IsOptional()
    @IsNumber()
    maxWbsDepth?: number;
  }
  ```

### Step 2: ProjectService에 create 메서드 구현
- 대상 파일: `apps/backend/src/modules/project/project.service.ts`
- 작업 내용:
  ```typescript
  async create(dto: CreateProjectDto, currentUser: any): Promise<ProjectDto> {
    // User의 Company 정보 조회
    const userWithCompany = await this.prisma.user.findUnique({
      where: { id: currentUser.id },
      include: { company: true },
    });

    if (!userWithCompany?.company) {
      throw new BadRequestException({
        code: 400000,
        message: '소속된 회사가 없어서 프로젝트를 생성할 수 없습니다.',
      });
    }

    // Transaction: Project + ProjectCompany + ProjectMember 동시 생성
    const result = await this.prisma.$transaction(async (tx) => {
      const project = await tx.project.create({
        data: {
          name: dto.name,
          code: dto.code,
          description: dto.description,
          startDate: dto.startDate,
          endDate: dto.endDate,
          budget: dto.budget,
          maxWbsDepth: dto.maxWbsDepth || 5,
        },
      });

      // 생성자의 Company를 주관사(PRIME)로 등록
      await tx.projectCompany.create({
        data: {
          projectId: project.id,
          companyId: userWithCompany.company.id,
          role: 'PRIME', // CompanyRole enum
          parentId: null,
        },
      });

      // 생성자를 주관사PM으로 등록
      // roleId는 시스템에서 정의된 PM role ID 필요 (별도 조회 또는 상수)
      const pmRole = await tx.role.findFirst({
        where: { code: 'PM' },
      });

      await tx.projectMember.create({
        data: {
          projectId: project.id,
          userId: currentUser.id,
          projectCompanyId: project.projectCompanies[0].id, // 방금 생성한 ProjectCompany
          roleId: pmRole.id,
          inputRate: 100,
        },
      });

      return project;
    });

    return {
      id: result.id,
      name: result.name,
      code: result.code,
      description: result.description,
      startDate: result.startDate,
      endDate: result.endDate,
      budget: result.budget,
      maxWbsDepth: result.maxWbsDepth,
      createdAt: result.createdAt,
      updatedAt: result.updatedAt,
    };
  }
  ```

### Step 3: ProjectController에 POST 엔드포인트 추가
- 대상 파일: `apps/backend/src/modules/project/project.controller.ts`
- 작업 내용:
  ```typescript
  import { Post, Body, UseGuards } from '@nestjs/common';
  import { CurrentUser } from '@/common/decorators/current-user.decorator';
  import { JwtAuthGuard } from '@/auth/guards/jwt-auth.guard';
  import { CreateProjectDto } from './dto/project.dto';

  @Controller('api/v1/projects')
  export class ProjectController {
    constructor(private readonly projectService: ProjectService) {}

    @Post()
    @UseGuards(JwtAuthGuard)
    async create(
      @Body() dto: CreateProjectDto,
      @CurrentUser() user: any,
    ): Promise<{
      code: number;
      result: { project: ProjectDto };
    }> {
      const project = await this.projectService.create(dto, user);
      return {
        code: 201000,
        result: { project },
      };
    }
  }
  ```

## 참고
- Prisma Transaction: 여러 작업을 원자성 있게 처리
- CompanyRole enum: PRIME(주관사), PARTNER(참여사), SUB(하청사) 등
- maxWbsDepth 기본값: 5
- CurrentUser 데코레이터: 현재 로그인한 사용자 정보 주입
