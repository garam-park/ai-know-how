# 페이지네이션 공통 컴포넌트

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~30분 |
| AI 에이전트 | ~12분 |

## What (무엇을)
페이지네이션 공통 컴포넌트를 만들고, 회사 목록과 프로젝트 목록에 적용합니다. 백엔드에서 이미 `page`, `limit`, `total` 파라미터를 지원하고 있으므로, 프론트엔드에서 이를 활용합니다.

## Why (왜)
현재 목록 페이지들은 전체 데이터를 한 번에 불러오므로, 데이터가 많아지면 성능 저하와 스크롤 문제가 발생합니다. 페이지네이션은 목록 UI의 기본 기능입니다.

## 선행 작업
- 없음 (독립 공통 컴포넌트)

## 수행 가이드

### Step 1: Pagination 컴포넌트 생성
- 대상 파일: `apps/frontend/src/components/Pagination.tsx` (신규)
- 작업 내용:
  ```typescript
  interface Props {
    page: number;
    totalPages: number;
    onPageChange: (page: number) => void;
  }

  export default function Pagination({ page, totalPages, onPageChange }: Props) {
    if (totalPages <= 1) return null;

    const pages: number[] = [];
    const start = Math.max(1, page - 2);
    const end = Math.min(totalPages, page + 2);
    for (let i = start; i <= end; i++) pages.push(i);

    return (
      <div style={{ display: 'flex', justifyContent: 'center', gap: 4, marginTop: 16 }}>
        <button disabled={page <= 1} onClick={() => onPageChange(page - 1)}
          style={{ padding: '6px 12px', border: '1px solid #e2e8f0', borderRadius: 4, cursor: page <= 1 ? 'default' : 'pointer', background: '#fff' }}>
          이전
        </button>
        {pages.map((p) => (
          <button key={p} onClick={() => onPageChange(p)}
            style={{ padding: '6px 12px', border: '1px solid #e2e8f0', borderRadius: 4, cursor: 'pointer',
              background: p === page ? '#2b6cb0' : '#fff', color: p === page ? '#fff' : '#4a5568' }}>
            {p}
          </button>
        ))}
        <button disabled={page >= totalPages} onClick={() => onPageChange(page + 1)}
          style={{ padding: '6px 12px', border: '1px solid #e2e8f0', borderRadius: 4, cursor: page >= totalPages ? 'default' : 'pointer', background: '#fff' }}>
          다음
        </button>
      </div>
    );
  }
  ```

### Step 2: 회사 목록에 페이지네이션 적용
- 대상 파일: `apps/frontend/src/pages/CompanyListPage.tsx`
- 작업 내용:
  ```typescript
  import Pagination from '../components/Pagination';

  const [page, setPage] = useState(1);
  const [totalPages, setTotalPages] = useState(1);
  const LIMIT = 20;

  const fetchCompanies = (kw?: string, p: number = 1) => {
    const params = new URLSearchParams();
    if (kw) params.set('keyword', kw);
    params.set('page', String(p));
    params.set('limit', String(LIMIT));

    api.get<{ companies: Company[]; total: number }>(`/companies?${params}`)
      .then((res) => {
        setCompanies(res.result.companies);
        setTotalPages(Math.ceil(res.result.total / LIMIT));
        setPage(p);
      })
      .catch(() => {});
  };

  // JSX 테이블 하단
  <Pagination page={page} totalPages={totalPages} onPageChange={(p) => fetchCompanies(keyword, p)} />
  ```
