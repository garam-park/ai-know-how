# 회사 상세 조회 및 수정 API

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~20분 |
| AI 에이전트 | ~8분 |

## What (무엇을)
GET `/api/v1/companies/:id` (상세 조회)와 PATCH `/api/v1/companies/:id` (수정) 엔드포인트를 구현합니다. 특정 회사의 상세 정보를 조회하고 회사 정보를 수정합니다.

## Why (왜)
등록된 회사의 정보를 보거나 수정이 필요할 때 사용합니다. 예를 들어 주소나 전화번호 변경, 사업자번호 추가 등의 업무에 필요합니다.

## 수행 가이드

### Step 1: CompanyService에 findById 메서드 구현
- 대상 파일: `apps/backend/src/modules/company/company.service.ts`
- 작업 내용:
  ```typescript
  async findById(id: string): Promise<CompanyDto> {
    const company = await this.prisma.company.findFirst({
      where: { id, deletedAt: null },
    });

    if (!company) {
      throw new NotFoundException({
        code: 404000,
        message: '회사 정보를 찾을 수 없습니다.',
      });
    }

    return {
      id: company.id,
      name: company.name,
      bizNo: company.bizNo,
      address: company.address,
      tel: company.tel,
      ownerId: company.ownerId,
      createdAt: company.createdAt,
      updatedAt: company.updatedAt,
    };
  }
  ```

### Step 2: CompanyService에 update 메서드 구현
- 대상 파일: `apps/backend/src/modules/company/company.service.ts`
- 작업 내용:
  ```typescript
  async update(id: string, dto: UpdateCompanyDto): Promise<CompanyDto> {
    // 존재 여부 확인
    await this.findById(id);

    // 사업자번호 변경 시 중복 체크
    if (dto.bizNo) {
      const existing = await this.prisma.company.findFirst({
        where: {
          bizNo: dto.bizNo,
          deletedAt: null,
          NOT: { id },
        },
      });
      if (existing) {
        throw new ConflictException({
          code: 409000,
          message: '이미 등록된 사업자번호입니다.',
        });
      }
    }

    const company = await this.prisma.company.update({
      where: { id },
      data: {
        ...(dto.name && { name: dto.name }),
        ...(dto.bizNo !== undefined && { bizNo: dto.bizNo }),
        ...(dto.address !== undefined && { address: dto.address }),
        ...(dto.tel !== undefined && { tel: dto.tel }),
      },
    });

    return {
      id: company.id,
      name: company.name,
      bizNo: company.bizNo,
      address: company.address,
      tel: company.tel,
      ownerId: company.ownerId,
      createdAt: company.createdAt,
      updatedAt: company.updatedAt,
    };
  }
  ```

### Step 3: CompanyController에 GET/:id 엔드포인트 추가
- 대상 파일: `apps/backend/src/modules/company/company.controller.ts`
- 작업 내용:
  ```typescript
  import { Get, Param } from '@nestjs/common';

  @Get(':id')
  async findById(@Param('id') id: string): Promise<{
    code: number;
    result: { company: CompanyDto };
  }> {
    const company = await this.companyService.findById(id);
    return {
      code: 200000,
      result: { company },
    };
  }
  ```

### Step 4: CompanyController에 PATCH/:id 엔드포인트 추가
- 대상 파일: `apps/backend/src/modules/company/company.controller.ts`
- 작업 내용:
  ```typescript
  import { Patch, Body } from '@nestjs/common';

  @Patch(':id')
  async update(
    @Param('id') id: string,
    @Body() dto: UpdateCompanyDto,
  ): Promise<{
    code: number;
    result: { company: CompanyDto };
  }> {
    const company = await this.companyService.update(id, dto);
    return {
      code: 200000,
      result: { company },
    };
  }
  ```
