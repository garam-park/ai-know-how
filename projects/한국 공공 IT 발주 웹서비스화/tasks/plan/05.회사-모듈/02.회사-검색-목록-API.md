# 회사 검색 목록 API

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~30분 |
| AI 에이전트 | ~10분 |

## What (무엇을)
GET `/api/v1/companies?keyword={keyword}&limit={limit}&offset={offset}` 엔드포인트를 구현합니다. 회사명으로 검색하여 기존 회사 목록을 반환합니다. 프로젝트에 컨소시엠사를 등록할 때 기존 회사를 검색하는 데 사용됩니다.

## Why (왜)
프로젝트에 참여사나 하청사를 추가할 때, 이미 시스템에 등록된 회사를 빠르게 찾아서 선택할 수 있어야 합니다. 회사명 검색 기능으로 입력 효율성을 높입니다.

## 수행 가이드

### Step 1: CompanyService에 search 메서드 구현
- 대상 파일: `apps/backend/src/modules/company/company.service.ts`
- 작업 내용:
  ```typescript
  async search(keyword: string, limit = 10, offset = 0): Promise<{
    companies: CompanyDto[];
    totalCount: number;
  }> {
    const where = {
      AND: [
        { deletedAt: null },
        keyword ? {
          name: {
            contains: keyword,
            mode: 'insensitive' as const,
          }
        } : {},
      ]
    };

    const [companies, totalCount] = await Promise.all([
      this.prisma.company.findMany({
        where,
        take: limit,
        skip: offset,
        orderBy: { name: 'asc' },
      }),
      this.prisma.company.count({ where }),
    ]);

    return {
      companies: companies.map(c => ({
        id: c.id,
        name: c.name,
        bizNo: c.bizNo,
        address: c.address,
        tel: c.tel,
        ownerId: c.ownerId,
        createdAt: c.createdAt,
        updatedAt: c.updatedAt,
      })),
      totalCount,
    };
  }
  ```

### Step 2: CompanyController에 search 엔드포인트 추가
- 대상 파일: `apps/backend/src/modules/company/company.controller.ts`
- 작업 내용:
  ```typescript
  import { Controller, Get, Query } from '@nestjs/common';
  import { CompanyService } from './company.service';
  import { CompanyDto } from './dto/company.dto';

  @Controller('api/v1/companies')
  export class CompanyController {
    constructor(private readonly companyService: CompanyService) {}

    @Get()
    async search(
      @Query('keyword') keyword?: string,
      @Query('limit') limit?: number,
      @Query('offset') offset?: number,
    ): Promise<{
      code: number;
      result: { companies: CompanyDto[]; totalCount: number };
    }> {
      const result = await this.companyService.search(
        keyword || '',
        limit ? parseInt(limit as any, 10) : 10,
        offset ? parseInt(offset as any, 10) : 0,
      );
      return {
        code: 200000,
        result,
      };
    }
  }
  ```

## 참고
- Prisma `findMany`: 다중 조회, offset/limit 기반 pagination
- `contains` with `insensitive` mode: 대소문자 무시 부분 일치 검색
- soft delete: deletedAt이 null인 레코드만 조회
