# 권한·WBS 시나리오 E2E 테스트

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~40분 |
| AI 에이전트 | ~15분 |

## What (무엇을)
권한 분기 및 WBS 핵심 기능 E2E 시나리오를 작성합니다. 주관사PM의 프로젝트 생성, 참여사PM의 WBS 조회, 실무자의 TASK 편집 권한 등을 검증합니다.

## Why (왜)
가장 복잡한 비즈니스 로직을 검증합니다. 권한 관리가 제대로 되지 않으면 데이터 무결성이 손상될 수 있습니다.

### Step 1: 권한 기반 프로젝트 생성 시나리오
`e2e/permissions.projectCreation.spec.ts` 파일을 생성합니다:

```typescript
import { test, expect } from '@playwright/test';
import { login, expectToast, navigateToProject } from './utils/helpers';

test.describe('권한: 프로젝트 생성', () => {
  test('PM은 새 프로젝트를 생성할 수 있다', async ({ page }) => {
    await login(page, 'pm@example.com', 'Test1234!');

    // 프로젝트 목록 페이지
    await expect(page).toHaveURL(/\/projects$/);

    // "새 프로젝트" 버튼 보이는지 확인
    const createButton = page.locator('button:has-text("+ 새 프로젝트")');
    await expect(createButton).toBeVisible();

    // 클릭해서 모달 열기
    await createButton.click();

    // 프로젝트 정보 입력
    const projectName = `E2E 프로젝트 ${Date.now()}`;
    await page.fill('input[placeholder*="프로젝트명"]', projectName);
    await page.fill('input[placeholder*="코드"]', 'E2E-001');

    // 날짜 입력
    const today = new Date().toISOString().split('T')[0];
    const nextMonth = new Date(Date.now() + 30 * 24 * 60 * 60 * 1000)
      .toISOString()
      .split('T')[0];

    await page.fill('input[name="startDate"]', today);
    await page.fill('input[name="endDate"]', nextMonth);

    // 생성
    await page.click('button:has-text("생성")');

    // 성공 메시지 확인
    await expectToast(page, '프로젝트가 생성되었습니다');

    // 목록에서 새 프로젝트 보이는지 확인
    await expect(page.locator(`text=${projectName}`)).toBeVisible();
  });

  test('일반 사용자는 새 프로젝트 버튼을 볼 수 없다', async ({ page }) => {
    // 실무자로 로그인
    await login(page, 'staff@example.com', 'Test1234!');

    // 프로젝트 목록 페이지
    await expect(page).toHaveURL(/\/projects$/);

    // "새 프로젝트" 버튼이 없어야 함
    const createButton = page.locator('button:has-text("+ 새 프로젝트")');
    await expect(createButton).not.toBeVisible();
  });

  test('PM이 생성한 프로젝트에는 자동으로 주관사가 추가된다', async ({
    page,
  }) => {
    await login(page, 'pm@example.com', 'Test1234!');

    // 프로젝트 생성
    await page.locator('button:has-text("+ 새 프로젝트")').click();
    const projectName = `권한 테스트 ${Date.now()}`;
    await page.fill('input[placeholder*="프로젝트명"]', projectName);
    await page.fill('input[name="startDate"]', new Date().toISOString().split('T')[0]);
    await page.fill('input[name="endDate"]',
      new Date(Date.now() + 30 * 24 * 60 * 60 * 1000).toISOString().split('T')[0]);

    await page.click('button:has-text("생성")');
    await expectToast(page, '프로젝트가 생성되었습니다');

    // 새 프로젝트로 이동
    await page.click(`text=${projectName}`);

    // 컨소시엄 탭에 주관사가 있는지 확인
    await page.click('text=컨소시엄');
    const consortiumCompany = page.locator('text=주관사');
    // 주관사가 보이거나, PM의 회사가 자동 추가되어야 함
  });
});
```

### Step 2: WBS 권한 시나리오
`e2e/permissions.wbs.spec.ts` 파일을 생성합니다:

```typescript
import { test, expect } from '@playwright/test';
import { login, navigateToProject } from './utils/helpers';

test.describe('권한: WBS 편집', () => {
  test('PM은 모든 WBS를 볼 수 있다', async ({ page }) => {
    await login(page, 'pm@example.com', 'Test1234!');

    // E2E 테스트 프로젝트로 이동
    await navigateToProject(page, 'E2E 테스트 프로젝트');

    // WBS 탭으로 이동
    await page.click('text=WBS');

    // WBS 테이블이 로드되고 모든 항목이 보임
    await page.waitForSelector('[data-testid="wbs-table"]');

    const rows = page.locator('[data-testid="wbs-row"]');
    const count = await rows.count();
    expect(count).toBeGreaterThan(0);

    // 상위 카테고리와 하위 작업이 모두 보여야 함
    const categoryRow = page.locator('[data-testid="wbs-row"]:has-text("요구사항 분석")');
    await expect(categoryRow).toBeVisible();
  });

  test('실무자는 자신의 회사의 TASK만 편집 가능하다', async ({ page }) => {
    await login(page, 'staff@example.com', 'Test1234!');

    // E2E 테스트 프로젝트로 이동
    await navigateToProject(page, 'E2E 테스트 프로젝트');

    // WBS 탭으로 이동
    await page.click('text=WBS');

    // 테이블 로드 대기
    await page.waitForSelector('[data-testid="wbs-table"]');

    // 자신의 회사(참여사)의 작업은 진척률 슬라이더가 활성화되어야 함
    const myCompanyRow = page.locator(
      '[data-testid="wbs-row"]:has-text("기본 요구사항 수집")'
    );
    const slider = myCompanyRow.locator('input[type="range"]');
    await expect(slider).toBeEnabled();

    // 다른 회사의 작업 슬라이더는 비활성화되어야 함 (또는 숨겨짐)
  });

  test('PM은 WBS 구조를 변경할 수 있다', async ({ page }) => {
    await login(page, 'pm@example.com', 'Test1234!');

    // E2E 테스트 프로젝트로 이동
    await navigateToProject(page, 'E2E 테스트 프로젝트');

    // WBS 탭으로 이동
    await page.click('text=WBS');

    // "작업 추가" 버튼 보이는지 확인
    const addButton = page.locator('button:has-text("+ 작업 추가")');
    await expect(addButton).toBeVisible();

    // 클릭해서 모달 열기
    await addButton.click();

    // 새 작업 입력
    await page.fill('input[placeholder*="코드"]', 'WBS-003');
    await page.fill('input[placeholder*="제목"]', '새로운 작업');

    // 추가
    await page.click('button:has-text("추가")');

    // 테이블에 새 작업 추가됨을 확인
    const newTask = page.locator('text=새로운 작업');
    await expect(newTask).toBeVisible();
  });

  test('실무자는 WBS 구조 변경 버튼을 볼 수 없다', async ({ page }) => {
    await login(page, 'staff@example.com', 'Test1234!');

    // E2E 테스트 프로젝트로 이동
    await navigateToProject(page, 'E2E 테스트 프로젝트');

    // WBS 탭으로 이동
    await page.click('text=WBS');

    // "작업 추가", "작업 삭제" 버튼이 없어야 함
    const addButton = page.locator('button:has-text("+ 작업 추가")');
    await expect(addButton).not.toBeVisible();

    const deleteButtons = page.locator('button:has-text("삭제")');
    const count = await deleteButtons.count();
    expect(count).toBe(0);
  });
});
```

### Step 3: 진척률 실시간 업데이트 시나리오
`e2e/wbs.progressUpdate.spec.ts` 파일을 생성합니다:

```typescript
import { test, expect } from '@playwright/test';
import { login, navigateToProject } from './utils/helpers';

test.describe('WBS: 진척률 업데이트', () => {
  test('TASK 진척률 변경 시 상위 CATEGORY도 자동 업데이트된다', async ({
    page,
  }) => {
    await login(page, 'staff@example.com', 'Test1234!');

    // E2E 테스트 프로젝트로 이동
    await navigateToProject(page, 'E2E 테스트 프로젝트');

    // WBS 탭으로 이동
    await page.click('text=WBS');

    // 테이블 로드 대기
    await page.waitForSelector('[data-testid="wbs-table"]');

    // "기본 요구사항 수집" TASK의 진척률 슬라이더 찾기
    const taskRow = page.locator(
      '[data-testid="wbs-row"]:has-text("기본 요구사항 수집")'
    );
    const slider = taskRow.locator('input[type="range"]');

    // 초기 진척률 확인
    const initialValue = await slider.inputValue();
    expect(initialValue).toBeDefined();

    // 슬라이더를 50%로 변경
    await slider.fill('50');

    // API 호출이 완료될 때까지 대기 (진척률 텍스트 업데이트)
    const progressText = taskRow.locator('[data-testid="wbs-progress-text"]');
    await expect(progressText).toContainText('50%');

    // 상위 CATEGORY("요구사항 분석")의 진척률도 업데이트되었는지 확인
    const categoryRow = page.locator(
      '[data-testid="wbs-row"]:has-text("요구사항 분석")'
    );
    // 계산된 진척률이 반영되어야 함
  });

  test('동시에 여러 TASK의 진척률을 변경할 수 있다', async ({ page }) => {
    await login(page, 'staff@example.com', 'Test1234!');

    // E2E 테스트 프로젝트로 이동
    await navigateToProject(page, 'E2E 테스트 프로젝트');

    // WBS 탭으로 이동
    await page.click('text=WBS');

    // 여러 슬라이더 변경
    const sliders = page.locator('input[type="range"]');
    const sliderCount = await sliders.count();

    if (sliderCount > 0) {
      // 첫 번째 슬라이더를 30%로
      await sliders.nth(0).fill('30');

      // 두 번째 슬라이더를 60%로 (있으면)
      if (sliderCount > 1) {
        await sliders.nth(1).fill('60');
      }

      // 모든 변경이 반영되었는지 확인
      await page.waitForTimeout(1000); // API 응답 대기

      const values = await sliders.allInputValues();
      expect(values[0]).toBe('30');
      if (sliderCount > 1) {
        expect(values[1]).toBe('60');
      }
    }
  });

  test('진척률 변경 시 대시보드가 업데이트된다', async ({ page }) => {
    await login(page, 'staff@example.com', 'Test1234!');

    // E2E 테스트 프로젝트로 이동
    await navigateToProject(page, 'E2E 테스트 프로젝트');

    // 초기 대시보드 진척률 확인
    await page.click('text=대시보드');
    const initialProgress = await page.locator(
      '[data-testid="overall-progress"]'
    ).textContent();

    // WBS 탭으로 이동
    await page.click('text=WBS');

    // 진척률 변경
    const slider = page.locator('input[type="range"]').first();
    await slider.fill('100');

    // 다시 대시보드로
    await page.click('text=대시보드');

    // 진척률이 업데이트되었는지 확인
    const updatedProgress = await page.locator(
      '[data-testid="overall-progress"]'
    ).textContent();

    expect(updatedProgress).not.toBe(initialProgress);
  });
});
```

### Step 4: 통합 시나리오
`e2e/integration.fullWorkflow.spec.ts` 파일을 생성합니다:

```typescript
import { test, expect } from '@playwright/test';
import { login, expectToast, navigateToProject } from './utils/helpers';

test.describe('통합 시나리오: 전체 워크플로우', () => {
  test('주관사PM이 프로젝트를 생성하고 관리할 수 있다', async ({ page }) => {
    // 주관사PM으로 로그인
    await login(page, 'pm@example.com', 'Test1234!');

    // 프로젝트 생성
    await page.locator('button:has-text("+ 새 프로젝트")').click();
    const projectName = `통합테스트 프로젝트 ${Date.now()}`;
    await page.fill('input[placeholder*="프로젝트명"]', projectName);
    await page.fill('input[name="startDate"]',
      new Date().toISOString().split('T')[0]);
    await page.fill('input[name="endDate"]',
      new Date(Date.now() + 90 * 24 * 60 * 60 * 1000).toISOString().split('T')[0]);

    await page.click('button:has-text("생성")');
    await expectToast(page, '프로젝트가 생성되었습니다');

    // 새 프로젝트로 이동
    await page.click(`text=${projectName}`);
    await page.waitForURL(/\/projects\/\w+$/);

    // 컨소시엄에 참여사 추가
    await page.click('text=컨소시엄');
    await page.locator('button:has-text("+ 회사 추가")').click();
    await page.fill('input[placeholder*="회사명"]', '참여사 A');
    await page.click('button:has-text("추가")');
    await expectToast(page, '회사가 추가되었습니다');

    // 멤버 초대
    await page.click('text=멤버 관리');
    await page.locator('button:has-text("+ 멤버 초대")').click();
    await page.fill('input[type="email"]', 'staff@example.com');

    // 회사 선택 드롭다운
    const companySelect = page.locator('select').nth(0);
    await companySelect.selectOption({ label: '참여사 A' });

    await page.click('button:has-text("초대")');
    await expectToast(page, '멤버 초대가 완료되었습니다');

    // WBS 작업 추가
    await page.click('text=WBS');
    await page.locator('button:has-text("+ 작업 추가")').click();
    await page.fill('input[placeholder*="코드"]', 'WBS-001');
    await page.fill('input[placeholder*="제목"]', '기본 설계');
    await page.click('button:has-text("추가")');
    await expectToast(page, '작업이 추가되었습니다');

    // 진척률 업데이트
    const slider = page.locator('input[type="range"]').first();
    await slider.fill('50');

    // 대시보드 확인
    await page.click('text=대시보드');
    await expect(page.locator('[data-testid="overall-progress"]'))
      .toContainText('50');
  });
});
```

## 참고
- 각 권한 수준별로 별도의 테스트 계정 사용
- 동시성(concurrency) 테스트로 race condition 확인
- UI 엘리먼트의 존재/부재로 권한 검증
- API 응답 대기 후 상태 확인
- 실제 데이터 처리가 정상인지 확인 (단순 UI 검증 아님)
