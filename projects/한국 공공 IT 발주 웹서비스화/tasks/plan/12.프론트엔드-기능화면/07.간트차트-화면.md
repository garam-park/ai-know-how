# 간트차트 화면

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~60분 |
| AI 에이전트 | ~25분 |

## What (무엇을)
WBS 데이터를 간트차트로 시각화합니다. Y축에 WBS 노드 목록, X축에 시간축을 배치하고, 각 작업의 바는 진척률을 색상으로 표시합니다.

## Why (왜)
일정 현황을 시각적으로 파악하는 핵심 뷰입니다. 프로젝트의 전체 타임라인과 진척도를 한눈에 볼 수 있습니다.

### Step 2: GanttPage 컴포넌트 생성
`apps/frontend/src/pages/projects/GanttPage.tsx` 파일을 생성합니다:

```typescript
import React, { useEffect, useState, useRef } from 'react';
import { useProjectContext } from '../../hooks/useProjectContext';
import { wbsService, WBSNode } from '../../services/wbsService';
import GanttChart from '../../components/GanttChart';
import Loading from '../../components/Loading';
import './GanttPage.css';

interface GanttNodeData extends WBSNode {
  barWidth: number;
  barOffset: number;
  progressColor: string;
}

const GanttPage: React.FC = () => {
  const { project } = useProjectContext();
  const [wbsData, setWBSData] = useState<WBSNode[]>([]);
  const [ganttData, setGanttData] = useState<GanttNodeData[]>([]);
  const [loading, setLoading] = useState(true);
  const [zoomLevel, setZoomLevel] = useState(1);

  useEffect(() => {
    fetchWBS();
  }, [project.id]);

  useEffect(() => {
    if (wbsData.length > 0) {
      const flat = flattenWBSTree(wbsData);
      const processed = processForGantt(flat, project);
      setGanttData(processed);
    }
  }, [wbsData, project]);

  const fetchWBS = async () => {
    try {
      setLoading(true);
      const data = await wbsService.getWBSTree(project.id);
      setWBSData(data);
    } catch (err) {
      console.error('WBS 로드 실패:', err);
    } finally {
      setLoading(false);
    }
  };

  if (loading) {
    return <Loading message="간트차트 로딩 중..." />;
  }

  return (
    <div className="gantt-page">
      <div className="gantt-page__header">
        <h2>간트차트</h2>
        <div className="gantt-page__controls">
          <label>
            확대:
            <input
              type="range"
              min="0.5"
              max="3"
              step="0.5"
              value={zoomLevel}
              onChange={(e) => setZoomLevel(parseFloat(e.target.value))}
            />
            <span>{Math.round(zoomLevel * 100)}%</span>
          </label>
        </div>
      </div>

      <GanttChart
        data={ganttData}
        projectStart={new Date(project.startDate)}
        projectEnd={new Date(project.endDate)}
        zoomLevel={zoomLevel}
      />
    </div>
  );
};

function flattenWBSTree(nodes: WBSNode[]): WBSNode[] {
  let result: WBSNode[] = [];
  for (const node of nodes) {
    result.push(node);
    if (node.children) {
      result = result.concat(flattenWBSTree(node.children));
    }
  }
  return result;
}

function processForGantt(
  nodes: WBSNode[],
  project: any
): GanttNodeData[] {
  const projectStart = new Date(project.startDate).getTime();
  const projectEnd = new Date(project.endDate).getTime();
  const projectDuration = projectEnd - projectStart;

  return nodes.map((node) => {
    const nodeStart = node.startDate
      ? new Date(node.startDate).getTime()
      : projectStart;
    const nodeEnd = node.endDate
      ? new Date(node.endDate).getTime()
      : projectEnd;

    const barOffset = ((nodeStart - projectStart) / projectDuration) * 100;
    const barWidth = ((nodeEnd - nodeStart) / projectDuration) * 100;
    const progressColor = getProgressColor(node.progress);

    return {
      ...node,
      barWidth,
      barOffset,
      progressColor,
    };
  });
}

function getProgressColor(progress: number): string {
  if (progress === 100) return '#4caf50'; // 초록색 (완료)
  if (progress >= 50) return '#2196f3'; // 파란색 (진행중)
  if (progress > 0) return '#ff9800'; // 주황색 (시작)
  return '#ccc'; // 회색 (미시작)
}

export default GanttPage;
```

### Step 3: GanttChart 컴포넌트 생성
`apps/frontend/src/components/GanttChart.tsx` 파일을 생성합니다:

```typescript
import React, { useMemo } from 'react';
import { WBSNode } from '../services/wbsService';
import './GanttChart.css';

interface GanttChartData extends WBSNode {
  barWidth: number;
  barOffset: number;
  progressColor: string;
}

interface GanttChartProps {
  data: GanttChartData[];
  projectStart: Date;
  projectEnd: Date;
  zoomLevel: number;
}

const GanttChart: React.FC<GanttChartProps> = ({
  data,
  projectStart,
  projectEnd,
  zoomLevel,
}) => {
  const today = new Date();
  const projectDuration = projectEnd.getTime() - projectStart.getTime();
  const dayCount = Math.ceil(projectDuration / (1000 * 60 * 60 * 24));
  const todayOffset =
    ((today.getTime() - projectStart.getTime()) / projectDuration) * 100;

  // 월 단위 열 생성
  const monthColumns = useMemo(() => {
    const columns = [];
    let current = new Date(projectStart);
    while (current <= projectEnd) {
      columns.push(new Date(current));
      current.setMonth(current.getMonth() + 1);
    }
    return columns;
  }, [projectStart, projectEnd]);

  return (
    <div className="gantt-chart">
      <div className="gantt-chart__container">
        {/* 왼쪽 패널: WBS 목록 */}
        <div className="gantt-chart__left">
          <div className="gantt-chart__header">
            <div className="gantt-chart__header-row">
              <div className="gantt-chart__code">코드</div>
              <div className="gantt-chart__title">제목</div>
              <div className="gantt-chart__company">담당회사</div>
            </div>
          </div>
          <div className="gantt-chart__body-left">
            {data.map((node) => (
              <div
                key={node.id}
                className={`gantt-chart__row ${
                  node.type === 'CATEGORY' ? 'gantt-chart__row--category' : ''
                }`}
                style={{ paddingLeft: `${node.depth * 20}px` }}
              >
                <div className="gantt-chart__code">{node.code}</div>
                <div className="gantt-chart__title">
                  <span className="gantt-chart__icon">
                    {node.type === 'CATEGORY' ? '📁' : '📝'}
                  </span>
                  {node.title}
                </div>
                <div className="gantt-chart__company">
                  {node.responsibleCompany || '-'}
                </div>
              </div>
            ))}
          </div>
        </div>

        {/* 오른쪽 패널: 간트차트 */}
        <div
          className="gantt-chart__right"
          style={{ '--zoom-level': zoomLevel } as React.CSSProperties}
        >
          {/* 헤더: 월 표시 */}
          <div className="gantt-chart__header">
            <div className="gantt-chart__months">
              {monthColumns.map((month, idx) => (
                <div
                  key={idx}
                  className="gantt-chart__month"
                  style={{
                    width: `${((month.getTime() + 30 * 24 * 60 * 60 * 1000 - projectStart.getTime()) / projectDuration) * 100}%`,
                  }}
                >
                  {month.toLocaleDateString('ko-KR', {
                    year: 'numeric',
                    month: 'short',
                  })}
                </div>
              ))}
            </div>
          </div>

          {/* 본문: 간트 바 */}
          <div className="gantt-chart__body-right">
            {/* 오늘 선 */}
            <div
              className="gantt-chart__today-line"
              style={{ left: `${todayOffset}%` }}
            />

            {data.map((node) => (
              <div
                key={node.id}
                className={`gantt-chart__row ${
                  node.type === 'CATEGORY' ? 'gantt-chart__row--category' : ''
                }`}
              >
                {/* 진척률 바 */}
                <div
                  className="gantt-chart__bar-container"
                  style={{
                    left: `${node.barOffset}%`,
                    width: `${node.barWidth}%`,
                  }}
                >
                  {node.barWidth > 5 && (
                    <div
                      className="gantt-chart__bar"
                      style={{
                        backgroundColor: node.progressColor,
                        width: `${node.progress}%`,
                      }}
                      title={`${node.title} - ${node.progress}%`}
                    >
                      {node.barWidth > 15 && (
                        <span className="gantt-chart__bar-text">
                          {node.progress}%
                        </span>
                      )}
                    </div>
                  )}
                </div>
              </div>
            ))}
          </div>
        </div>
      </div>

      {/* 범례 */}
      <div className="gantt-chart__legend">
        <div className="gantt-chart__legend-item">
          <div
            className="gantt-chart__legend-color"
            style={{ backgroundColor: '#4caf50' }}
          />
          <span>완료</span>
        </div>
        <div className="gantt-chart__legend-item">
          <div
            className="gantt-chart__legend-color"
            style={{ backgroundColor: '#2196f3' }}
          />
          <span>진행중</span>
        </div>
        <div className="gantt-chart__legend-item">
          <div
            className="gantt-chart__legend-color"
            style={{ backgroundColor: '#ff9800' }}
          />
          <span>시작</span>
        </div>
        <div className="gantt-chart__legend-item">
          <div
            className="gantt-chart__legend-color"
            style={{ backgroundColor: '#ccc' }}
          />
          <span>미시작</span>
        </div>
      </div>
    </div>
  );
};

export default GanttChart;
```

## 완료 기준
- [ ] WBS 데이터가 간트차트로 렌더링
- [ ] Y축: WBS 노드 목록 (트리 구조 반영, 깊이별 들여쓰기)
- [ ] X축: 프로젝트 기간 (월 단위 구분)
- [ ] 각 작업 바: 진척률로 채워짐 (색상 구분: 완료/진행중/시작/미시작)
- [ ] 오늘 날짜 마커 표시 (빨간 수직선)
- [ ] CATEGORY는 약간 다른 배경색으로 구분
- [ ] 범례 표시 (색상별 상태)
- [ ] 확대/축소 기능 (슬라이더)
- [ ] 가로 스크롤로 전체 기간 조회 가능

## 참고
- 진척률은 채색된 바의 너비로 표현
- 각 월을 구분선으로 명확하게 표시
- 반응형 레이아웃: 좌측 목록 고정, 우측 스크롤
- 클릭 시 상세 정보 팝업 추가 가능 (향후 기능)
