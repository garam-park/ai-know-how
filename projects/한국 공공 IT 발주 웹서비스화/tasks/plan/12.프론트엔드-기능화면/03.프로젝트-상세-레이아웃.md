# 프로젝트 상세 레이아웃

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~30분 |
| AI 에이전트 | ~10분 |

## What (무엇을)
/projects/:id 페이지에서 프로젝트 정보를 헤더에 표시하고, 탭 네비게이션(대시보드/WBS/멤버/컨소시엄)을 통해 하위 기능들을 표시합니다. 사용자 권한에 따라 탭을 표시/숨김 처리합니다.

## Why (왜)
프로젝트 상세 페이지의 컨테이너 역할을 하며, 모든 프로젝트 관련 기능을 통합합니다.

## 선행 작업
- `12.프론트엔드-기능화면/01.프로젝트-목록-화면.md`

## 수행 가이드

### Step 1: ProjectDetailPage 컴포넌트 생성
`apps/frontend/src/pages/projects/ProjectDetailPage.tsx` 파일을 생성합니다:

```typescript
import React, { useEffect, useState } from 'react';
import { useParams, Outlet, useNavigate, useLocation } from 'react-router-dom';
import { projectService, ProjectDto } from '../../services/projectService';
import { useAuth } from '../../hooks/useAuth';
import Loading from '../../components/Loading';
import './ProjectDetailPage.css';

const ProjectDetailPage: React.FC = () => {
  const { id } = useParams<{ id: string }>();
  const navigate = useNavigate();
  const location = useLocation();
  const { user } = useAuth();

  const [project, setProject] = useState<ProjectDto | null>(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);

  useEffect(() => {
    if (!id) return;

    const fetchProject = async () => {
      try {
        setLoading(true);
        const data = await projectService.getProjectById(id);
        setProject(data);
      } catch (err) {
        setError(
          err instanceof Error ? err.message : '프로젝트를 불러올 수 없습니다'
        );
      } finally {
        setLoading(false);
      }
    };

    fetchProject();
  }, [id]);

  const tabs = [
    { label: '대시보드', path: `/projects/${id}/dashboard` },
    { label: 'WBS', path: `/projects/${id}/wbs` },
    { label: '멤버 관리', path: `/projects/${id}/members` },
  ];

  // PM 또는 ADMIN만 컨소시엄 탭 표시
  if (user?.role === 'ADMIN' || user?.role === 'PM') {
    tabs.push({ label: '컨소시엄', path: `/projects/${id}/consortium` });
  }

  const currentTabPath = location.pathname;

  if (loading) {
    return <Loading message="프로젝트 로딩 중..." />;
  }

  if (error || !project) {
    return <div className="project-detail__error">{error || '프로젝트를 찾을 수 없습니다'}</div>;
  }

  return (
    <div className="project-detail">
      <div className="project-detail__header">
        <div className="project-detail__info">
          <h1 className="project-detail__title">{project.name}</h1>
          <p className="project-detail__meta">
            <span>{new Date(project.startDate).toLocaleDateString('ko-KR')}</span>
            <span>~</span>
            <span>{new Date(project.endDate).toLocaleDateString('ko-KR')}</span>
            <span>|</span>
            <span>진척도: {project.progress}%</span>
          </p>
        </div>
        <div className="project-detail__header-actions">
          <button
            className="project-detail__back-button"
            onClick={() => navigate('/projects')}
          >
            ← 목록으로
          </button>
        </div>
      </div>

      <div className="project-detail__tabs">
        <nav className="project-detail__tab-nav">
          {tabs.map((tab) => (
            <button
              key={tab.path}
              className={`project-detail__tab-button ${
                currentTabPath === tab.path ? 'project-detail__tab-button--active' : ''
              }`}
              onClick={() => navigate(tab.path)}
            >
              {tab.label}
            </button>
          ))}
        </nav>
      </div>

      <div className="project-detail__content">
        <Outlet context={{ project }} />
      </div>
    </div>
  );
};

export default ProjectDetailPage;
```

### Step 3: useOutletContext 훅 생성 (선택)
`apps/frontend/src/hooks/useProjectContext.ts` 파일을 생성합니다:

```typescript
import { useOutletContext } from 'react-router-dom';
import { ProjectDto } from '../services/projectService';

export const useProjectContext = () => {
  return useOutletContext<{ project: ProjectDto }>();
};
```

## 참고
- useOutletContext: 부모 컴포넌트에서 자식으로 데이터 전달
- 각 탭 페이지는 독립적인 컴포넌트로 구현
- 권한(user.role)에 따른 탭 표시/숨김
- URL 기반 탭 활성화로 북마크 가능
