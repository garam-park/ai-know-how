# 프로젝트 수정/삭제 기능

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~35분 |
| AI 에이전트 | ~12분 |

## What (무엇을)
`ProjectDetailPage`에 프로젝트 정보 수정 기능과 삭제 기능을 추가합니다. 프로젝트 이름, 코드, 설명, 기간, 예산을 수정할 수 있고, 프로젝트를 소프트 삭제(soft delete)할 수 있습니다.

## Why (왜)
현재 프로젝트 상세 페이지는 정보를 읽기만 가능하고 수정할 수 없습니다. 사업 기간 변경, 예산 조정, 잘못 생성한 프로젝트 삭제 등은 일상적인 운영 작업입니다.

## 선행 작업
- 없음 (백엔드 PATCH/DELETE `/api/v1/projects/:id` 이미 구현 완료)

## 수행 가이드

### Step 1: 프로젝트 수정 모달 컴포넌트 생성
- 대상 파일: `apps/frontend/src/components/ProjectEditModal.tsx` (신규)
- 작업 내용:
  ```typescript
  import { useState, useEffect } from 'react';
  import { api } from '../services/api';

  interface ProjectDetail {
    id: number;
    name: string;
    code?: string;
    description?: string;
    startDate: string;
    endDate: string;
    budget?: number;
  }

  interface Props {
    project: ProjectDetail;
    open: boolean;
    onClose: () => void;
    onUpdated: (updated: ProjectDetail) => void;
  }

  export default function ProjectEditModal({ project, open, onClose, onUpdated }: Props) {
    const [form, setForm] = useState({
      name: project.name,
      code: project.code || '',
      description: project.description || '',
      startDate: project.startDate.slice(0, 10),
      endDate: project.endDate.slice(0, 10),
      budget: project.budget?.toString() || '',
    });
    const [error, setError] = useState('');

    useEffect(() => {
      // 모달 오픈 시 현재 값으로 초기화
      setForm({
        name: project.name,
        code: project.code || '',
        description: project.description || '',
        startDate: project.startDate.slice(0, 10),
        endDate: project.endDate.slice(0, 10),
        budget: project.budget?.toString() || '',
      });
    }, [project, open]);

    if (!open) return null;

    const handleSubmit = async (e: React.FormEvent) => {
      e.preventDefault();
      setError('');
      try {
        const res = await api.patch<{ project: ProjectDetail }>(`/projects/${project.id}`, {
          name: form.name,
          code: form.code || undefined,
          description: form.description || undefined,
          startDate: form.startDate,
          endDate: form.endDate,
          budget: form.budget ? Number(form.budget) : undefined,
        });
        onUpdated(res.result.project);
        onClose();
      } catch (err: any) {
        setError(err?.message || '프로젝트 수정에 실패했습니다.');
      }
    };

    return (
      <div style={{ position: 'fixed', inset: 0, background: 'rgba(0,0,0,0.4)', display: 'flex', alignItems: 'center', justifyContent: 'center', zIndex: 1000 }}>
        <div style={{ background: '#fff', borderRadius: 8, padding: 24, width: 560, maxWidth: '90vw' }}>
          <h2 style={{ fontSize: 18, marginBottom: 16 }}>프로젝트 수정</h2>
          <form onSubmit={handleSubmit}>
            {/* ProjectCreatePage와 동일한 필드 구성, 기존 값 pre-fill */}
            {/* 저장 / 취소 버튼 */}
          </form>
        </div>
      </div>
    );
  }
  ```
