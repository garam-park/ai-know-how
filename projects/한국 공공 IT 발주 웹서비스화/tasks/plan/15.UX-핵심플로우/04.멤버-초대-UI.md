# 멤버 초대/역할변경/제거 UI

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~50분 |
| AI 에이전트 | ~20분 |

## What (무엇을)
`MembersTab`에 멤버 초대 폼, 역할 변경, 투입률 변경, 멤버 제거 기능을 추가합니다. 이메일로 기존 사용자를 검색하여 프로젝트에 초대하고, 소속 회사와 역할을 지정할 수 있습니다.

## Why (왜)
현재 멤버 탭은 읽기 전용 테이블만 보여주고, 멤버를 추가하거나 관리할 수 없습니다. PM이 팀을 구성하려면 멤버 초대가 필수이며, 투입률이나 역할 변경도 프로젝트 운영에 핵심적인 기능입니다.

## 선행 작업
- `15.UX-핵심플로우/03.컨소시엄사-추가삭제-UI.md` (멤버는 소속 회사가 등록되어야 초대 가능)

## 수행 가이드

### Step 1: 멤버 초대 폼 컴포넌트 생성
- 대상 파일: `apps/frontend/src/components/InviteMemberForm.tsx` (신규)
- 작업 내용:
  ```typescript
  import { useState, useEffect } from 'react';
  import { api } from '../services/api';

  interface ProjectCompany { id: number; company: { id: number; name: string }; }
  interface Role { id: number; name: string; }

  interface Props {
    projectId: string;
    onInvited: () => void;
  }

  export default function InviteMemberForm({ projectId, onInvited }: Props) {
    const [email, setEmail] = useState('');
    const [projectCompanyId, setProjectCompanyId] = useState('');
    const [roleId, setRoleId] = useState('');
    const [inputRate, setInputRate] = useState('100');
    const [companies, setCompanies] = useState<ProjectCompany[]>([]);
    const [roles, setRoles] = useState<Role[]>([]);
    const [error, setError] = useState('');

    useEffect(() => {
      // 컨소시엄사 목록 조회 (flat)
      api.get<{ companies: any[] }>(`/projects/${projectId}/companies`)
        .then((res) => {
          const flatten = (nodes: any[]): ProjectCompany[] =>
            nodes.flatMap((n) => [{ id: n.id, company: n.company }, ...flatten(n.children || [])]);
          setCompanies(flatten(res.result.companies));
        })
        .catch(() => {});

      // 역할 목록 조회 (시스템 역할)
      // 백엔드에 역할 목록 API가 없으면 하드코딩 또는 추가 구현 필요
    }, [projectId]);

    const handleSubmit = async (e: React.FormEvent) => {
      e.preventDefault();
      setError('');
      try {
        await api.post(`/projects/${projectId}/members`, {
          email,
          projectCompanyId: Number(projectCompanyId),
          roleId: Number(roleId),
          inputRate: Number(inputRate),
        });
        setEmail('');
        onInvited();
      } catch (err: any) {
        setError(err?.message || '멤버 초대에 실패했습니다.');
      }
    };

    return (
      <form onSubmit={handleSubmit}>
        {/* 이메일 입력, 소속회사 선택, 역할 선택, 투입률 입력, 초대 버튼 */}
      </form>
    );
  }
  ```

### Step 4: 멤버 제거 버튼 추가
- 대상 파일: `apps/frontend/src/pages/MembersTab.tsx`
- 작업 내용:
  - 테이블 마지막 컬럼에 "제거" 버튼 추가
  - DELETE `/projects/:projectId/members/:memberId` API 호출
  ```typescript
  const handleRemoveMember = async (memberId: number) => {
    if (!confirm('해당 멤버를 프로젝트에서 제거하시겠습니까?')) return;
    try {
      await api.delete(`/projects/${projectId}/members/${memberId}`);
      fetchMembers();
    } catch (err: any) {
      alert(err?.message || '멤버 제거에 실패했습니다.');
    }
  };
  ```
