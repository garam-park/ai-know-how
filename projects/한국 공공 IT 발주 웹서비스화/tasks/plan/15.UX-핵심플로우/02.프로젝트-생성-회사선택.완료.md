# 프로젝트 생성 시 회사 선택 + 자동 PM 등록

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~40분 |
| AI 에이전트 | ~15분 |

## What (무엇을)
`ProjectCreatePage`에 소속 회사 선택 드롭다운을 추가합니다. 프로젝트 생성 시 선택된 회사가 자동으로 PRIME(주관사)로 등록되고, 생성자가 해당 프로젝트의 PM(프로젝트 매니저)으로 자동 등록됩니다.

## Why (왜)
현재 프로젝트 생성 시 회사 선택 UI가 없어서, 생성 후에도 컨소시엄에 회사가 없고 멤버도 없는 상태가 됩니다. 생성자 본인조차 프로젝트에 연결되지 않아 권한 시스템이 동작하지 않습니다. 프로젝트 생성 → 주관사 등록 → PM 자동 배정은 핵심 비즈니스 플로우입니다.

## 선행 작업
- `15.UX-핵심플로우/01.회사-등록-폼.md` (회사가 있어야 선택 가능)

## 수행 가이드

### Step 1: 회사 목록 조회 API 호출 추가
- 대상 파일: `apps/frontend/src/pages/ProjectCreatePage.tsx`
- 작업 내용:
  ```typescript
  import { useState, useEffect } from 'react';

  interface Company {
    id: number;
    name: string;
  }

  // 컴포넌트 내부
  const [companies, setCompanies] = useState<Company[]>([]);

  useEffect(() => {
    api.get<{ companies: Company[] }>('/companies')
      .then((res) => setCompanies(res.result.companies))
      .catch(() => {});
  }, []);
  ```

### Step 2: 회사 선택 드롭다운 UI 추가
- 대상 파일: `apps/frontend/src/pages/ProjectCreatePage.tsx`
- 작업 내용:
  - form 상태에 `companyId` 필드 추가
  - 사업명 필드 아래에 회사 선택 `<select>` 추가 (필수)
  ```typescript
  const [form, setForm] = useState({
    name: '',
    code: '',
    description: '',
    startDate: '',
    endDate: '',
    budget: '',
    companyId: '',  // 추가
  });

  // JSX
  <div style={{ marginBottom: 16 }}>
    <label style={{ display: 'block', marginBottom: 4, fontWeight: 600 }}>소속 회사 *</label>
    <select value={form.companyId} onChange={(e) => setForm((prev) => ({ ...prev, companyId: e.target.value }))}
      required style={inputStyle}>
      <option value="">회사를 선택하세요</option>
      {companies.map((c) => (
        <option key={c.id} value={c.id}>{c.name}</option>
      ))}
    </select>
  </div>
  ```

### Step 3: API 요청에 companyId 포함
- 대상 파일: `apps/frontend/src/pages/ProjectCreatePage.tsx`
- 작업 내용:
  - handleSubmit에서 companyId를 Number로 변환하여 전송
  ```typescript
  const res = await api.post<{ project: { id: number } }>('/projects', {
    name: form.name,
    code: form.code || undefined,
    description: form.description || undefined,
    startDate: form.startDate,
    endDate: form.endDate,
    budget: form.budget ? Number(form.budget) : undefined,
    companyId: Number(form.companyId),  // 추가
  });
  ```
  - 백엔드 `project.service.ts`의 `create` 메서드는 이미 `companyId`가 있으면 PRIME 회사 + PM 멤버를 자동 생성하도록 구현되어 있음

### Step 4: 회사 없을 때 안내 메시지
- 대상 파일: `apps/frontend/src/pages/ProjectCreatePage.tsx`
- 작업 내용:
  - 회사가 0개일 때 "먼저 회사를 등록해주세요" 안내와 회사 목록 페이지 링크 표시
  ```typescript
  {companies.length === 0 && (
    <p style={{ fontSize: 13, color: '#e53e3e', marginTop: 4 }}>
      등록된 회사가 없습니다. <Link to="/companies" style={{ color: '#2b6cb0' }}>회사 등록</Link> 후 프로젝트를 생성하세요.
    </p>
  )}
  ```
