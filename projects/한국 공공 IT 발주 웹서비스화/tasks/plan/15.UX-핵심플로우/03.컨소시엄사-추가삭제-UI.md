# 컨소시엄사 추가/삭제 UI

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~50분 |
| AI 에이전트 | ~20분 |

## What (무엇을)
`CompaniesTab`에 컨소시엄사 추가 폼과 삭제 버튼을 추가합니다. 회사 선택(드롭다운) + 역할(PARTNER/SUB) 선택 + 상위사 선택으로 컨소시엄 트리를 구성할 수 있도록 합니다.

## Why (왜)
현재 컨소시엄 탭은 읽기 전용으로, 프로젝트에 참여사나 하청사를 추가할 방법이 없습니다. 컨소시엄 구성은 프로젝트 운영의 핵심이며, 멤버 초대의 전제 조건입니다 (멤버는 반드시 소속 회사가 있어야 함).

## 선행 작업
- `15.UX-핵심플로우/02.프로젝트-생성-회사선택.md` (PRIME 회사가 등록되어야 하위 추가 가능)

## 수행 가이드

### Step 1: 컨소시엄사 추가 폼 컴포넌트 생성
- 대상 파일: `apps/frontend/src/components/AddConsortiumForm.tsx` (신규)
- 작업 내용:
  ```typescript
  import { useState, useEffect } from 'react';
  import { api } from '../services/api';

  interface Company { id: number; name: string; }
  interface ExistingCompanyNode { id: number; company: { id: number; name: string }; role: string; }

  interface Props {
    projectId: string;
    existingCompanies: ExistingCompanyNode[];
    onAdded: () => void;
  }

  export default function AddConsortiumForm({ projectId, existingCompanies, onAdded }: Props) {
    const [allCompanies, setAllCompanies] = useState<Company[]>([]);
    const [companyId, setCompanyId] = useState('');
    const [role, setRole] = useState<'PARTNER' | 'SUB'>('PARTNER');
    const [parentId, setParentId] = useState('');
    const [error, setError] = useState('');

    useEffect(() => {
      api.get<{ companies: Company[] }>('/companies')
        .then((res) => setAllCompanies(res.result.companies))
        .catch(() => {});
    }, []);

    // 이미 등록된 회사 제외
    const availableCompanies = allCompanies.filter(
      (c) => !existingCompanies.some((ec) => ec.company.id === c.id)
    );

    const handleSubmit = async (e: React.FormEvent) => {
      e.preventDefault();
      setError('');
      try {
        await api.post(`/projects/${projectId}/companies`, {
          companyId: Number(companyId),
          role,
          parentId: parentId ? Number(parentId) : undefined,
        });
        setCompanyId('');
        setParentId('');
        onAdded();
      } catch (err: any) {
        setError(err?.message || '컨소시엄사 추가에 실패했습니다.');
      }
    };

    return (
      <form onSubmit={handleSubmit} style={{ display: 'flex', gap: 8, alignItems: 'flex-end', flexWrap: 'wrap', marginBottom: 16 }}>
        {/* 회사 선택 드롭다운, 역할 선택, 상위사 선택, 추가 버튼 */}
      </form>
    );
  }
  ```

### Step 2: CompaniesTab에 추가 폼 연동
- 대상 파일: `apps/frontend/src/pages/CompaniesTab.tsx`
- 작업 내용:
  - `AddConsortiumForm` import
  - 컨소시엄 트리 위에 추가 폼 배치
  - 추가 완료 시 목록 재조회
  ```typescript
  import AddConsortiumForm from '../components/AddConsortiumForm';

  // flattenNodes: 트리를 1차원 배열로 변환 (상위사 선택 드롭다운용)
  const flattenNodes = (nodes: CompanyNode[]): CompanyNode[] => {
    return nodes.flatMap((n) => [n, ...flattenNodes(n.children)]);
  };

  // fetchCompanies 함수 분리
  const fetchCompanies = () => {
    api.get<{ companies: CompanyNode[] }>(`/projects/${projectId}/companies`)
      .then((res) => setCompanies(res.result.companies))
      .catch(() => {});
  };
  ```

### Step 3: 삭제 버튼 추가
- 대상 파일: `apps/frontend/src/pages/CompaniesTab.tsx`
- 작업 내용:
  - `CompanyTree` 컴포넌트에 삭제 버튼 추가 (PRIME 제외)
  - DELETE `/projects/:projectId/companies/:companyId` API 호출
  ```typescript
  // CompanyTree props에 onDelete 추가
  function CompanyTree({ node, depth = 0, onDelete }: { node: CompanyNode; depth?: number; onDelete: (id: number) => void }) {
    // ...
    {node.role !== 'PRIME' && (
      <button onClick={() => onDelete(node.id)}
        style={{ marginLeft: 'auto', background: 'none', border: 'none', color: '#e53e3e', cursor: 'pointer', fontSize: 13 }}>
        삭제
      </button>
    )}
  }
  ```

### Step 4: 삭제 확인 및 실행
- 대상 파일: `apps/frontend/src/pages/CompaniesTab.tsx`
- 작업 내용:
  ```typescript
  const handleDelete = async (projectCompanyId: number) => {
    if (!confirm('해당 컨소시엄사를 삭제하시겠습니까? 하위 회사와 소속 멤버도 함께 제거됩니다.')) return;
    try {
      await api.delete(`/projects/${projectId}/companies/${projectCompanyId}`);
      fetchCompanies();
    } catch (err: any) {
      alert(err?.message || '삭제에 실패했습니다.');
    }
  };
  ```
