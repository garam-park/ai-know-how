# 회사 등록 및 프로젝트 생성 수동 검증 가이드

## 1. 회사 등록 확인 (`CompanyListPage.tsx`)
1. **경로**: `/companies` 접속 (회사 목록 페이지)
2. **"회사 등록" 팝업 동작 점검**:
   - 상단의 `+ 회사 등록` 버튼 클릭
   - 팝업이 노출되는지 확인 (화면 가운데 모달 레이어)
3. **유효성 검사 (Validation)**:
   - 회사명(필수) 입력 없이 `등록` 클릭 리 HTML `required` 툴팁 표시 여부 확인
4. **정상 동작 확인**:
   - `회사명`, `사업자번호` 등 입력 후 `등록` 클릭
   - 팝업 자동 닫힘 및 오른쪽 위 "회사 등록이 완료되었습니다." Toast 성공 알람 확인
   - 회사 목록 테이블에 방금 등록한 회사가 실시간 노출되는지 확인

## 2. 프로젝트 생성 동작 확인 (`ProjectCreatePage.tsx`)
1. **경로**: `/projects/new` (새 프로젝트 생성 페이지)
2. **안내 문구 확인**:
   - 회사를 한 곳도 등록하지 않은 초기 상태라면, 소속 회사(주관사) 드롭다운 아래에 `⚠️ 등록된 회사가 없습니다. 회사 목록 관리에서 회사를 먼저 등록해 주세요.` 문구 및 빨간색 글씨 표출 여부 확인
3. **정상 제출 확인**:
   - **소속 회사(주관사)** 드롭다운에 앞서 등록한 회사가 정상 노출되는지 확인
   - 필수값(사업명, 소속 회사, 시작일, 종료일) 입력. (시작일이 종료일보다 늦으면 등록 버튼 비활성화 되는지도 확인)
   - 달력(Date) 요소 동작 확인
   - `생성` 클릭 시 방금 생성한 프로젝트 상세 페이지(`/projects/:id`)로 이동하는지 확인 및 "프로젝트가 성공적으로 생성되었습니다" Toast 성공 알람 노출 여부 점검
