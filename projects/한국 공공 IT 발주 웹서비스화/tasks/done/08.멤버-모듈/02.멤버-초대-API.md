# 멤버 초대 API

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~30분 |
| AI 에이전트 | ~10분 |

## What (무엇을)
POST `/api/v1/projects/:projectId/members` 엔드포인트를 구현합니다. 기존 사용자를 프로젝트에 초대하여 역할과 소속사를 배정합니다. 동일 프로젝트에 중복 초대되지 않도록 검사합니다.

## Why (왜)
프로젝트에 인력을 배정하기 위한 핵심 기능입니다. 이메일로 사용자를 검색하고, 특정 회사(ProjectCompany)와 역할(Role)을 지정하여 멤버로 추가합니다.

### Step 1: InviteMemberDto 검증 규칙 추가
- 대상 파일: `apps/backend/src/modules/project-member/dto/project-member.dto.ts`
- 작업 내용:
  ```typescript
  import { IsEmail, IsUUID, IsOptional, IsNumber, Min, Max } from 'class-validator';

  export class InviteMemberDto {
    @IsEmail()
    email: string;

    @IsUUID()
    projectCompanyId: string;

    @IsUUID()
    roleId: string;

    @IsOptional()
    @IsNumber()
    @Min(0)
    @Max(100)
    inputRate?: number;
  }
  ```

### Step 2: ProjectMemberService에 invite 메서드 구현
- 대상 파일: `apps/backend/src/modules/project-member/project-member.service.ts`
- 작업 내용:
  ```typescript
  async invite(
    projectId: string,
    dto: InviteMemberDto,
  ): Promise<ProjectMemberDetailDto> {
    // 사용자 조회 (이메일 기준)
    const user = await this.prisma.user.findFirst({
      where: { email: dto.email, deletedAt: null },
    });

    if (!user) {
      throw new NotFoundException({
        code: 404000,
        message: '해당 이메일의 사용자를 찾을 수 없습니다.',
      });
    }

    // 프로젝트 존재 확인
    const project = await this.prisma.project.findFirst({
      where: { id: projectId, deletedAt: null },
    });

    if (!project) {
      throw new NotFoundException({
        code: 404000,
        message: '프로젝트를 찾을 수 없습니다.',
      });
    }

    // @@unique([projectId, userId]) 중복 확인
    const existing = await this.prisma.projectMember.findFirst({
      where: { projectId, userId: user.id, deletedAt: null },
    });

    if (existing) {
      throw new ConflictException({
        code: 409001,
        message: '이미 프로젝트에 참여 중인 사용자입니다.',
      });
    }

    // ProjectCompany 유효성 확인
    const projectCompany = await this.prisma.projectCompany.findFirst({
      where: { id: dto.projectCompanyId, projectId, deletedAt: null },
      include: { company: true },
    });

    if (!projectCompany) {
      throw new BadRequestException({
        code: 400000,
        message: '해당 프로젝트에 속한 회사가 아닙니다.',
      });
    }

    // Role 존재 확인
    const role = await this.prisma.role.findFirst({
      where: { id: dto.roleId, deletedAt: null },
      include: { permissions: true },
    });

    if (!role) {
      throw new NotFoundException({
        code: 404000,
        message: '역할을 찾을 수 없습니다.',
      });
    }

    // ProjectMember 생성
    const member = await this.prisma.projectMember.create({
      data: {
        projectId,
        userId: user.id,
        projectCompanyId: dto.projectCompanyId,
        roleId: dto.roleId,
        inputRate: dto.inputRate || 100,
      },
      include: {
        user: true,
        projectCompany: { include: { company: true } },
        role: { include: { permissions: true } },
      },
    });

    return {
      id: member.id,
      projectId: member.projectId,
      userId: member.userId,
      projectCompanyId: member.projectCompanyId,
      roleId: member.roleId,
      inputRate: member.inputRate,
      createdAt: member.createdAt,
      updatedAt: member.updatedAt,
      user: {
        id: member.user.id,
        email: member.user.email,
        name: member.user.name,
      },
      company: {
        id: member.projectCompany.company.id,
        name: member.projectCompany.company.name,
      },
      role: {
        id: member.role.id,
        name: member.role.name,
        code: member.role.code,
      },
    };
  }
  ```

### Step 3: ProjectMemberController에 POST 엔드포인트 추가
- 대상 파일: `apps/backend/src/modules/project-member/project-member.controller.ts`
- 작업 내용:
  ```typescript
  import { Post, Body, Param, UseGuards } from '@nestjs/common';
  import { JwtAuthGuard } from '@/auth/guards/jwt-auth.guard';
  import { PermissionGuard } from '@/auth/guards/permission.guard';
  import { RequirePermission } from '@/common/decorators/require-permission.decorator';
  import { InviteMemberDto } from './dto/project-member.dto';

  @Post()
  @UseGuards(JwtAuthGuard, PermissionGuard)
  @RequirePermission('MEMBER', 'INVITE')
  async invite(
    @Param('projectId') projectId: string,
    @Body() dto: InviteMemberDto,
  ): Promise<{
    code: number;
    result: { member: ProjectMemberDetailDto };
  }> {
    const member = await this.projectMemberService.invite(projectId, dto);
    return {
      code: 201000,
      result: { member },
    };
  }
  ```
