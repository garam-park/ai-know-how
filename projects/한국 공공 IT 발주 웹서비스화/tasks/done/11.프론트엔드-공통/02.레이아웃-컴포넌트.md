# 레이아웃 컴포넌트

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~30분 |
| AI 에이전트 | ~10분 |

## What (무엇을)
로그인 후 페이지에서 사용할 AppLayout(Header, Sidebar, MainContent)과 로그인/회원가입 페이지에서 사용할 AuthLayout을 구현합니다.

## Why (왜)
모든 페이지에서 일관된 UI 구조를 제공하고, 권한에 따라 메뉴를 동적으로 표시/숨김으로써 사용자 경험을 향상시킵니다.

## 수행 가이드

### Step 1: AppLayout 컴포넌트 생성
`apps/frontend/src/layouts/AppLayout.tsx` 파일을 생성합니다:

```typescript
import React, { useState } from 'react';
import { Outlet, useParams } from 'react-router-dom';
import { useAuth } from '../hooks/useAuth';
import Header from '../components/Header';
import Sidebar from '../components/Sidebar';
import './AppLayout.css';

export const AppLayout: React.FC = () => {
  const [sidebarOpen, setSidebarOpen] = useState(true);
  const { user, logout } = useAuth();
  const { id: projectId } = useParams();

  const handleLogout = async () => {
    await logout();
  };

  return (
    <div className="app-layout">
      <Header
        projectId={projectId}
        userName={user?.name || ''}
        onLogout={handleLogout}
        onToggleSidebar={() => setSidebarOpen(!sidebarOpen)}
      />
      <div className="app-layout__container">
        <Sidebar isOpen={sidebarOpen} onClose={() => setSidebarOpen(false)} />
        <main className="app-layout__main">
          <Outlet />
        </main>
      </div>
    </div>
  );
};

export default AppLayout;
```

### Step 3: Sidebar 컴포넌트 생성
`apps/frontend/src/components/Sidebar.tsx` 파일을 생성합니다:

```typescript
import React from 'react';
import { useNavigate, useParams } from 'react-router-dom';
import { useAuth } from '../hooks/useAuth';
import './Sidebar.css';

interface SidebarProps {
  isOpen: boolean;
  onClose: () => void;
}

const Sidebar: React.FC<SidebarProps> = ({ isOpen, onClose }) => {
  const navigate = useNavigate();
  const { id: projectId } = useParams();
  const { user } = useAuth();

  const menuItems = [
    { label: '프로젝트 목록', path: '/projects' },
  ];

  // projectId가 있으면 프로젝트별 메뉴 추가
  if (projectId) {
    menuItems.push(
      { label: '대시보드', path: `/projects/${projectId}/dashboard` },
      { label: 'WBS', path: `/projects/${projectId}/wbs` },
      { label: '멤버 관리', path: `/projects/${projectId}/members` },
    );

    // 관리자 권한이면 컨소시엄 탭 표시
    if (user?.role === 'ADMIN' || user?.role === 'PM') {
      menuItems.push({
        label: '컨소시엄',
        path: `/projects/${projectId}/consortium`,
      });
    }
  }

  const handleMenuClick = (path: string) => {
    navigate(path);
    onClose();
  };

  return (
    <aside className={`sidebar ${isOpen ? 'sidebar--open' : ''}`}>
      <nav className="sidebar__nav">
        <ul className="sidebar__menu">
          {menuItems.map((item, index) => (
            <li key={index} className="sidebar__menu-item">
              <button
                className="sidebar__menu-button"
                onClick={() => handleMenuClick(item.path)}
              >
                {item.label}
              </button>
            </li>
          ))}
        </ul>
      </nav>
    </aside>
  );
};

export default Sidebar;
```
