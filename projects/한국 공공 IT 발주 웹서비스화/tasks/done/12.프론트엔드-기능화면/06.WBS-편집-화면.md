# WBS 편집 화면

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~60분 |
| AI 에이전트 | ~25분 |

## What (무엇을)
/projects/:id/wbs에서 프로젝트의 WBS를 트리 테이블 형태로 표시합니다. 엑셀처럼 행 선택, 인라인 편집, 진척률 슬라이더, 노드 추가/삭제, 드래그&드롭 순서 변경을 지원합니다.

## Why (왜)
WBS는 이 서비스의 핵심 기능입니다. 사용자가 직관적으로 작업 계획과 진척을 관리할 수 있어야 합니다.

### Step 1: WBS API 서비스 생성
`apps/frontend/src/services/wbsService.ts` 파일을 생성합니다:

```typescript
import apiClient from './api';

export interface WBSNode {
  id: string;
  parentId?: string;
  code: string;
  title: string;
  type: 'CATEGORY' | 'TASK';
  depth: number;
  sequence: number;
  responsible?: string;
  responsibleCompany?: string;
  startDate?: string;
  endDate?: string;
  weight?: number;
  progress: number;
  children?: WBSNode[];
}

export interface UpdateProgressRequest {
  progress: number;
}

export interface UpdateProgressResponse {
  node: WBSNode;
  affectedNodes: WBSNode[];
}

export const wbsService = {
  getWBSTree: async (projectId: string): Promise<WBSNode[]> => {
    const response = await apiClient.get(`/projects/${projectId}/wbs/tree`);
    return response.data;
  },

  createNode: async (
    projectId: string,
    data: {
      parentId?: string;
      code: string;
      title: string;
      type: 'CATEGORY' | 'TASK';
      startDate?: string;
      endDate?: string;
      responsible?: string;
      weight?: number;
    }
  ): Promise<WBSNode> => {
    const response = await apiClient.post(`/projects/${projectId}/wbs`, data);
    return response.data;
  },

  updateNode: async (
    projectId: string,
    nodeId: string,
    data: Partial<WBSNode>
  ): Promise<WBSNode> => {
    const response = await apiClient.patch(
      `/projects/${projectId}/wbs/${nodeId}`,
      data
    );
    return response.data;
  },

  updateProgress: async (
    projectId: string,
    nodeId: string,
    progress: number
  ): Promise<UpdateProgressResponse> => {
    const response = await apiClient.patch(
      `/projects/${projectId}/wbs/${nodeId}/progress`,
      { progress }
    );
    return response.data;
  },

  reorderNode: async (
    projectId: string,
    nodeId: string,
    parentId: string | null,
    sequence: number
  ): Promise<void> => {
    await apiClient.patch(`/projects/${projectId}/wbs/${nodeId}/move`, {
      parentId,
      sequence,
    });
  },

  deleteNode: async (projectId: string, nodeId: string): Promise<void> => {
    await apiClient.delete(`/projects/${projectId}/wbs/${nodeId}`);
  },
};
```

### Step 2: WBSPage 컴포넌트 생성

```typescript
import React, { useEffect, useState } from 'react';
import { useProjectContext } from '../../hooks/useProjectContext';
import { useAuth } from '../../hooks/useAuth';
import { wbsService, WBSNode } from '../../services/wbsService';
import WBSTable from '../../components/WBSTable';
import Button from '../../components/Button';
import Modal from '../../components/Modal';
import Input from '../../components/Input';
import Loading from '../../components/Loading';
import Toast from '../../components/Toast';
import './WBSPage.css';

const WBSPage: React.FC = () => {
  const { project } = useProjectContext();
  const { user } = useAuth();

  const [wbsData, setWBSData] = useState<WBSNode[]>([]);
  const [loading, setLoading] = useState(true);
  const [toast, setToast] = useState<{
    message: string;
    type: 'success' | 'error';
  } | null>(null);

  const [isAddModalOpen, setIsAddModalOpen] = useState(false);
  const [addFormData, setAddFormData] = useState({
    parentId: '',
    code: '',
    title: '',
    type: 'TASK' as 'CATEGORY' | 'TASK',
    startDate: '',
    endDate: '',
  });

  const canEdit = user?.role === 'ADMIN' || user?.role === 'PM';

  useEffect(() => {
    fetchWBS();
  }, [project.id]);

  const fetchWBS = async () => {
    try {
      setLoading(true);
      const data = await wbsService.getWBSTree(project.id);
      setWBSData(data);
    } catch (err) {
      setToast({
        message: err instanceof Error ? err.message : 'WBS를 불러올 수 없습니다',
        type: 'error',
      });
    } finally {
      setLoading(false);
    }
  };

  const handleProgressChange = async (nodeId: string, progress: number) => {
    try {
      const response = await wbsService.updateProgress(
        project.id,
        nodeId,
        progress
      );

      // 영향받은 노드들 업데이트
      setWBSData((prev) =>
        updateWBSTree(prev, response.node.id, response.node)
      );
    } catch (err) {
      setToast({
        message: err instanceof Error ? err.message : '진척률 업데이트 실패',
        type: 'error',
      });
    }
  };

  const handleAddNode = async (e: React.FormEvent<HTMLFormElement>) => {
    e.preventDefault();

    if (!addFormData.code.trim() || !addFormData.title.trim()) {
      setToast({ message: '필수 필드를 입력하세요', type: 'error' });
      return;
    }

    try {
      const newNode = await wbsService.createNode(project.id, {
        parentId: addFormData.parentId || undefined,
        code: addFormData.code,
        title: addFormData.title,
        type: addFormData.type,
        startDate: addFormData.startDate || undefined,
        endDate: addFormData.endDate || undefined,
      });

      // 트리에 새 노드 추가
      setWBSData((prev) => addNodeToTree(prev, newNode));
      setToast({ message: '작업이 추가되었습니다', type: 'success' });
      setAddFormData({
        parentId: '',
        code: '',
        title: '',
        type: 'TASK',
        startDate: '',
        endDate: '',
      });
      setIsAddModalOpen(false);
    } catch (err) {
      setToast({
        message: err instanceof Error ? err.message : '작업 추가 실패',
        type: 'error',
      });
    }
  };

  const handleDeleteNode = async (nodeId: string) => {
    if (!window.confirm('이 작업을 삭제하시겠습니까?')) {
      return;
    }

    try {
      await wbsService.deleteNode(project.id, nodeId);
      setWBSData((prev) => removeNodeFromTree(prev, nodeId));
      setToast({ message: '작업이 삭제되었습니다', type: 'success' });
    } catch (err) {
      setToast({
        message: err instanceof Error ? err.message : '작업 삭제 실패',
        type: 'error',
      });
    }
  };

  if (loading) {
    return <Loading message="WBS 로딩 중..." />;
  }

  return (
    <div className="wbs-page">
      <div className="wbs-page__header">
        <h2>WBS 계획</h2>
        {canEdit && (
          <Button
            variant="primary"
            onClick={() => setIsAddModalOpen(true)}
          >
            + 작업 추가
          </Button>
        )}
      </div>

      {wbsData.length === 0 ? (
        <div className="wbs-page__empty">
          <p>아직 등록된 작업이 없습니다</p>
        </div>
      ) : (
        <WBSTable
          nodes={wbsData}
          onProgressChange={handleProgressChange}
          onDeleteNode={handleDeleteNode}
          canEdit={canEdit}
        />
      )}

      <Modal
        isOpen={isAddModalOpen}
        title="작업 추가"
        onClose={() => setIsAddModalOpen(false)}
        size="medium"
      >
        <form onSubmit={handleAddNode} className="wbs-page__form">
          <div className="wbs-page__field">
            <label className="wbs-page__label">작업 유형</label>
            <select
              value={addFormData.type}
              onChange={(e) =>
                setAddFormData((prev) => ({
                  ...prev,
                  type: e.target.value as 'CATEGORY' | 'TASK',
                }))
              }
              className="wbs-page__select"
            >
              <option value="TASK">작업</option>
              <option value="CATEGORY">분류</option>
            </select>
          </div>

          <div className="wbs-page__field">
            <Input
              label="코드 *"
              type="text"
              value={addFormData.code}
              onChange={(e) =>
                setAddFormData((prev) => ({
                  ...prev,
                  code: e.target.value,
                }))
              }
              placeholder="WBS-001"
            />
          </div>

          <div className="wbs-page__field">
            <Input
              label="제목 *"
              type="text"
              value={addFormData.title}
              onChange={(e) =>
                setAddFormData((prev) => ({
                  ...prev,
                  title: e.target.value,
                }))
              }
              placeholder="작업 제목"
            />
          </div>

          <div className="wbs-page__field">
            <Input
              label="시작일"
              type="date"
              value={addFormData.startDate}
              onChange={(e) =>
                setAddFormData((prev) => ({
                  ...prev,
                  startDate: e.target.value,
                }))
              }
            />
          </div>

          <div className="wbs-page__field">
            <Input
              label="종료일"
              type="date"
              value={addFormData.endDate}
              onChange={(e) =>
                setAddFormData((prev) => ({
                  ...prev,
                  endDate: e.target.value,
                }))
              }
            />
          </div>

          <div className="wbs-page__footer">
            <Button
              variant="secondary"
              onClick={() => setIsAddModalOpen(false)}
            >
              취소
            </Button>
            <Button variant="primary" type="submit">
              추가
            </Button>
          </div>
        </form>
      </Modal>

      {toast && (
        <Toast
          message={toast.message}
          type={toast.type}
          onClose={() => setToast(null)}
        />
      )}
    </div>
  );
};

// 헬퍼 함수들
function updateWBSTree(nodes: WBSNode[], nodeId: string, updatedNode: WBSNode): WBSNode[] {
  return nodes.map((node) => {
    if (node.id === nodeId) {
      return updatedNode;
    }
    if (node.children) {
      return {
        ...node,
        children: updateWBSTree(node.children, nodeId, updatedNode),
      };
    }
    return node;
  });
}

function addNodeToTree(nodes: WBSNode[], newNode: WBSNode): WBSNode[] {
  if (!newNode.parentId) {
    return [...nodes, newNode];
  }

  return nodes.map((node) => {
    if (node.id === newNode.parentId) {
      return {
        ...node,
        children: [...(node.children || []), newNode],
      };
    }
    if (node.children) {
      return {
        ...node,
        children: addNodeToTree(node.children, newNode),
      };
    }
    return node;
  });
}

function removeNodeFromTree(nodes: WBSNode[], nodeId: string): WBSNode[] {
  return nodes
    .filter((node) => node.id !== nodeId)
    .map((node) => ({
      ...node,
      children: node.children ? removeNodeFromTree(node.children, nodeId) : undefined,
    }));
}

export default WBSPage;
```

### Step 3: WBSTable 컴포넌트 생성
`apps/frontend/src/components/WBSTable.tsx` 파일을 생성합니다:

```typescript
import React, { useState } from 'react';
import { WBSNode } from '../services/wbsService';
import WBSRow from './WBSRow';
import './WBSTable.css';

interface WBSTableProps {
  nodes: WBSNode[];
  onProgressChange: (nodeId: string, progress: number) => void;
  onDeleteNode: (nodeId: string) => void;
  canEdit: boolean;
}

const WBSTable: React.FC<WBSTableProps> = ({
  nodes,
  onProgressChange,
  onDeleteNode,
  canEdit,
}) => {
  const [expandedNodes, setExpandedNodes] = useState<Set<string>>(
    new Set(nodes.map((n) => n.id))
  );

  const toggleExpand = (nodeId: string) => {
    const newExpanded = new Set(expandedNodes);
    if (newExpanded.has(nodeId)) {
      newExpanded.delete(nodeId);
    } else {
      newExpanded.add(nodeId);
    }
    setExpandedNodes(newExpanded);
  };

  return (
    <div className="wbs-table">
      <table>
        <thead>
          <tr>
            <th className="wbs-table__code">코드</th>
            <th className="wbs-table__title">제목</th>
            <th className="wbs-table__responsible">담당회사</th>
            <th className="wbs-table__dates">기간</th>
            <th className="wbs-table__progress">진척율</th>
            {canEdit && <th className="wbs-table__actions">작업</th>}
          </tr>
        </thead>
        <tbody>
          {nodes.map((node) => (
            <WBSRow
              key={node.id}
              node={node}
              depth={0}
              expanded={expandedNodes.has(node.id)}
              onToggleExpand={toggleExpand}
              onProgressChange={onProgressChange}
              onDelete={canEdit ? onDeleteNode : undefined}
            />
          ))}
        </tbody>
      </table>
    </div>
  );
};

export default WBSTable;
```

```typescript
import React from 'react';
import { WBSNode } from '../services/wbsService';
import './WBSRow.css';

interface WBSRowProps {
  node: WBSNode;
  depth: number;
  expanded: boolean;
  onToggleExpand: (nodeId: string) => void;
  onProgressChange: (nodeId: string, progress: number) => void;
  onDelete?: (nodeId: string) => void;
}

const WBSRow: React.FC<WBSRowProps> = ({
  node,
  depth,
  expanded,
  onToggleExpand,
  onProgressChange,
  onDelete,
}) => {
  const hasChildren = node.children && node.children.length > 0;
  const isCategory = node.type === 'CATEGORY';

  return (
    <>
      <tr className={`wbs-row ${isCategory ? 'wbs-row--category' : ''}`}>
        <td className="wbs-row__code">
          {hasChildren && (
            <button
              className={`wbs-row__expand ${
                expanded ? 'wbs-row__expand--open' : ''
              }`}
              onClick={() => onToggleExpand(node.id)}
            >
              ▶
            </button>
          )}
          {!hasChildren && <span className="wbs-row__expand-spacer" />}
          <span className="wbs-row__icon">{isCategory ? '📁' : '📝'}</span>
          <span>{node.code}</span>
        </td>
        <td className="wbs-row__title" style={{ paddingLeft: `${depth * 20}px` }}>
          {node.title}
        </td>
        <td className="wbs-row__responsible">
          {node.responsibleCompany}
        </td>
        <td className="wbs-row__dates">
          {node.startDate && node.endDate && (
            <>
              {new Date(node.startDate).toLocaleDateString('ko-KR', {
                month: 'short',
                day: 'numeric',
              })}
              {' ~ '}
              {new Date(node.endDate).toLocaleDateString('ko-KR', {
                month: 'short',
                day: 'numeric',
              })}
            </>
          )}
        </td>
        <td className="wbs-row__progress">
          <input
            type="range"
            min="0"
            max="100"
            value={node.progress}
            onChange={(e) =>
              onProgressChange(node.id, parseInt(e.target.value, 10))
            }
            className="wbs-row__slider"
          />
          <span className="wbs-row__progress-text">{node.progress}%</span>
        </td>
        {onDelete && (
          <td className="wbs-row__actions">
            <button
              className="wbs-row__delete"
              onClick={() => onDelete(node.id)}
            >
              삭제
            </button>
          </td>
        )}
      </tr>

      {hasChildren && expanded && (
        <>
          {node.children!.map((child) => (
            <WBSRow
              key={child.id}
              node={child}
              depth={depth + 1}
              expanded={expanded}
              onToggleExpand={onToggleExpand}
              onProgressChange={onProgressChange}
              onDelete={onDelete}
            />
          ))}
        </>
      )}
    </>
  );
};

export default WBSRow;
```

## 완료 기준
- [ ] WBS 데이터가 트리 테이블 형태로 표시
- [ ] CATEGORY는 폴더 아이콘, TASK는 문서 아이콘
- [ ] 확장/축소 버튼으로 하위 항목 표시/숨김
- [ ] 진척률 슬라이더로 조정 시 실시간 API 호출
- [ ] 진척률 변경 후 상위 CATEGORY도 자동 업데이트
- [ ] PM/ADMIN만 작업 추가/삭제 버튼 표시
- [ ] 작업 추가 모달에서 유형/코드/제목/기간 입력
- [ ] 깊이별 들여쓰기로 계층 구조 시각화

## 참고
- 엑셀처럼 행 단위로 보기 쉽게 구성
- 진척률 변경 시 부모 노드 진척률도 자동 계산
- 드래그&드롭은 향후 기능으로 추가 가능
- Scope 기반 편집 제한은 추후 권한 시스템과 연계
