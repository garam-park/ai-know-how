# WBS 생성 API

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~30분 |
| AI 에이전트 | ~10분 |

## What (무엇을)
POST /api/v1/projects/:id/wbs-nodes 엔드포인트를 구현하여 WBS 노드(CATEGORY 또는 TASK)를 생성합니다.

## Why (왜)
WBS 노드 생성은 프로젝트의 계층적 작업 구조를 구성하는 첫 단계입니다. CATEGORY(카테고리)와 TASK(작업)의 두 가지 유형을 지원하며, 자동으로 깊이(depth)와 순서(order)를 계산합니다.

### Step 1: CreateWbsNodeDto 정의
`apps/backend/src/modules/wbs/dto/create-wbs-node.dto.ts`:
```typescript
import { IsEnum, IsString, IsOptional, IsNumber, IsDateString } from 'class-validator';
import { Type } from 'class-transformer';

enum WbsType {
  CATEGORY = 'CATEGORY',
  TASK = 'TASK',
}

export class CreateWbsNodeDto {
  @IsOptional()
  @IsNumber()
  @Type(() => Number)
  parentId?: number;

  @IsEnum(WbsType)
  type: WbsType;

  @IsString()
  title: string;

  @IsOptional()
  @IsString()
  description?: string;

  @IsOptional()
  @IsDateString()
  startDate?: string;

  @IsOptional()
  @IsDateString()
  endDate?: string;

  @IsOptional()
  @IsString()
  code?: string;

  @IsOptional()
  @IsNumber()
  @Type(() => Number)
  weight?: number;

  @IsOptional()
  @IsNumber()
  @Type(() => Number)
  assigneeId?: number;

  @IsOptional()
  @IsNumber()
  @Type(() => Number)
  projectCompanyId?: number;
}
```

### Step 2: WbsService에 노드 생성 로직 추가
`apps/backend/src/modules/wbs/wbs.service.ts`:
```typescript
import { Injectable, BadRequestException } from '@nestjs/common';
import { PrismaService } from 'src/prisma/prisma.service';
import { CreateWbsNodeDto } from './dto/create-wbs-node.dto';

@Injectable()
export class WbsService {
  constructor(
    private prisma: PrismaService,
  ) {}

  async createWbsNode(projectId: number, createDto: CreateWbsNodeDto) {
    // 1. parentId가 있으면 부모 노드가 CATEGORY인지 확인
    if (createDto.parentId) {
      const parent = await this.prisma.wbsNode.findUnique({
        where: { id: createDto.parentId },
      });
      
      if (!parent) {
        throw new BadRequestException('Parent node not found');
      }
      
      if (parent.type !== 'CATEGORY') {
        throw new BadRequestException('TASK 노드 하위에는 노드를 생성할 수 없습니다');
      }
    }

    // 2. depth 자동 계산
    let depth = 0;
    if (createDto.parentId) {
      const parent = await this.prisma.wbsNode.findUnique({
        where: { id: createDto.parentId },
      });
      depth = parent.depth + 1;
    }

    // 3. maxWbsDepth 초과 검증
    const project = await this.prisma.project.findUnique({
      where: { id: projectId },
    });
    if (depth > (project.maxWbsDepth || 5)) {
      throw new BadRequestException('400002', 'WBS depth 초과');
    }

    // 4. order 자동 계산 (같은 parent의 마지막 order + 1)
    let order = 0;
    if (createDto.parentId) {
      const lastChild = await this.prisma.wbsNode.findFirst({
        where: {
          parentId: createDto.parentId,
          deletedAt: null,
        },
        orderBy: { order: 'desc' },
      });
      order = (lastChild?.order || 0) + 1;
    } else {
      const lastRoot = await this.prisma.wbsNode.findFirst({
        where: {
          projectId,
          parentId: null,
          deletedAt: null,
        },
        orderBy: { order: 'desc' },
      });
      order = (lastRoot?.order || 0) + 1;
    }

    // 5. 노드 생성
    const node = await this.prisma.wbsNode.create({
      data: {
        projectId,
        parentId: createDto.parentId || null,
        type: createDto.type,
        depth,
        order,
        code: createDto.code || null,
        title: createDto.title,
        description: createDto.description || null,
        startDate: createDto.startDate ? new Date(createDto.startDate) : null,
        endDate: createDto.endDate ? new Date(createDto.endDate) : null,
        progress: 0,
        weight: createDto.weight || 1,
        assigneeId: createDto.assigneeId || null,
        projectCompanyId: createDto.projectCompanyId || null,
      },
      include: {
        assignee: true,
        projectCompany: true,
      },
    });

    return node;
  }
}
```

### Step 3: WbsController에 생성 엔드포인트 추가
`apps/backend/src/modules/wbs/wbs.controller.ts`:
```typescript
import { Controller, Post, Param, Body } from '@nestjs/common';
import { WbsService } from './wbs.service';
import { CreateWbsNodeDto } from './dto/create-wbs-node.dto';
import { RequirePermission } from 'src/auth/decorators/require-permission.decorator';
import { Resource, Action } from 'src/auth/types/permission.types';

@Controller('api/v1/projects/:id/wbs-nodes')
export class WbsController {
  constructor(private wbsService: WbsService) {}

  @Post()
  @RequirePermission(Resource.WBS, Action.EDIT)
  async createWbsNode(
    @Param('id') projectId: string,
    @Body() createDto: CreateWbsNodeDto,
  ) {
    const node = await this.wbsService.createWbsNode(
      parseInt(projectId),
      createDto,
    );
    
    return {
      code: 201000,
      result: {
        node,
      },
    };
  }
}
```
