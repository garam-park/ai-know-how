## 1. 설치

### 방법 A: 스크립트 설치 (가장 빠름)

```bash
curl -fsSL https://opencode.ai/install | bash
```

### 방법 B: 패키지 매니저

```bash
# macOS / Linux (권장 — 항상 최신 버전)
brew install anomalyco/tap/opencode

# npm / bun / pnpm
npm i -g opencode-ai@latest

# Arch Linux
paru -S opencode-bin

# Windows (Scoop)
scoop install opencode
```

### 설치 확인

```bash
opencode --version
```

> 💡 **Windows 사용자**: WSL(Windows Subsystem for Linux) 환경에서 사용하면 성능과 호환성이 훨씬 좋습니다.

## 2. 첫 실행

```bash
# 프로젝트 폴더로 이동 후 실행
cd my-project
opencode
```

TUI(Terminal User Interface)가 열립니다. 처음엔 당황스러울 수 있지만, 몇 가지 핵심 키만 알면 됩니다.

---

## 3. 핵심 단축키 (외워야 할 것들)

| 키 | 기능 |
|----|------|
| `Tab` | **에이전트 전환** (build ↔ plan) |
| `@` | **파일 퍼지 검색** — 컨텍스트에 파일 추가 |
| `/connect` | LLM 제공자 & API 키 연결 |
| `/models` | 사용 가능한 모델 목록 보기 |
| `Ctrl+C` | 현재 작업 취소 |
| `Ctrl+L` | 화면 초기화 |
| `↑ / ↓` | 이전 메시지 탐색 |

## 4. LLM 모델 연결

### 옵션 A: OpenCode Zen (초보자 추천)
LLM API 키 없이 바로 시작할 수 있는 가장 간단한 방법입니다.
- 사이트에서 가입: https://opencode.ai/ko/zen
- 코딩 에이전트에 최적화된 검증 모델 제공

### 옵션 B: GitHub Copilot 계정 사용

```bash
# TUI 안에서
/connect
# → GitHub 선택 → 브라우저 인증
```

### 옵션 C: API 키 직접 설정

```bash
# Anthropic Claude
/connect
# → anthropic 선택 → API 키 입력

# 또는 환경변수로 설정
export ANTHROPIC_API_KEY="sk-ant-..."
```
