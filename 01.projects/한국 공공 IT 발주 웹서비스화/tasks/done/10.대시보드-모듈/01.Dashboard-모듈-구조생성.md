# Dashboard 모듈 구조 생성

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~15분 |
| AI 에이전트 | ~5분 |

## What (무엇을)
`apps/backend/src/modules/dashboard/` 디렉토리를 생성하고 DashboardModule을 AppModule에 등록합니다. 대시보드는 프로젝트의 전체 현황을 한눈에 보여주는 집계 모듈입니다.

## Why (왜)
프로젝트 관리자와 참여사 PM이 프로젝트의 진행 상태를 신속하게 파악할 수 있는 대시보드가 필요합니다. 이를 위해 ProjectMember와 WbsNode 데이터를 집계하는 별도의 모듈을 구성합니다.

## 선행 작업
- `08.멤버-모듈/03.멤버-목록-수정-삭제-API.md`
- `09.WBS-모듈/05.진척률-가중평균-재계산.md`

## 수행 가이드

### Step 1: 디렉토리 구조 생성
```bash
mkdir -p apps/backend/src/modules/dashboard/{dto,utils}
```

### Step 2: 대시보드 DTO 파일 생성
`apps/backend/src/modules/wbs/dto/dashboard.dto.ts`:
```typescript
export class ProjectSummaryDto {
  id: number;
  name: string;
  startDate: Date;
  endDate: Date;
  daysLeft: number; // endDate - today
}

export class ProgressByCompanyDto {
  company: {
    id: number;
    name: string;
  };
  progress: number;
}

export class MembersByCompanyDto {
  company: {
    id: number;
    name: string;
  };
  count: number;
}

export class WbsSummaryDto {
  total: number;
  completed: number; // progress === 100
  inProgress: number; // 0 < progress < 100
  notStarted: number; // progress === 0
}

export class DashboardResponse {
  project: ProjectSummaryDto;
  progress: {
    overall: number;
    byCompany: ProgressByCompanyDto[];
  };
  members: {
    total: number;
    byCompany: MembersByCompanyDto[];
  };
  wbs: WbsSummaryDto;
}
```

**`apps/backend/src/modules/dashboard/dashboard.controller.ts`:**
```typescript
import { Controller } from '@nestjs/common';
import { DashboardService } from './dashboard.service';

@Controller('api/v1/projects/:projectId/dashboard')
export class DashboardController {
  constructor(private dashboardService: DashboardService) {}
}
```

**`apps/backend/src/modules/dashboard/dashboard.module.ts`:**
```typescript
import { Module } from '@nestjs/common';
import { DashboardService } from './dashboard.service';
import { DashboardController } from './dashboard.controller';
import { WbsModule } from '../wbs/wbs.module';

@Module({
  imports: [WbsModule],
  controllers: [DashboardController],
  providers: [DashboardService],
  exports: [DashboardService],
})
export class DashboardModule {}
```

### Step 4: AppModule에 DashboardModule 등록
`apps/backend/src/app.module.ts`의 imports 배열에 추가:
```typescript
import { DashboardModule } from './modules/dashboard/dashboard.module';

@Module({
  imports: [
    // ...
    DashboardModule,
  ],
})
export class AppModule {}
```

### Step 5: 빌드 검증
```bash
cd apps/backend
npm run build
```
