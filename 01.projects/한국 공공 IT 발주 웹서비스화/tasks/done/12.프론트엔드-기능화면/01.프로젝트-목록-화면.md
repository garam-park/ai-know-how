# 프로젝트 목록 화면

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~40분 |
| AI 에이전트 | ~15분 |

## What (무엇을)
/projects 페이지에서 로그인한 사용자가 참여하는 모든 프로젝트를 카드 형태로 표시합니다. 각 카드는 프로젝트명, 기간, 남은 일수, 진척도, 내 역할, 내 회사 정보를 포함합니다.

## Why (왜)
로그인 후 첫 화면으로 사용자가 어떤 프로젝트에 참여하고 있는지 한눈에 파악할 수 있게 합니다.

## 선행 작업
- `11.프론트엔드-공통/05.로그인-회원가입-UI.md`
- `06.프로젝트-모듈/03.프로젝트-목록-상세-API.md`

## 수행 가이드

### Step 1: ProjectListPage 컴포넌트 생성
`apps/frontend/src/pages/projects/ProjectListPage.tsx` 파일을 생성합니다:

```typescript
import React, { useEffect, useState } from 'react';
import { useNavigate } from 'react-router-dom';
import { projectService, ProjectDto } from '../../services/projectService';
import Card from '../../components/Card';
import Button from '../../components/Button';
import Loading from '../../components/Loading';
import './ProjectListPage.css';

const ProjectListPage: React.FC = () => {
  const navigate = useNavigate();
  const [projects, setProjects] = useState<ProjectDto[]>([]);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);

  useEffect(() => {
    const fetchProjects = async () => {
      try {
        setLoading(true);
        const response = await projectService.getProjects(1, 100);
        setProjects(response.items);
      } catch (err) {
        setError(err instanceof Error ? err.message : '프로젝트 목록을 불러올 수 없습니다');
      } finally {
        setLoading(false);
      }
    };

    fetchProjects();
  }, []);

  const handleProjectClick = (projectId: string) => {
    navigate(`/projects/${projectId}`);
  };

  const handleCreateProject = () => {
    navigate('/projects/new');
  };

  if (loading) {
    return <Loading message="프로젝트 목록 로딩 중..." />;
  }

  if (error) {
    return <div className="project-list__error">{error}</div>;
  }

  return (
    <div className="project-list">
      <div className="project-list__header">
        <h1>내 프로젝트</h1>
        <Button variant="primary" onClick={handleCreateProject}>
          + 새 프로젝트
        </Button>
      </div>

      {projects.length === 0 ? (
        <div className="project-list__empty">
          <p>아직 참여하는 프로젝트가 없습니다</p>
          <Button variant="primary" onClick={handleCreateProject}>
            첫 프로젝트 생성하기
          </Button>
        </div>
      ) : (
        <div className="project-list__grid">
          {projects.map((project) => (
            <Card
              key={project.id}
              clickable
              onClick={() => handleProjectClick(project.id)}
              className="project-list__card"
            >
              <div className="project-card">
                <div className="project-card__header">
                  <h2 className="project-card__name">{project.name}</h2>
                  <span className="project-card__role">{project.myRole}</span>
                </div>

                <p className="project-card__company">
                  소속: {project.myCompany}
                </p>

                <div className="project-card__dates">
                  <span>
                    {new Date(project.startDate).toLocaleDateString('ko-KR')}
                  </span>
                  <span>~</span>
                  <span>
                    {new Date(project.endDate).toLocaleDateString('ko-KR')}
                  </span>
                </div>

                <div className="project-card__progress">
                  <div className="project-card__progress-label">
                    <span>진척도</span>
                    <span>{project.progress}%</span>
                  </div>
                  <div className="project-card__progress-bar">
                    <div
                      className="project-card__progress-fill"
                      style={{ width: `${project.progress}%` }}
                    />
                  </div>
                </div>

                <div className="project-card__footer">
                  <div className="project-card__dday">
                    <span className={project.daysLeft < 0 ? 'danger' : ''}>
                      D{project.daysLeft >= 0 ? '-' : ''}{Math.abs(project.daysLeft)}
                    </span>
                  </div>
                  <span className="project-card__link">상세 보기 →</span>
                </div>
              </div>
            </Card>
          ))}
        </div>
      )}
    </div>
  );
};

export default ProjectListPage;
```

### Step 3: 프로젝트 서비스 확장 (필요시)
기존 `projectService.ts`에 다음 메서드 추가:

```typescript
export const projectService = {
  // ... 기존 코드 ...

  getProjects: async (page = 1, pageSize = 10): Promise<PaginatedResponse<ProjectDto>> => {
    const response = await apiClient.get(
      `/projects?page=${page}&pageSize=${pageSize}`
    );
    return response.data;
  },

  // ...
};
```

## 완료 기준
- [ ] ProjectListPage가 GET /projects API 호출
- [ ] 프로젝트 목록이 카드 그리드 형태로 표시
- [ ] 각 카드에 프로젝트명, 기간, 진척도, daysLeft, myRole, myCompany 표시
- [ ] 카드 클릭 시 /projects/:id로 네비게이션
- [ ] 프로젝트가 없으면 빈 상태 메시지 표시
- [ ] 로딩 중일 때 Loading 컴포넌트 표시
- [ ] 에러 발생 시 에러 메시지 표시
- [ ] "새 프로젝트" 버튼 클릭 시 생성 페이지로 이동

- D-day 계산: endDate에서 오늘 날짜 차이
- 진척도 바의 색상은 프로젝트 상태에 따라 변경 가능
- 페이지네이션 추가 시 useEffect 의존성 배열에 page 상태 추가
