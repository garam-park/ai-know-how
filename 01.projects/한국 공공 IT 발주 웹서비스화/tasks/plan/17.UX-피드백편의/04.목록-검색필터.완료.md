# 목록 검색/필터 강화

## 예상 소요시간
| 구분 | 시간 |
|------|------|
| 사람 (숙련 개발자) | ~30분 |
| AI 에이전트 | ~12분 |

## What (무엇을)
프로젝트 목록에 검색 기능을 추가하고, 프로젝트 상태(진행중/완료/전체)별 필터를 구현합니다. 회사 목록의 기존 검색을 debounce 방식으로 개선합니다.

## Why (왜)
프로젝트가 늘어나면 원하는 프로젝트를 찾기 어렵습니다. 현재 프로젝트 목록에는 검색이나 필터가 전혀 없고, 회사 목록의 검색은 버튼 클릭 방식이라 사용성이 떨어집니다.

## 선행 작업
- `17.UX-피드백편의/03.페이지네이션-컴포넌트.md` (페이지네이션과 함께 동작)

## 수행 가이드

### Step 1: useDebounce 커스텀 훅 생성
- 대상 파일: `apps/frontend/src/hooks/useDebounce.ts` (신규)
- 작업 내용:
  ```typescript
  import { useState, useEffect } from 'react';

  export function useDebounce<T>(value: T, delay: number = 300): T {
    const [debouncedValue, setDebouncedValue] = useState(value);

    useEffect(() => {
      const timer = setTimeout(() => setDebouncedValue(value), delay);
      return () => clearTimeout(timer);
    }, [value, delay]);

    return debouncedValue;
  }
  ```

### Step 2: 프로젝트 목록에 검색 추가
- 대상 파일: `apps/frontend/src/pages/ProjectListPage.tsx`
- 작업 내용:
  ```typescript
  import { useDebounce } from '../hooks/useDebounce';

  const [keyword, setKeyword] = useState('');
  const debouncedKeyword = useDebounce(keyword, 300);

  useEffect(() => {
    fetchProjects(debouncedKeyword);
  }, [debouncedKeyword]);

  // JSX: 검색 입력 필드
  <input value={keyword} onChange={(e) => setKeyword(e.target.value)}
    placeholder="사업명으로 검색..."
    style={{ padding: 8, border: '1px solid #cbd5e0', borderRadius: 4, width: 300 }} />
  ```

### Step 3: 프로젝트 상태 필터 추가
- 대상 파일: `apps/frontend/src/pages/ProjectListPage.tsx`
- 작업 내용:
  - 프로젝트 상태: 종료일 기준 (종료일 > 오늘 → 진행중, 종료일 ≤ 오늘 → 완료)
  - 프론트엔드에서 필터링 (데이터 양이 적을 때) 또는 백엔드 파라미터 (데이터 양이 많을 때)
  ```typescript
  type FilterStatus = 'ALL' | 'ACTIVE' | 'COMPLETED';
  const [filter, setFilter] = useState<FilterStatus>('ALL');

  const filteredProjects = projects.filter((p) => {
    if (filter === 'ALL') return true;
    const isCompleted = new Date(p.endDate) < new Date();
    return filter === 'COMPLETED' ? isCompleted : !isCompleted;
  });

  // JSX: 필터 버튼 그룹
  <div style={{ display: 'flex', gap: 4 }}>
    {(['ALL', 'ACTIVE', 'COMPLETED'] as FilterStatus[]).map((f) => (
      <button key={f} onClick={() => setFilter(f)}
        style={{ padding: '6px 12px', border: '1px solid #e2e8f0', borderRadius: 4, cursor: 'pointer',
          background: filter === f ? '#2b6cb0' : '#fff', color: filter === f ? '#fff' : '#4a5568' }}>
        {{ ALL: '전체', ACTIVE: '진행중', COMPLETED: '완료' }[f]}
      </button>
    ))}
  </div>
  ```

### Step 4: 회사 목록 검색을 debounce로 개선
- 대상 파일: `apps/frontend/src/pages/CompanyListPage.tsx`
- 작업 내용:
  - 기존 form submit 방식 → debounce 입력 방식으로 변경
  - 검색 버튼은 유지하되, 입력만으로도 300ms 후 자동 검색
  ```typescript
  import { useDebounce } from '../hooks/useDebounce';

  const debouncedKeyword = useDebounce(keyword, 300);

  useEffect(() => {
    fetchCompanies(debouncedKeyword);
  }, [debouncedKeyword]);
  ```
