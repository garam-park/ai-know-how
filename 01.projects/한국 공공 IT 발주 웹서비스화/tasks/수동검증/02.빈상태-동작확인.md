# 수동 검증: Empty State 동작 확인

## 개요
기존의 다소 건조했던 `데이터가 없습니다.` 메시지를 직관적인 아이콘과 함께 `EmptyState` 컴포넌트로 분리한 내용이 정상적으로 적용되었는지 확인합니다.

## 검증 대상 페이지/탭
- 프로젝트 목록, 회사 목록
- 프로젝트 개별 탭: `WbsTab`, `MembersTab`, `CompaniesTab`

## 검증 시나리오
1. **아이콘 및 문구 렌더링 검증**
   - 데이터가 없는 화면으로 진입했을 때 단순 텍스트가 아닌 **회색조의 큰 이모지(아이콘)와 볼드체 제목 안내문**이 함께 출력되는지 확인합니다.

2. **사용자 유도 액션 버튼 검증**
   - 빈 상태 하단에 `[회사 등록]`, `[프로젝트 생성]`, `[멤버 초대]` 등 각각의 흐름에 맞는 파란색 액션 버튼이 정상적으로 붙어있는지 조회합니다. 
   
3. **토스트(Toast) 메시지 연동 확인**
   - 아직 기능이 구현되지 않은 `[회사 등록]`, `[참여사 추가]`, `[멤버 초대]`, `[루트 카테고리 추가]` 버튼을 우클릭이나 탭에서 클릭해 봅니다.
   - 우측 상단에 파란색 Info 테마의 **"~ 기능은 준비중입니다."** 라는 Toast 팝업이 표시되고, 약 3초 뒤에 부드럽게 화면 밖으로 사라지는지 점검합니다.
