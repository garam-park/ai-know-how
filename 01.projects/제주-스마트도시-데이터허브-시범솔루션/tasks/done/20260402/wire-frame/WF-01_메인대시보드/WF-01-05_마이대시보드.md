## 화면 구성

### 헤더 영역 (상단)
- 제목: "마이 대시보드"
- 우측 버튼 영역:
  - "편집 모드" 토글 버튼 (ON/OFF 상태 시각화, 아이콘+텍스트)
  - 설정 버튼 (톱니바퀴 아이콘)

### 편집 모드 상태 표시 (상단, 편집 모드 활성 시)
- 메시지: "편집 모드: 위젯을 드래그앤드롭으로 이동할 수 있습니다"
- 배경색으로 강조 (흐린 노란색/파란색)
- "저장" 버튼, "취소" 버튼

### 메인 콘텐츠 영역 (2열 레이아웃)

#### 좌측: 그리드 레이아웃 (약 75% 너비)
**개인화 위젯 대시보드** (그리드 레이아웃, 2행 구성):

위젯들이 반응형 그리드에 배치됨. 편집 모드일 때 드래그앤드롭 가능.
편집 모드가 아닐 때는 드래그 커서 표시 안 함.

##### 위젯 1: 민원 요약 (크기: 1x1)
- 제목: "민원 요약"
- 내용:
  - 금일 접수: 156건
  - 미처리: 23건
  - 완료율: 85.3%
- 색상: 파란색 테마

##### 위젯 2: 이벤트 요약 (크기: 1x1)
- 제목: "이벤트 요약"
- 내용:
  - 발생 이벤트: 8건
  - 해결됨: 6건
  - 진행 중: 2건
- 색상: 주황색 테마

##### 위젯 3: 주간 트렌드 (크기: 2x1)
- 제목: "주간 민원 트렌드"
- 내용: 간단한 라인차트 (주 단위, 7일)
- 가로로 넓은 위젯

##### 위젯 4: 주요 알림 (크기: 2x1)
- 제목: "최근 알림"
- 내용: 최근 5개 알림 목록 (타이틀 + 시각)
- 가로로 넓은 위젯

##### 위젯 5: 날씨 (크기: 1x1, 선택적)
- 제목: "제주 날씨"
- 내용: 현재 온도, 습도, 풍속, 아이콘
- 더미 데이터: 15°C, 습도 65%, 남동풍 3m/s

##### 위젯 6: 빠른 액세스 (크기: 1x1)
- 제목: "빠른 액세스"
- 내용: 자주 사용하는 메뉴 3~4개 버튼
  - 민원 조회
  - 보고서
  - 설정
  - 로그아웃

#### 우측: 위젯 추가 패널 (약 25% 너비)
**추가 가능한 위젯 목록** (스크롤 가능, 카드 형태):

- 각 위젯 카드:
  - 위젯 아이콘
  - 위젯 이름
  - 설명 텍스트 (한두 줄)
  - "추가" 버튼

##### 추가 가능 위젯 예시:
1. **AI 성능 요약**
   - "AI 모델별 성능 지표 표시"
   - 추가 버튼

2. **시스템 상태**
   - "서버/DB/스토리지 상태 모니터링"
   - 추가 버튼

3. **부서별 현황**
   - "부서별 민원 처리 현황 차트"
   - 추가 버튼

4. **나의 작업**
   - "할당된 민원 및 업무 목록"
   - 추가 버튼

5. **공지사항**
   - "최근 공지사항 피드"
   - 추가 버튼

6. **날씨 상세**
   - "제주 지역별 날씨 예보"
   - 추가 버튼

### 하단 영역 (편집 모드 활성 시)
**레이아웃 관리 버튼**
- "저장" 버튼 (위젯 배치 저장)
- "초기화" 버튼 (기본 레이아웃으로 복원)
- "취소" 버튼 (편집 모드 종료, 변경사항 미적용)

### 빈 위젯 상태 (위젯 추가 전)
- 회색 박스에 "+" 아이콘
- "위젯을 추가하세요"
- 우측 패널에서 선택하여 추가

## 사용 컴포넌트
- WF-00-01: 기본 레이아웃 (헤더, 사이드바)
- WF-00-02: 공통 카드 컴포넌트
- WF-00-03: 토글 버튼, 드롭다운
- WF-00-04: 간단한 차트 (라인차트)
