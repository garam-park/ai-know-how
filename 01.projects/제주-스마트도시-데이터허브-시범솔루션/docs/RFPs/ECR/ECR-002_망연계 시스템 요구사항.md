# ECR-002 망연계 시스템 요구사항

| 항목                  | 내용                   |
| --------------------- | ---------------------- |
| **요구사항 분류**     | 시스템 구성 요구사항   |
| **요구사항 고유번호** | ECR-002                |
| **요구사항 명칭**     | 망연계 시스템 요구사항 |

## 정의

망연계 시스템 세부 규격

## 세부내용

- 수량 : 4식
- 기능 :
  - 국정원 EAL4 CC 인증,
  - 한국정보통신협회 GS인증(1등급)
  - CentOS 7.x 이상의 OS를 제공
  - 국정원 암호화 검증필 탑재 제품
  - 차세대 IP 주소체계 대응을 위한 IPv6 인증 제품
  - 물리적으로 송신(TX)채널과 수신(RX)채널을 분리하여 데이터 전송 일방향성 제공
  - 망간통신은 일반적인 TCP/IP가 아닌 안전한 전용프로토콜을 통한 기밀성을 보장

> ※ 동등 이상 기능 제공하여야 함
