# TER-001 테스트 공통사항

| 항목                  | 내용            |
| --------------------- | --------------- |
| **요구사항 분류**     | 테스트 요구사항 |
| **요구사항 고유번호** | TER-001         |
| **요구사항 명칭**     | 테스트 공통사항 |

## 정의

테스트 요구 공통사항

## 세부내용

- 목표시스템의 구축 단계별 테스트 계획을 수립하여 테스트 할 수 있도록 제시해야 함.
- 개발초기부터 구축완료까지 지속적인 테스트와 해당 결과를 반영하여 요구사항이 모두 충족되는 목표시스템이 될 수 있도록 제시해야 함
- 목표시스템 가동을 위한 방안을 제시하고 발주기관이 요구하는 기간, 횟수에 차질 없이 시행할 수 있는 방안을 제시해야 함
- 목표시스템의 기능을 명확하게 확인할 수 있는 방안, 기법, 대상 기능, 도구, 일정 등을 포함한 계획을 수립하여 제시해야 함
- 목표시스템은 단위테스트, 통합테스트 등을 포함하여 제시해야 함
- 목표시스템의 테스트는 방법, 시나리오, 일정, 사용도구 등을 계획서에 포함시켜 제시해야 함
- 테스트와 관련된 산출물로 단계별 시험계획서, 통합시험계획서와 각 테스트 결과서를 제출할 수 있도록 제시해야 함

## 산출정보

단위/종합 테스트 계획서/결과서
